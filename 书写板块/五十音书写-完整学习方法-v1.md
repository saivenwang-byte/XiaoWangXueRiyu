## 1. 初学者为什么要单独练「写」

| 能力 | 入口 | 本 App 做什么 |
|------|------|----------------|
| **认** | 底栏「注音」`intro.html` | 五十音全景、🔊、易错提示 |
| **听** | 同上 + 课文会話 | 不重复做 TTS 课 |
| **写** | 底栏「书写」`#view-write` | 笔顺 + 田字格摹写 |

书写**不计入课内海星**（见 `docs/我们-产品愿景-学习の道.md`），避免与四关抢注意力；有独立进度 `已练 n/46`。

推荐路径：

```text
① 注音：先认清 46+ん 读音与行段
② 书写：按笔顺在米字格摹写（本板块）
③ 课文：会話/单词巩固（不强制先写完 46 字）
```

## 2. 学习方法（四步循环 · 每字 3–5 分钟）

### 步骤 A · 看（一览笔顺）

1. 打开 **书写** → 点选假名（如 む）  
2. 默认 **一览**：所有笔划**分色**同时显示（对齐 [カラーで書き順](https://happylilac.net/hiraganahyo-kakizyun.html)）  
3. 读图例：**数字 = 笔序 · 起点小箭头 = 起笔方向 · 虚线 = 抬笔移笔**（默认一览分色）

### 步骤 B · 跟（专注单笔，可选）

1. 点 **专注** 或 ‹ ›，只强调当前第 n 笔  
2. 看下行文案：「起笔后 从左向右」等  
3. 对照教材标注：起点数字 + 短箭头（静态，无动画）

### 步骤 C · 写（摹写）

1. 对照 **分色笔顺路径**（一览或专注）在米字格内跟写  
2. 手指/触控笔在 **Canvas** 层书写（笔触较细，见 `STROKE_WIDTH`）  
3. 可关 **笔顺** 勾选，仅留格线 + 自写  

### 步骤 D · 完成（勾选）

1. 点 **完成** → 清音表该字 ✓  
2. 不要求字形评分；重练可再次进入并 **清除** 笔迹

## 3. 屏上视觉语法（与教材对齐）

| 图层 | 含义 | 颜色/样式 |
|------|------|-----------|
| 田字格 | 间架 | 外框 + 横竖中线四分（无对角，对齐标日教辅方格） |
| 第 n 笔笔划 | 笔顺路径（无描红底字叠影） | **红→绿→蓝→紫** 循环（与标日教辅笔顺表一致） |
| 起点数字 | 笔序 | 与笔划同色，贴在起笔旁；第 1 笔加粗 |
| 黑色短箭头 | 起笔方向 | 仅起笔处一小段，不沿整笔长线 |
| 笔间虚线 | 抬笔后移笔 | 粉色点线 50% |
| 学员笔迹 | 最上层 | 深灰 #37474F |

**不做**：笔顺动画、自动打分、椭圆/三角结构辅助线（P1+ 再议）。

## 6. 验收（手机 · v347 · P0.5）

1. 底栏 **书写** → 「初学者怎么写」默认折叠；假名切换 **上行日文 / 下行中文**  
2. 进入单字 → 工具栏一行：**笔顺** + **一览/专注** + 步进；无「描红/自由写」重复入口  
3. **默认一览** 分色全笔；起点 **数字 + 黑箭头**  
4. **を** 读音 **o**；**す** 等 2 笔与标日笔顺表一致（字形以笔序为准，见审计 §4）  
5. 无动画；可关笔顺层；课文海星不变
