## 2. 用户认可 · 屏上执行标准（重写真源）

### 2.1 共同层（平片共用）

```text
田字格（红细线四分）
  └─ 层 A：淡灰整字摹写底（outline 填色 · 细 · 不挡色）
  └─ 层 B：分色笔顺细线 + 起点数字 + 短箭头 + 笔间虚线
  └─ 层 C：学员 Canvas（最上 · #37474F · 2.6px）
```

- **默认一览**：层 B 全笔同时显示  
- **专注**：当前笔 100%，已完成笔 ~28% 透明  
- **无动画**

### 2.2 Phase A · 平假名（ひらがな）— **先做、做满 46**

| 维度 | 标准 |
|------|------|
| **数据键** | `KANA_STROKE_GUIDE['あ']` … 平假名键 |
| **灰底** | `guide.outline[]` SVG 填色；无 outline 时 `.write-l0-trace-char` 淡灰字体 |
| **笔顺线** | `tension: 0.38–0.42` · `pathW overview≈10` · `cornerDeg≈132` |
| **弧点** | 每笔 ≤14 点（保留弧线，勿压成 2–3 直点） |
| **色序** | `#E53935` `#43A047` `#1E88E5` `#8E24AA` `#FB8C00` 循环 |
| **手工字** | 仅抽检失败字进 `HIRA_MANUAL_FULL`（非全表重写） |
| **验收** | あいうえお行 + さかき + なよ + むす を手机对照 happylilac / nazorigaki PDF |

### 2.3 Phase B · 片假名（カタカナ）— **平假名 Phase A 通过后再做**

| 维度 | 标准 |
|------|------|
| **数据键** | **独立** 片假名键 `ア`…（animCJK 片假名 SVG + `KATA_MANUAL_*`） |
| **灰底** | 片假名 `outline` 或 trace-char **カタカナ** 字体（略瘦、略深） |
| **笔顺线** | `tension: 0.24–0.26` · 略利、保留折角感 |
| **弧点** | 每笔 ≤11 点 |
| **与平假名关系** | **笔数可对齐**；**路径不共用**平假名坐标 |
| **已知难点** | ヨ 四笔、オ/ホ 无竖钩、マ 对角、フ 非ふ 误植（v372 清单） |
| **顶栏** | `シ` `し` · さ行 + 「笔顺按平假名（标日）」直至片假名路径验收完 |

### 2.4 渲染禁令（v373 教训）

| 禁止 | 原因 |
|------|------|
| `renderGhostFromStrokes()` + `ghostW>20` | 灰团、双环 |
| 平片共用同一 `tension/ghostW` | 片假名失真 |
| 用 animCJK 中线当「教科书体整字」不校形 | あ 等笔划分解错误 |
