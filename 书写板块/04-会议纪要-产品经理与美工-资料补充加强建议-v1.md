## 3. 产品经理建议

### P0 · 优先拍板（建议 1–2 周内）

1. **描红模式接上 trace 资产**  
   笔顺开 = panel；描红开 / 笔顺关 = trace 淡底 + Canvas。与「无动画、无评分」一致。

2. **46×2 字 QA 验收表**  
   PM 出表：裁切完整性、例词解析、片假名页序；美工勾 FAIL，工程只修 FAIL。

3. **`?` 学法抽屉补内容**  
   - 怎么写（田字格 + とめる/はらう/はねる 白话）  
   - 打印练习纸链接  
   - 延伸听/看（Podcast/YouTube）

4. **文档与实现拉齐**  
   `02-用户认可方案` Phase B 仍写「片假名独立 SVG 路径」；v382 已以 **PDF nazorigaki** 为主。  
   纪要建议：**屏上片假名以 PDF 为主、SVG 仅降级**。

### P1 · 体验加强

5. 片假名例词 **小插图**（美工从 PDF 提炼线稿，非整页贴图）  
6. 打印 / 分享练习纸（`matome-hiragana-rensyu-1` 等）  
7. **V2-C / V2-D**：抽屉抛光 + 390×844 真机验收（见 `03-版面重设计-v2`）  
8. **meta 补洞**：人工或规则补全 `examples` / `techniques` 空项

### P2 · 二期 / 明确不做

9. 浊音・拗音・课内词书写 — 二期  
10. 笔顺动画、自动评分、错字本 — P0 禁止  
11. 主题 PDF（情人节等）— 运营素材，不进清音主路径

## 4. 美工建议

### 4.1 单字页（屏 B）

| 项 | 现状 | 建议 |
|----|------|------|
| 田字格 | 十字虚线、无对角 ✅ | `美工第一版` 提纲含对角线 → **以产品纪要为准**，改提纲 |
| nazorigaki 画幅 | `aspect-ratio: 1` | 粉框与格线同心；抽检验 あ/イ/ハ/ヨ 等 |
| 层叠 | panel 盖住格线 | 描红态：trace 底 → 格线 → panel 可关 → Canvas 顶 |
| 片假名顶栏 | 大字カタ + 小平 | nazorigaki 模式下无「笔顺按平假名」降级角标 |

### 4.2 瓦片棋盘（屏 A）

- 已练 ✓ 保持轻量  
- 片假名模式：大字 + 角标平假名，与单字页一致  
- **不需**每格插图

### 4.3 可从 PDF 再提炼（不必整页贴）

| 资产 | 用途 |
|------|------|
| panel | 笔顺主图（已自动化） |
| trace 2×2 | 描红底（待接 UI） |
| 例词插图 | P1 知识条 32×32 线稿 |
| 打印整页 | P1 下载，不屏内嵌 |

### 4.4 美工不建议再做

- 46 字手绘 SVG 笔顺（与 PDF 真源重复）  
- PDF 整页塞进 H5（与「单字练习帖」定位冲突）  
- 恢复对角米字格（与现规格冲突）
