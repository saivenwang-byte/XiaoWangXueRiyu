## 4. 信息架构 · 四栏与 L0 书写（第一版）

```mermaid
flowchart LR
  subgraph nav [底栏 L0 左到右]
    K[注音 intro.html]

  end
  K -->|查听| Intro[Part-0 五十音全景]

  W -->|P0| L0[清音表 行选择]
  L0 --> Cell[米字格画布]
  Cell --> M1[描红]
  Cell --> M2[自由写]
  N -->|错题/笔记| Notes[现有笔记]
```

### 4.1 L0 屏结构（P0 · 单屏流程）

1. **入口**：底栏「书写」→ 默认 **清音表**（与入門 `intro_seion` 同序：あ行…わ行 + ん）
2. **选字**：点一格（如 さ）→ 进入 **书写页**
3. **书写页布局**（390×844 真机框）  
   - 顶：← 返回 · **さ · 清音** · 🔊  
   - 中：**一个大米字格**（≈ 72–80% 屏宽，正方形）  
     - 层1：米字格虚线  
     - 层2：描红底字（可开关）  
     - 层3：Canvas 笔迹（手指/笔）  
   - 下：**描红 | 自由写** 切换 · **清除** · **完成**  
4. **完成**：标记该字「已练」→ 回表；**不**弹复杂星级（P0 可 ✓ 勾）

### 4.2 与标日 / 单元的关联（P0 弱关联 · P1 强关联）

| 阶段 | 关联方式 |
|------|----------|
| **P0** | 仅对齐 **入門清音表** + 文案「学完入門再来练写」；**不**按第 N 课解锁 |
| **P1** | 第 1–4 单元 · 当课 **5 词书写**（PRD L1） |
| **P2** | 第 5–12 课词 + 汉字笔顺 |
| **P3** | 第 13–24 短句听写 |

梯次难度：**笔画数 ↑ → 浊音/拗音 → 单词 → 句子**，与 PRD 一致，但 **工程分期**。
