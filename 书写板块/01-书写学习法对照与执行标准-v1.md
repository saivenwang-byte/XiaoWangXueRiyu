## 3. happylilac 与「粉色标注表」两种语法（均静态）

同一站点 [書き順つき](https://happylilac.net/hiraganahyo-kakizyun.html) 实际有 **两条产品线**，本 App 建议 **合并取用**：

### 3.1 A 线 · カラーで書き順（分色一览）

```text
整字浅底 + 每笔一色（红蓝绿紫橙循环）
全笔同时可见 · 无动画
```

### 3.2 B 线 · 粉色笔顺标注表（你最新截图）

在黑色示范字上叠 **粉色教学层**（幼儿园/笔顺表常见）：

| 图层 | 作用 | P0.5 是否做 |
|------|------|-------------|
| 笔序数字 `1 2 3` | 起笔顺序 | ✅ |
| 起点空心圆 | 起笔位置 | ❌ P0.5 已去掉 |
| 端点/沿线小箭头 | 行笔方向 | ✅ |
| **笔间虚线** | 抬笔后移笔路径（空中连线） | ✅（animCJK 首尾坐标可算） |
| 结构辅助（椭圆/三角/十字/对齐线/角度） | 比例与间架 | ⏸ P1+（需逐字版面数据，animCJK 无） |

```text
整字（浅灰）= 摹写底
粉色/分色层 = 笔顺教学（静态）
Canvas = 学员笔迹（最上）
无动画
```

与当前工程 **v336** 的差异：

| 项 | 当前 v336 | 建议 v337（对齐 happylilac） |
|----|-----------|------------------------------|
| 笔划颜色 | 当前笔红色，已完成灰 | **每笔固定色**（全字同时显示） |
| 起笔 | 红点 | **数字 + 箭头** 在起点旁 |
| 方向 | 笔划内一条线+箭头 | **起点小箭头**即可（与教材一致） |
| 模式 | 逐笔高亮（‹ ›） | 保留逐笔可选；**默认全笔分色一览** |
| 底字 | 大号浅灰假名 | 保留；色深略浅于笔顺色 |

## 4. 推荐产品方案（本小程序 · P0 定稿 + P0.5 增强）

### 4.1 信息架构（不变）

```text
注音 intro → 认·听·全景表
书写 view-write → 写·笔顺·摹写
课文 home → 四关（不含写）
笔记 me → 错题与复习
```

### 4.2 单字书写页（推荐交互 · 静态）

**布局（自上而下）**

1. 顶：← 返回 · 假名 · 行名 · 🔊  
2. 开关：描红底字 | 笔顺提示（默认开）  
3. **米字格区**（红细线，与教材格纸一致）  
   - 层 A：浅粉/浅灰 **整字描红**（opacity ≈ 0.12–0.18）  
   - 层 B：**分色笔划** + 起点 `n→`（animCJK 路径）  
   - 层 C：学员 Canvas 笔迹（最上）  
4. 文案：`第1笔→第2笔→…` 或单行「按数字顺序跟写」  
5. **完成** = 勾选进度（仍不评星）

**两种查看模式（建议都保留）**

| 模式 | 场景 | UI |
|------|------|-----|
| **一览**（默认） | 对照整张笔顺表 | 全笔分色 + 数字箭头同时显示 |
| **专注** | 只练当前笔 | ‹ › 只强调第 n 笔，其余笔 25% 透明度 |

### 4.3 清音表入口（与 intro 同序）

- 格表点击进入单字页（现有）  
- 已练 ✓ 角标（现有）  
- P0.5 可选：表内缩略「分色笔顺」小图（性能允许再做）

### 4.4 分期（避免 scope 膨胀）

| 阶段 | 内容 |
|------|------|
| **P0（已交付）** | 四栏 · 米字格 · Canvas · animCJK · 静态提示雏形 |
| **P0.5（建议下一刀）** | A 分色 + B 起点圆·数字·箭头·**笔间虚线**；默认一览；‹› 专注单笔 |
| **P1** | 浊音/拗音表；片假名同页对照（教材「ひらがな·カタカナ·ローマ字」） |
| **P2** | 课内 5 词书写；仍不计海星 |

## 5. 执行标准（美工 + 工程 · 可验收）

### 5.1 视觉

| 要素 | 标准 |
|------|------|
| 田字格 | 外框红 `#C62828` 45% · 横竖中线 24% · **无对角** |
| 描红底字 | `opacity 0.14–0.18`；不挡分色笔划 |
| 笔划色序 | `#E53935` `#1E88E5` `#43A047` `#8E24AA` `#FB8C00` 循环（与截图一致） |
| 笔划线宽 | 视口内等效 36–42px（1024 坐标系） |
| 起点标记 | 数字 10–12px 等效 + 箭头长 ≈ 1.2 字高；**第 1 笔数字可用红色加粗** |
| 动画 | **禁止**（含自动播放、dashoffset 动画） |

### 5.2 数据

| 项 | 真源 |
|----|------|
| 笔划路径 | `js/data/kana-stroke-guide.js`（`python scripts/build-kana-stroke-guide.py`） |
| 字表顺序 | `INTRO_GOJUON_SEION` |
| 缺失字 | 不进入书写页；表内标注「准备中」 |

### 5.3 工程文件（P0.5 拟改）

- `js/write-kana-stroke-ui.js` — 分色渲染 + 起点数字箭头  
- `css/write-l0.css` — 格线/图例  
- `书写板块/P0-L0-范围锁定.md` — 更新描红提示描述  
- `docs/DESIGN-L0-图标与人设全景.md` — 底栏四格（与现网对齐）  
- bump `CACHE_VER` · `发布前自检.bat`

### 5.4 验收步骤（手机 · v 待定）

1. 底栏 **书写** → 选 **む** 或 **あ**  
2. 见 **多色笔划** + 每笔起点 **1→ 2→ 3** 与箭头方向  
3. 无动画；开关「笔顺提示」可隐藏层 B  
4. 手指书写在 Canvas 层；**完成** 后清音表 ✓  
5. 回课文：**海星与关锁不变**
