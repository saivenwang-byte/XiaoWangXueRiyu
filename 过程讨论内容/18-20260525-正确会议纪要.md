## 一、数据架构重构

**决策**：放弃多层 patch/merge/inject 架构，从 PRD 源文件直接生成单一数据文件。

| 旧 | 新 |
|----|-----|
| 6个数据文件 + merge + patch + inject | `generate-all-lessons.py` → `lessons-data.js`（834KB） |

**生成结果**：24课 × 982词 × 105语法 × 229测验 × 376对话

**删除**：`lessons-stage2-mvp.js`、`lessons-prd-unreleased-mvp.js`、`lessons-supplement-mvp.js`、`lesson-vocab-biaori.js`、`lessons-mvp-depth.js`

### 生成器修复

| Bug | 修复 |
|-----|------|
| titleZh 取教师备注 | 优先取 `意义：` 文本 |
| 例句只取第一条 | 全量收集 + 中文翻译到 `exampleZh` |
| 测验截断在6题 | 放开上限 |
| 表格格式例句未提取 | 增加 `示例`/`用例` 列解析 |
| 部分节点无 titleZh | body 首行兜底 |

## 二、L1 课文简介页

### 入口流程

```
首页 → 点课 → 课文简介页（gate=-1）→ 选模块 → 进入五关
```

### 布局

- 课号 + 日文标题 + 主题中文
- 统计条：词数/句数/语法数/题数
- 课文简介折叠（默认展开）
- 5个模块折叠卡（手风琴，同时只展开一个）
- 每卡左侧颜色条 + 预览 + 「进入XX关 →」按钮
- 底部传导提示链：单词→会话→文法→作業→拡張

### 模块颜色

| 模块 | 色值 |
|------|------|
| 単語 | 蓝 #90caf9 |
| 会話 | 绿 #a5d6a7 |
| 文法 | 橙 #ffcc80 |
| 作業 | 紫 #ce93d8 |
| 拡張 | 青绿 #80cbc4 |

## 三、L1 五关 UI 修改

### 3.1 Tab 栏

- 去掉 emoji 图标（🃏💬📖📝📋）→ 纯文字
- 去掉 ✅ 完成标记
- 选中态：淡色背景 + 深色文字
- 默认进入单词关（gate=0）
- 每关完成后底部显示「进入 XX →」传导按钮

### 3.2 单词关

- 28词表格：序号 + 日语（ruby）+ 假名 + 声调 + 品词 + 中文 + 🔊
- 单词列表下方追加三个折叠区块：
  - 🗣 発音ポイント（默认展开）
  - 📜 語源メモ
  - 🔮 活用予告
- 每词可关联知识卡片（L1KnowledgeTips）

### 3.3 会话关

- 去掉 ①②③ 数字标记
- 改为折叠横条，一句一条
- 开场白（A）灰色底单独显示
- B 回应以折叠条带竖排
- 会话要点折叠在顶部

### 3.4 文法关

- 语法卡片：日文标题 + 中文标题 + 日文说明 + 中文说明 + 例句 + 🔊
- `exampleZh` 兼容数组格式（防崩溃）
- 例句下显示中文翻译

### 3.5 拡張关（まとめ）

- 折叠分类：发音/语法/活用预告/敬语/语源/会话要点/角色扮演/复习总结
- 每类内含「先生のひとこと」子折叠
- 底部「完了 → ホーム」按钮

### 3.6 箭头统一

- 所有折叠 ▶ 旋转90°
- 隐藏默认 marker
- 强制 `list-style:none`

## 四、「笔记」页面

### 4.1 命名

`我的` → `笔记`

### 4.2 计时器

- 新增 `study-timer.js`
- 前台自动计分钟，切后台暂停
- 显示：累计总小时 + 今日分钟

### 4.3 三维知识分层

每课一个折叠卡片：

| 维度 | 标签色 | 来源 |
|------|--------|------|
| 📖 课本 | 橙 | grammarNodes.titleZh + explanationZh |
| 👨‍🏫 老师 | 蓝/紫/红/绿 | 发音/语源/敬语/预告/会话/常见错误 |
| 🧠 学霸 | 紫虚线 | 核心句型/一句话总结/易错提醒 |

### 4.4 薄弱点 TOP3

- 按课统计，非全册
- 红色排名 ①②③ + 错题次数

### 4.5 关联知识卡片

- 从 `📖 これまでのまとめ` 自动解析
- 点击弹出知识卡片弹窗（**不跳转**）
- 卡片：关联说明 + 关联语法点 + 记忆要点
- 右上角小圆形 ✕ 关闭

### 4.6 自学笔记

- 按课一个输入框
- 失焦自动保存 localStorage
- 有笔记的课 ● 标记
- 纯本地，不分享
