## 1. 先回答：三层 Tab 为什么让人懵

### 1.1 之前的问题（v59）

| 位置 | 写了什么 | 问题 |
|------|----------|------|
| 第一行 Tab | `基础` `协同` `核心` | 中文层名 |
| 第二行 Tab | `点格听音` `分块练` `跟读进站` | 又像一层名，又像说明 |
| 再往上 | ①②③④⑤ 五步行法 | 和「三层」概念叠在一起 |

所以会像「基础协同核心写了两遍」——**不是中日双语**，而是 **层名 + 动作说明挤在同一个按钮里**，两行都像标题。

「核心」层日文也是「核心」，中日相同，更容易误会成重复。

### 1.2 v60 改法（已做）

- **一层按钮 = 中日双语**：上 **基礎**（日文）/ 下 **基础**（中文）  
- **动作**（点格听音等）收到 `title` 长按提示，不占第二行  
- 顶部 **①～⑤ 去掉**，只留 **●●○ 2/3** 圆点 + 分数  

请刷新 `intro.html?v=60` 看是否清楚。

## 2. 象音 / 场景音（你提的记忆法）

日本小学更常用 **ことば遊び（言语游戏）**、**音読（听范读跟读）**，而不是汉语式谐音字。

| 类型 | 例子 | 我们怎么用 |
|------|------|------------|
| **动物・物体叫声** | あひる「あー」、吹蜡烛「ふー」 | 先生卡片 **象音** 栏 + emoji |
| **拍子・身体** | 手拍子数拍、促音っ 中间顿 | 协同层「长·促·拨」对比 🔊 |
| **最小对立** | おばさん / おばあさん | 已有，继续强化听辨 |

v60 已在 `intro-kana-tips.js` 为 あ・し・ち・つ・ふ 等加 **象音** 段；其余字可逐字补（讨论后批量写真源）。

**原则**：象音是「帮助点开回忆」，**标准音仍以 🔊 为准**。

## 3. 日本幼儿园・小学怎么学（可借鉴清单）

> 中国学习者 ≠ 日本儿童，但 **过程** 可借：听→说→节奏→字，不借「从小在日语环境」。

### A. 听觉与节奏（最该借）

| 做法 | 日本侧 | 可引进我们 App |
|------|--------|----------------|
| **先生范读** | 教师读，学生跟 | 每格 🔊 + 点字先生卡 |
| **音読・追読** | 师读一句、生跟一句 | 寒暄/示范词「🔊 后跟读」（v59+ 可加强按住跟读） |
| **齐读节奏** | 全班同一拍 | 促音っ：拍手动画或「哒-停-哒」 |
| **あいうえおであそぼう** | 五十音歌谣找规律 | 入門可选 **1 条短歌谣**（30 秒），不背歌词只跟拍 |
| **拍子意识** | ねっこ 3 拍 / がっこう 4 拍 | 长促拨块：显示拍数点 |

### B. 字形（选择性借）

| 做法 | 日本侧 | 可引进 |
|------|--------|--------|
| **笔顺・書き順** | 指书、空书、描写 | **B 期**；A 期只认读不写 |
| **ひらがな先、カタカナ后** | 小学顺序 | 核心层 SK-08 已分三种字 |
| **相似字对比** | シ/ツ、ン/ソ | カタカナ难点可单独 SK（后续） |

### C. 不硬借的

| 做法 | 原因 |
|------|------|
| 一天学完 46 音 | 日本也分周/月；我们已是三层渐进 |
| 纯抄写默写 | 成人学习目的不同 |
| 过多汉字讲解 | 入門只做「三种字外套」比喻 |

## 5. 导航 / 索引栏简化方案（讨论）

### 现状

- **首页**：学习の道 地图 + 24 课 + 00 入門  
- **入門内**：进度点 + 三层 Tab + 协同四色块（仍偏长）  
- **课内**：四关 Tab（単語/会話/文法/テスト）

### 方案 A · 入門只保留「一层导航」（推荐先试）

```text
顶：●●○ 2/3  +  [基礎|協調|核心]  （仅一套）
中：当前层内容（可折叠块默认只展开当前 SK）
底：完成本层 | 第1課→
```

- 协同层四块改为 **手风琴**：同时只展开一块，减下滑。

### 方案 B · 全 App 底栏 3 项

| 图标 | 含义 |
|------|------|
| 道 | 首页地图 |
| 入門 | 00 全景（含发音词典） |
| マイ | 进度/督导（P1b） |

课内仍顶部返回，**不再从首页塞二级说明**。

### 方案 C · 入門 = 发音词典模式

学完三层后，默认只显示 **可搜索五十音表** + 点字卡，三层 Tab 收到「学习模式 / 查阅模式」切换。
