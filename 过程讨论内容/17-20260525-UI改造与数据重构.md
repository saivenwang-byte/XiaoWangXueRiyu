## 一、数据架构重构（12:00–14:00）

### 1.1 单数据源方案

**决策**：放弃多层 patch/merge/inject 架构，改为从 PRD 源文件直接生成单一数据文件。

| 旧架构 | 新架构 |
|--------|--------|
| 6个数据文件 + mergeStageLessonsIntoMvp + applyLessonDepthPatches + injectSupplementMeta | `generate-all-lessons.py` → `lessons-data.js`（834KB） |

**生成结果**：24课 × 982词 × 105语法 × 229测验 × 376对话

**删除文件**：`lessons-stage2-mvp.js`、`lessons-prd-unreleased-mvp.js`、`lessons-supplement-mvp.js`、`lesson-vocab-biaori.js`、`lessons-mvp-depth.js`（共5个）

### 1.2 生成器修复

| Bug | 修复 |
|-----|------|
| titleZh 取教师备注而非"意义"字段 | 优先取 `意义：` 文本 |
| 例句只取第一条 | 收集全部例句+中文翻译到 `exampleZh` |
| 测验截断在6题 | 放开上限为99 |
| 表格格式例句未提取 | 增加 `示例`/`用例` 列 fallback |
| 部分节点无 titleZh | 增加 body 首行兜底 |
