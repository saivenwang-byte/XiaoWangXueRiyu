## 1. INTEPOINT 文件在做什么（我们提炼的「元结构」）

不是「知识点清单」，而是一套 **可导航的全景操作系统**：

| 元组件 | INTEPOINT 里是什么 | 作用 |
|--------|-------------------|------|
| **领域池** | 20 项知识领域（哲学、符号学、教育方法论…） | 横向穷尽：没有遗漏的大类 |
| **三层架构** | 核心 / 协同 / 通用基础 | 纵向深度：什么要精通、什么要会联动、什么只打底 |
| **角色/岗位** | CEO、Hermes、A001… | 谁在用这套能力：视角不同，挂载不同 |
| **双轨赋能** | 古典宗师 + 2000–2026 当代人物 | 传统根基 + 当代语境（AI、扁平、实证） |
| **Skill 模组** | 古典模组 + 时代新增模组 | **可调用的能力包**（不是抽象素质） |
| **作业法** | 固定步骤链（如 赛道定调→制度推演→…） | **怎么干活**的一键流程 |
| **金句锚定** | 痛苦+反思=进步 等 | 记忆钩子、决策锚点 |
| **人格/精神内核** | 顶层格局·熵减进化… | 学习/工作时的「气质」与原则 |

**对我们的启发**：五十音不该做成「网页版五十音表」，而应做成 **「日语发音与文字 · 能力全景图」**——零基础的人一眼知道「要学哪些块、先深后浅、怎么验」。

## 2. 建议：「五十音能力全景模型」V0.1（对标 INTEPOINT）

### 2.1 领域池（8 项 · 对标 20 知识领域）

| # | 能力域 | 对应 PRD | 备注 |
|---|--------|----------|------|
| D1 | 五十音结构认知 | 五十音図・行/段 | **全景地图** |
| D2 | 清音听辨与认读 | 清音 46+ん | 拼音母表 |
| D3 | 浊音·半浊音 | だくおん | 加记号规则 |
| D4 | 拗音合成 | ようおん | 拼音 j/q/x 组合 |
| D5 | 节奏音（长·促·拨） | ちょう・そく・はつ | 拍子与对立 |
| D6 | 文字系统 | ひらがな/カタカナ/漢字 | 三种「字形」 |
| D7 | 寒暄与教室语 | 挨拶・教室表現 | 开口产出 |
| D8 | 学习元能力 | 听→说→读→测 | 与正课四关对齐 |

### 2.2 三层架构（对标 核心/协同/通用）

| 层 | 五十音里是什么 | 学习者要到达的状态 |
|----|----------------|-------------------|
| **通用基础** | 知道日语有几种音、几种字；App 怎么用；每天 10 分钟 | 「我不懵」 |
| **协同** | 清+浊+拗+长促拨 **联动**；罗马字与 🔊 配合；对比听 | 「我会拼、会辨」 |
| **核心** | あ行稳固 + 四易错音 + 3 句寒暄可跟读 | 「能进第 1 課」 |

**教学顺序（固定规则，仿 INTEPOINT 升级逻辑）**：

1. 先 **通用基础** = 全景表 + App 防呆四步  
2. 再 **协同** = 分块（浊/拗/节奏）+ 合成规则  
3. 最后 **核心** = 易错点 + 开口验收  

（与你说的「先整体、再分块加强、再组合」一致。）

### 2.3 双轨赋能（古典 + 当代）

| 轨道 | 五十音教学 | UI 落点 |
|------|------------|---------|
| **古典/标日** | 行段表、标日 Part-0 节点、书写とめはねはらい | 总表、单元顺序 |
| **当代/认知+AI** | 间隔重复、最小对立、图像联想、🔊 跟读、卡尼曼式「先听不猜」 | 点格听音、对比卡、进度条 |

### 2.4 角色视角（对标 CEO / Hermes / A00x）

| 角色 | 谁 | 首页/入門给什么 |
|------|-----|----------------|
| **Hermes · 总控** | 第一次打开 App 的人 | 防呆：你在哪、三步走、怎么验证、回哪 |
| **访客 · 查字典** | 学到第 N 课来查 sh/tsu | 入門当 **リファレンス**（PRD 原意） |
| **学员 · 通关** | 按地图顺序学 | 当前駅高亮 + 阶段完成勾选 |
| **家长 · 督导** | 看进度 | マイ（天数/薄弱）— 入門只露「阶段 2/3」 |

### 2.5 Skill 模组（= 可点选的能力，不是长文）

仿「古典模组 / 时代新增模组」，入門页应用 **能力按钮**：

- `SK-01` 全景浏览（总表一屏）  
- `SK-02` 点听单音（🔊）  
- `SK-03` 行块跟读（あ行/か行…）  
- `SK-04` 易错四音对比（し/す…）  
- `SK-05` 浊音对比（か/が）  
- `SK-06` 拗音公式（き+ゃ）  
- `SK-07` 长音对立听（おばさん/おばあさん）  
- `SK-08` 寒暄跟读  
- `SK-09` 五题入門测（PRD Q1–Q5）  

每个 Skill 有：**1 图 + 1 练 + 1 验**（对标 INTEPOINT 每项 Skill 可执行）。

### 2.6 作业法（固定链 · 对标「核心作业法」）

```text
全景盘点 → 分块突破 → 规则合成 → 开口验证 → 进站第1課
   ↑           ↑           ↑           ↑
 阶段一       阶段二       阶段二末     阶段三
```

与 INTEPOINT「全局盘点→规则建模→任务拆解→节点闭环」同构。

### 2.7 金句锚定（记忆钩子 · 对标 CEO 金句）

| 锚定句 | 用途 |
|--------|------|
| 五十音不是字母，是**音表** | 阶段一总述 |
| 浊音 = 清音 + **两点** | 阶段二 B |
| きゃ 是一拍，不是 ki-ya 两拍 | 拗音 |
| 长音差一拍，**意思就换** | おばさん/おばあさん |
| ひらがな/カタカナ = 同一首歌 **两种字体** | 阶段三 |
| **先听不猜**，🔊 为准 | 当代轨 |

## 7. 与「拼音教学」的统一说法

INTEPOINT 解决的是「人才在组织里**站在哪、会什么、怎么协作**」。  
五十音全景要解决「学员在路径上**先看到全貌、再分块练、再组合开口**」——**同一套元逻辑**：

> **池子穷尽 → 分层深浅 → 双轨古今 → 模组可调用 → 作业法固定 → 锚点好记 → 角色有总控**

你确认后，我们再把 v56 升级成「全景模型版」入門，而不是继续堆段落。
