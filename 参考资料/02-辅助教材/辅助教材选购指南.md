# 标日配套辅助教材 · 选购指南

---

## 1. 《标准日本语同步课课练》N5

| 项目 | 内容 |
|------|------|
| ISBN | 9787568560207 |
| 作者 | 赵明阳等 |
| 出版社 | 大连理工大学出版社 |
| 定价 | ~33元 |

**特点**：与教材课课同步，一课一练。覆盖单词、语法、听力、阅读四大模块。适合大量练习巩固知识、准备JLPT考试。

**每课结构**：单词填空 → 语法选择 → 听力判断 → 阅读理解

---

## 2. 《新版中日交流标准日本语语法手册》

| 项目 | 内容 |
|------|------|
| ISBN | 9787515908236 |
| 作者 | 赵文娟等 |
| 出版社 | 中国宇航出版社 |

**特点**：重新梳理初级全册所有语法点，建立语法框架。适合觉得教材语法点零散、需要系统性梳理的同学。

**结构**：按语法类型分类（助词篇/动词篇/形容词篇/句型篇），非按课次排列。

---

## 3. 《标准日语语法（第2版）》

| 项目 | 内容 |
|------|------|
| 作者 | 顾明耀 |
| 出版社 | 高等教育出版社 |

**特点**：从更高维度系统讲解日语语法体系。不局限于某套教材，适合学完初级后想要融会贯通的学习者。对语法细节有深度解释。

## 4. 《新版中日交流标准日本语 同步练习》

| 项目 | 内容 |
|------|------|

**特点**：官方配套的巩固练习。单元末有模拟能力考试的小测试，与教材内容100%同步。
