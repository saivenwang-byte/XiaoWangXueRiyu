隐私政策

更新日期：2026年5月30日 · 生效日期：2026年5月30日

产品说明：「日语初级课后练习」（学习の道）为课后练习辅助工具，非人民教育出版社官方产品，与《标准日本语》为课后辅助关系。主要面向在校学习日语的初中生等学员。

1. 运营者

由个人开发者运营（H5 网页 + 可选微信小程序壳）。无注册登录、无自建用户账号服务器。

2. 我们实际收集与使用的数据

我们不主动收集姓名、手机号、身份证号、精确地理位置等可识别个人身份的信息。下列数据均保存在您设备本地，用于学习功能，不上传至开发者服务器：

数据项存储位置用途
学习进度（各课关卡完成、测验对错） localStorage 键 hyouga_mvp_v1 续学、地图通关显示、错题复习
自学笔记（课次/单元/全册） 同上 学员自行记录
单词卡片进度、错题列表 同上 复习与测验
五十音书写练习记录 同上 书写页完成标记
入門（五十音）学习进度 hyouga_intro_progress_v1 入門关完成状态
今日/累计学习时长（分钟） hyouga_study_time 界面展示学习日数（非精确行为追踪）
是否显示中文释义 hyouga_mvp_v1 界面偏好
语音资源缓存版本号 hyouga_tts_sw_purge_* 更新朗读 MP3 缓存
开发者测试开关（可选） hyouga_dev_catalog 等 仅开发/内测，普通学员可忽略

您可在「マイ → 设置 · 分享链接 → 清除本机学习数据」一键删除上述学习相关本地数据（不含浏览器对整站的清除，后者请在系统/浏览器设置中操作）。

3. 设备权限与即时处理的数据

3.1 麦克风（可选）

在「会話」跟读中，仅当您点击录音时，系统会请求麦克风。音频在设备本地用于发音比对与即时评语，不上传、不保存录音文件至服务器。

3.2 扬声器 / 媒体播放

播放课文朗读（缓存 MP3 或系统语音），不涉及个人身份信息。

3.3 剪贴板（可选）

点击「复制学习链接」时，将公网学习网址写入剪贴板，便于微信转发。

4. 第三方

  • GitHub Pages：托管静态页面与语音文件；访问产生托管方常规访问日志。
  • 微信：通过微信打开链接或小程序时，适用微信隐私规则。

当前版本不嵌入广告 SDK、第三方统计 SDK 或社交登录。

5. 未成年人保护

本产品定位包含在校日语学习的初中生（通常未满 18 周岁）。我们采取以下措施：

  • 不收集未成年人身份信息，不要求实名注册；
  • 学习数据仅存本机，不上传云端账号；
  • 跟读录音需家长或监护人知情同意后，由学员在本人设备上主动授权麦克风；
  • 建议家长陪同首次使用，并定期使用「清除本机学习数据」管理本机记录；
  • 如发现未成年人不当使用或隐私疑问，请通过分享渠道联系开发者。

成年用户同样适用本政策：我们不因其年龄额外收集信息；自备正版教材后亦可自学使用。

6. 您的权利

  • 查询与删除：通过应用内「清除本机学习数据」或删除浏览器/微信对该站点的数据；
  • 撤回麦克风授权：在系统或浏览器设置中关闭；
  • 停止使用：关闭页面即可，无需注销账号。

7. 政策更新与联系

更新版本发布于本页。隐私问题请通过您获取链接的渠道联系开发者。
