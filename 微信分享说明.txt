【结论】
  不要发 html 文件、不要发 zip、不要只发「手机微信版」文件夹。
  正确做法：把整个项目部署到公网 → 得到 https 链接 → 微信里发链接。
  对方点开就能用，和「小程序」体验类似（其实是手机网页版）。

───────────────────────────────────────────────────────
一、你要做的（只做一次，约 5～10 分钟）
───────────────────────────────────────────────────────

1. 双击运行：打开Netlify拖拽部署.bat
   （浏览器会打开 https://app.netlify.com/drop ）

2. 注册/登录 Netlify（可用 GitHub 邮箱）

3. 把整个文件夹拖进网页中间虚线框：
     D:\【私人】\【小王】\日语学习
   （必须是含 index.html 的那一层，不要只拖子文件夹）

4. 等出现 “Site is live”，记下地址，例如：
     https://qingjing-biaori.netlify.app

5. 可选：Netlify 控制台 → Domain management → Edit site name
   改成好记的英文名（小写+连字符）

6. 自检（重要）：
   · 电脑浏览器打开：https://你的域名.netlify.app/index.html
   · 手机关掉 WiFi，用 4G/5G 也能打开
   · 打开：https://你的域名.netlify.app/share.html
     应显示绿色「已是公网 HTTPS，可发给任何人」

───────────────────────────────────────────────────────
二、微信里怎么发给所有人
───────────────────────────────────────────────────────

【推荐链接】（发这个）
  https://你的域名.netlify.app/index.html

【带二维码的分享页】（自己用或发家长群）
  https://你的域名.netlify.app/share.html

发法：
  1. 在 share.html 点「复制学习链接」或首页右上角 📤
  2. 打开微信 → 文件传输助手 / 班级群 / 好友
  3. 粘贴发送
  4. 对方点链接 → 自动打开 → 选第 14/16/18 课学习

也可点「复制推荐话术」，自带一句说明+链接。

───────────────────────────────────────────────────────
三、对方手机上怎么用
───────────────────────────────────────────────────────

  首页 → 选一课 → ①语法网络 ②练对话 ③测考点
  底部：复习（错题）、我的（家长看进度）

  【录音第二关】微信内置浏览器可能不让录音：
    微信里打开链接 → 右上角 … →「在浏览器中打开」
    或「在 Safari 中打开」→ 再进第二关录音

  【像小程序一样放桌面】
    Safari 打开 → 底部分享 →「添加到主屏幕」

───────────────────────────────────────────────────────
四、和「正式微信小程序」的区别
───────────────────────────────────────────────────────

  当前是「网页版」，分享链接即可，无需微信审核、无需 AppID。

  若要做微信商店里可搜索的「小程序」：
    需注册微信公众平台（小程序）→ 开发者工具上传
    → 提交审核（数日）→ 才能搜到。
    项目里 japanese_learning_miniapp 是骨架，完整功能仍建议用 H5 链接。

  对「发给全班/朋友随便用」：H5 + https 链接最简单。

───────────────────────────────────────────────────────
五、更新内容后怎么重新发布
───────────────────────────────────────────────────────

  再次打开 https://app.netlify.com/drop ，拖入更新后的文件夹。
  或 Netlify 控制台 → 你的站点 → Deploys → 上传新文件夹。

  域名不变，已发出的微信链接自动打开新版本。

───────────────────────────────────────────────────────
六、国内备选（Netlify 偶尔慢时）
───────────────────────────────────────────────────────

  · Gitee Pages：仓库公开 → 上传本文件夹 → 开启 Pages
  · Cloudflare Pages：dash.cloudflare.com → Upload assets

  原则相同：必须是 https:// 开头。
