  标日课后巩固 · Netlify 部署指南（微信可分享 https 链接）
═══════════════════════════════════════════════════════

全程免费，无需写代码。部署完成后会得到永久（免费套餐）的
https://xxxx.netlify.app 地址。

───────────────────────────────────────────────────────
第一步：准备文件夹
───────────────────────────────────────────────────────

确认本目录里有这些文件（不要只上传某一个子文件夹）：

  index.html
  share.html
  manifest.json
  netlify.toml
  css 文件夹
  js 文件夹
  icons 文件夹

【重要】拖进 Netlify 的应是整个「日语学习」文件夹本身，
或选中文件夹里的全部内容打包；不要让 index.html 藏在
再套一层的子目录里，否则打开网站会 404。

───────────────────────────────────────────────────────
第二步：拖拽部署（推荐）
───────────────────────────────────────────────────────

1. 双击运行：打开Netlify拖拽部署.bat
   （或浏览器打开 https://app.netlify.com/drop ）

2. 若没有账号：
   · 点 Sign up
   · 可用 Email，或 Continue with GitHub（推荐）

3. 登录后，页面中间有大虚线框 “Drag and drop your site…”

4. 从资源管理器把文件夹
     D:\【私人】\【小王】\日语学习
   整个拖进虚线框（松开鼠标）

5. 等待上传与发布（约 30 秒～2 分钟）
   出现 “Site is live” 或绿色勾即成功

6. 页面上会显示你的网站地址，形如：
     https://random-name-12345.netlify.app
   点开应能看到「标日课后巩固」首页（三课卡片）

───────────────────────────────────────────────────────
第三步：复制要发微信的链接
───────────────────────────────────────────────────────

【学习主页】（发这个给朋友最常用）
  https://你的站点名.netlify.app/index.html

【分享页】（带二维码、复制按钮）
  https://你的站点名.netlify.app/share.html

建议：自己先手机用流量（关掉 WiFi）点开测一遍。

───────────────────────────────────────────────────────
第四步：改一个好记的网址名（可选）
───────────────────────────────────────────────────────

1. 打开 https://app.netlify.com 登录
2. 点击你刚创建的那个 Site
3. 左侧 Domain management → Options → Edit site name
4. 改成例如：qingjing-biaori
   则地址变为：https://qingjing-biaori.netlify.app
   （只能用小写英文、数字、连字符；若已被占用需换一个）

───────────────────────────────────────────────────────
第五步：微信里怎么用
───────────────────────────────────────────────────────

1. 微信发给「文件传输助手」或好友：
     https://你的站点名.netlify.app/index.html

2. 点开链接即可学习（第 14/16/18 课 · 三关）

3. 【第二关录音】需要麦克风：
   · 微信内打开后 → 右上角 … 
   · 选择「在浏览器中打开」或「在 Safari 中打开」
   · 再进入「② 练对话」→ 录音回听

4. 首页右上角 📤 或 share.html 可一键复制链接发微信

5. 想固定到手机桌面（像小程序）：
   Safari 打开 → 底部分享 →「添加到主屏幕」

───────────────────────────────────────────────────────
以后更新了代码怎么重新发布？
───────────────────────────────────────────────────────

方式 1（最简单）：
  再次打开 https://app.netlify.com/drop
  把更新后的「日语学习」文件夹再拖一次
  （会生成一个新站点；旧链接仍可用，可删旧站）

方式 2（同一站点覆盖）：
  Netlify 控制台 → 你的 Site → Deploys
  → Drag and drop（或 Deploy manually → Upload folder）

方式 3（进阶，自动发布）：
  把项目推到 GitHub，在 Netlify 里 Import from Git，
  以后每次 push 自动更新（见 Netlify 文档）。

───────────────────────────────────────────────────────
常见问题
───────────────────────────────────────────────────────

Q：打开是 404 Page not found
A：多半是上传目录层级错了。确保 index.html 在网站根目录。
   正确：https://xxx.netlify.app/index.html
   错误：https://xxx.netlify.app/日语学习/index.html

Q：微信里打不开或提示不安全
A：Netlify 默认是 HTTPS，检查链接是否完整 https:// 开头。
   不要用 http://

Q：按住说话没反应
A：微信内置浏览器限制多。必须用 Safari 打开（见第五步）。

Q：需要登录密码吗？
A：默认公开。若只想自己用：Site → Site configuration
   → Access control → Password protection（部分套餐功能）。

Q：数据会丢吗？
A：学习进度在各自手机浏览器里，不在 Netlify 服务器。
   换手机或清缓存会清空，属正常。
