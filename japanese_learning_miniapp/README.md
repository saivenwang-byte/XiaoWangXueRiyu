# 标日课后巩固 · 微信小程序骨架

与仓库根目录 **PRD 三关**（`index.html`）共用 `js/data/lessons-mvp.js` 课程数据。

## 入口

| 入口 | 说明 |
|------|------|
| `h5/index.html` | **完整三关**（语法翻转/链接、录音、测验），引用 `../../js/*` |
| 微信开发者工具 | 打开本目录，`app.json` 已配置 **首页 / 复习 / 我的** |
| `../../legacy.html` | 旧版 24 课情景互动 |

## 小程序页

- **首页**：三课卡片 + 三关进度
- **lessonHub**：进入 ①语法 ②对话 ③测试（精简实现）
- **复习 / 我的**：错题提醒、热力图、TOP3

完整交互（ruby、【拓】、对比视图、录音）请优先使用 **H5** 或根目录 `index.html`。

## 主题色

珊瑚橙 `#FF8A80` + 奶油白 `#FFF8E1`（见 `app.json`）
