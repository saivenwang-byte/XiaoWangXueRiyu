発音ポイント（发音要点）
「あります」和「います」 的长音与拨音：「あります」的「す」母音无声化；「います」同样。「す」在词尾时发音很轻。

「と」「や」「など」 这些助词多与前面单词连读。例如「机といす」中的「と」要轻轻带过。

方位词：うえ、した、まえ、うしろ等多为平板型或头高型，注意重音位置。

語源メモ（词源注释）
「あります」 源自古典日语「あり」，对应现代汉语“有/存在”。

「います」 原本表示“坐着”或“停留”，后用于有生命的物体存在。

「どこにも」 中的「にも」是「に」+「も」表示“任何地方都（不）”，强调全否定。

活用予告（活用预告）
本课出现动词 「あります」「います」。将来会学习它们的否定形「ありません」「いません」、过去形「ありました」「いました」等（第5课）。

注意「います」是2类动词（上一段），「あります」是1类动词（五段），但ます形变化规则第5课才系统学。

【语法】
ノード1：あります / います（存在句）
句型：

名1 に 名2 が あります：某处有某物（非生物）

名1 に 名2 が います：某处有某生物/人

例句：

部屋に 机があります。 → 房间里有书桌。

庭に 犬がいます。 → 院子里有狗。

会社に 田中さんがいます。 → 公司里有田中先生。

否定形式：

ありません / いません

例句：机の上に 本がありません。

疑问形式：

名1 に 名2 が ありますか / いますか

肯定回答：はい、あります / います

否定回答：いいえ、ありません / いません

注意：

生物与非生物的区别：树木、植物、机械等通常用「あります」，但有时人与动物也可以使用「あります」是绝对错误。

某地存在某物时，地点后必须用「に」，事物后必须用「が」。

ノード2：～は～にあります / います（事物/人物的所在）
句型：名2 は 名1 に あります / います
意义：某物/人在某处。

例句：

机は 部屋にあります。 → 书桌在房间里。

犬は 庭にいます。 → 狗在院子里。

对比：

部屋に 机があります。（强调“存在”）

机は 部屋にあります。（强调“位置”）

「は」与「が」的再次对比：
在存在句中，「が」提示存在的主体（新信息），「は」提示已知主题（旧信息）。

常用应答：

Q：本は どこにありますか。
A：机の上にあります。

ノード3：〜や〜など（部分列举）
句型：名1 や 名2 など
意义：列举事例（非全部），相当于“像…啦…啦之类的”。

对比「と」：

「と」：全部列举（A和B和C……）

「や～など」：示例列举（A呀B呀等等）

例句：

机の上に 本やペンなどがあります。 → 书桌上有书啦、笔啦之类的。

庭に 猫や犬がいます。 → 院子里有猫啊狗啊。（省略「など」也可以）

かばんの中に 財布や鍵などがあります。

ノード4：方位詞（位置关系）
本课需要掌握的方位词（前、后、左、右、上、下、中、外、隣、近く、間等）。

用法：名1 の ＋方位词 ＋ に ＋ あります/います。

例句：

机の上に 本があります。 → 书桌上面有书。

椅子の下に 猫がいます。 → 椅子下面有猫。

コンビニの隣に 銀行があります。 → 便利店旁边有银行。

公園の中に 池があります。 → 公园里面有池塘。

常见方位词表：

日语	汉语	注意
上（うえ）	上面	点接触面的上方
下（した）	下面	正下方
前（まえ）	前面	面向的方向
後ろ（うしろ）	后面	背后
左（ひだり）	左边	以说话人或参照物为准
右（みぎ）	右边	同样相对方向

外（そと）	外面	外部

【助詞カード】第4课：格助詞「に」「が」以及「と」「や」
助詞	用法	例句
に	存在的地点（场所）	部屋に机があります。
が	存在的主体（主格）	机があります。
と	全部列举	机といすがあります。
や	部分列举	本やペンなどがあります。
注意：

「に」还表示时间点（第5课）、移动目的地（第6课）、对象（第8课）等，本课只学存在场所。

「が」还出现在第3课“新信息”以及后续的“能力/愿望”对象（第11、17课）。

【対比表】第4课：「あります」vs「います」
項目	あります	います
使用对象	无生命、植物、机械等	人、动物、昆虫等有生命者
例句	本があります。	学生がいます。
否定	ありません	いません
过去	ありました	いました
注意：机器人、僵尸等可根据语境选择；但初学严格区分。

【対比表】第4课：「と」vs「や」
項目	と	や（～など）
列举性质	全部列举	示例列举
语感	“A和B和C……”	“A啦B啦等等”
例句	机といすがあります。	机や本などがあります。
【敬語レベル表示】（本课）
「あります/います」的丁寧形就是其本身（ます体），无更尊敬或更随便的变化。但谈到对方存在时，一般不用敬语直接表达存在，而是用别的方式（「いらっしゃいます」等中高级）。

【会话】
基本课文（4句型）
部屋に 机といすがあります。

机の上に 本が三冊あります。

庭に 猫がいます。

かばんの 中に 何がありますか。
— お金と 写真があります。

应用课文：部屋の様子（房间的样子）
场景：小李在日本公司宿舍，给同事小野展示自己的房间。

人物：

李秀麗（り しゅうれい）

小野緑（おの みどり）

对话正文：

小野：李さん、部屋は どうですか。
李：とても きれいです。
小野：何が ありますか。
李：机と 椅子と ベッドが あります。机の上に パソコンや 本などが あります。
小野：窓の 隣に 何が ありますか。
李：本棚が あります。
小野：ベッドの 下に 何か ありますか。
李：はい、靴が あります。
小野：テレビは どこに ありますか。
李：テレビは 机の 前の 壁に かかっています。
小野：そうですか。猫は いますか。
李：いいえ、猫は いません。でも、庭に 小鳥が います。
小野：いい 部屋ですね。

会話のキーポイント（会话要点）
「部屋はどうですか」：「どう」询问“怎么样”，常用语。

「何かありますか」：用「何か」表示“有什么东西吗？”，比「何がありますか」更委婉，不预设一定有。

「テレビは机の前の壁にかかっています」：「かかっています」是动词「かける」的て形+いる，表示“悬挂着的状态”。超出本课范围，但作为接触。

「いい部屋ですね」：共感评价。再次出现「ね」。

ロールプレイ課題（角色扮演）
课题1：两人一组，A询问B的房间有什么，B用本课句型描述，至少使用5个方位词。

课题2：将对话中的“部屋”换成“教室”，改编对话。

课题3：描述自己家的客厅（リビング），使用「～や～など」「～に～があります」。

【作业】
1. 発音練習（发音练习）
朗读「あります」「います」及其否定「ありません」「いません」。

朗读方位词：うえ、した、まえ、うしろ、ひだり、みぎ、なか、そと、となり。

朗读完整句子：部屋に机があります。椅子は机の前にあります。壁に絵がかかっています。

2. 活用ドリル（活用操练）
（本课动词「あります」「います」的否定和疑问形式已完成，未来会学习其他动词）

3. 選択問題（选择题）
Q1：部屋に 机 ＿＿＿ あります。
A. を　B. が　C. に　D. で
→ B（存在主体用「が」）

Q2：庭に 猫 ＿＿＿ いますか。 — はい、います。
A. を　B. が　C. に　D. で
→ B

Q3：机の上に 本 ＿＿＿ ペン ＿＿＿ があります。
A. と・と　B. や・など　C. や・や　D. AもBも正しい
→ D（两者皆可，语感不同）

Q4：猫は 椅子の ＿＿＿ にいます。
A. 上　B. 下　C. 中　D. 外
→ B（常识，猫可能在下，但任意合理均可）

4. 穴埋め問題（填空题）
Q5：かばんの 中に 何＿＿＿ ありますか。
→ が

Q6：犬は 部屋の 中には いません。 庭＿＿＿ います。
→ に

Q7：コンビニの ＿＿＿ に 郵便局があります。（隣）
→ 隣

5. 翻訳問題（日→中）
Q8：机の上に ノートと ペンがあります。
→ 书桌上有笔记本和笔。

Q9：かばんの中に 財布や 鍵などがあります。
→ 包里有钱包啦钥匙啦等等。

6. 翻訳問題（中→日）
Q10：院子里有狗吗？——是的，有。
→ 庭に 犬がいますか。 — はい、います。

Q11：便利店在哪里？——在银行旁边。
→ コンビニは どこにありますか。 — 銀行の 隣にあります。

7. 間違い探し（改错题）
Q12：部屋に 猫があります。
错误：猫是生物，应用「います」。
正解：部屋に 猫がいます。

Q13：机は 部屋が あります。
错误：存在场所应用「に」。
正解：机は 部屋に あります。

8. 作文ミニタスク（微写作）
課題：写5-6句话介绍你的房间或书房。描述有什么家具、物品的位置关系。
要求：使用「～に～があります/います」「～は～にあります」「～や～など」。
例：
私の 部屋に 机と 椅子と ベッドがあります。机の上に パソコンや 本などがあります。本棚は 机の 隣にあります。壁に ポスターが かかっています。窓の 外に 花が あります。とても 快適です。

9. 聴解問題（听力题，需音频，预留）
Q14：听录音，判断物品所在的正确位置（选择图片）。
Q15：听对话，回答「何が どこに ありますか」。

【复习与扩展】
📖 これまでのまとめ（迄今总结）
第4课是第1单元的最后一课，核心是存在句「あります・います」，以及位置关系的表达和「と」「や」的区别。
跨课链接：

第1课：名词句「AはBです」；本课拓展为「Aは場所にあります」。

第2课：指示词「これ・それ・あれ」；本课可搭配成「ここに～があります」。

第3课：场所指示「ここ・そこ・あそこ」；本课加入方位词，描述更精确。

第13课：数量词与存在句结合（机の上に本が三冊あります），届时会更深入学习助数词。
