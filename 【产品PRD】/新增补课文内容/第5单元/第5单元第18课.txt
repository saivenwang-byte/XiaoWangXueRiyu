第18課：携帯電話はとても小さくなりました
本课定位：第5单元（迎新春）的第二课。学习变化表达「～くなります」「～になります」（客观变化）和「～くします」「～にします」（主观改变），以及建议「～ほうがいいです」、推测「～でしょう」「～かもしれません」。本课首次系统引入自动词/他动词的概念，是初级语法的重要分水岭。

発音ポイント
「小さくなる」 中「ちい」长音，「さ」轻读。注意「ちいさい」的长音保持。

「便利になる」 「べんり」的「り」和「に」连读。

「でしょう」 口语常说「でしょ」，声调下降。

「かもしれません」 长句，注意「かも」的「も」和「しれません」的节奏。

語源メモ
「なる」 自动词，表示自然、客观的变化。

「する」 他动词，表示人为的改变、选择。

「～くなる」「～になる」 分别来自形容词和名词的连用形。

「ほうがいい」 源自「方」（方面）的比较用法。

活用予告
本课核心是变化表达（なる/する）和推测（でしょう/かもしれません）。「なる」是自动词，「する」是他动词，需区分。同时学习「ほうがいい」（建议）和「でしょう」（推测）。第19课学习义务/禁止。

【语法】
ノード1：～くなります（イ形容詞の変化）
句型：イ形容詞（去い）＋く なります
意义：表示客观状态的变化（变～了）。

例：

携帯電話はとても小さくなりました。 → 手机变得非常小了。

最近、寒くなりましたね。 → 最近变冷了呢。

日本語が上手になりました。 → 日语变好了。（上手是ナ形容詞，见ノード2）

否定：～くなりません（不会变～）/ ～くなりませんでした（没变～）

ノード2：～になります（ナ形容詞・名詞の変化）
句型：ナ形容詞語幹／名詞 ＋ に なります
意义：表示变成某种状态或结果。

例：

生活が便利になりました。 → 生活变方便了。

息子が医者になりました。 → 儿子成了医生。

友達になりました。 → 成了朋友。

注意：ナ形容詞（～だ）去だ加「に」；名词加「に」。

ノード3：～くします / ～にします（人为改变）
句型：

イ形容詞（去い）＋く します

ナ形容詞語幹／名詞 ＋ に します

意义：表示主观意志促使变化（使～变得～；决定选择～）。

例：

温度を高くします。 → 把温度调高。

部屋をきれいにします。 → 把房间打扫干净。

コーヒーにします。 → 我决定要咖啡。（点餐选择）

旅行は来月にします。 → 旅行定在下个月。

对比「なる」vs「する」：

项目	なる	する
意志	客观、自然	主观、人为
自动词/他动词	自动词（无宾语）	他动词（可带を）
例	寒くなった	温度を下げた（类似）
ノード4：～ほうがいいです（建议）
句型：动词（た形／ない形）＋ほうがいいです
意义：表示建议“最好～”。也可用形容词。

例：

もう少し早く行ったほうがいいです。 → 最好早点去。

無理をしないほうがいいです。 → 最好不要勉强。

健康のためには、よく寝たほうがいいです。 → 为了健康，最好多睡。

注意：肯定建议用た形，否定建议用ない形。

ノード5：でしょう / かもしれません（推测）
でしょう（丁寧推测）：

接续：动词/形容词/名词 + でしょう

意义：大概～吧（确信度较高）

例：明日は晴れるでしょう。 → 明天会晴吧。

かもしれません（可能性推测）：

接续：动词/形容词/名词 + かもしれません

意义：也许～（确信度较低）

例：明日は雨が降るかもしれません。 → 明天也许会下雨。

确信度顺序：
でしょう（80%） ＞ かもしれません（50%） ＞ かもしれない（口语，同程度）

【助詞カード】第18课：変化の「に」「く」
助詞	用法	例
に	ナ形容詞／名詞 + に + なる/する	静かになる、医者になる
く	イ形容詞（去い）＋く + なる/する	安くなる、高くする
【対比表】なる vs する
項目	なる	する
自他	自動詞	他動詞
意思	客观变化	主观改变
接续	イ形：～くなる
ナ形/名：～になる	イ形：～くする
ナ形/名：～にする
例	値段が安くなった	値段を安くした
【対比表】たほうがいい vs ないほうがいい
建議類型	句型	例
肯定建议	た形 + ほうがいい	行ったほうがいい
否定建议	ない形 + ほうがいい	行かないほうがいい
【対比表】でしょう vs かもしれません
項目	でしょう	かもしれません
确信度	较高（～80%）	较低（～50%）
口語形	でしょ	かも
回答	はい、そうでしょう	そうかもしれませんね
【敬語レベル表示】
「でしょう」是「だろう」的丁寧体。「かもしれません」无更敬语形式。建议「ほうがいいです」足够礼貌。

【会话】
基本课文（4句型）
携帯電話はとても小さくなりました。

テレビの音を小さくしてください。

健康のために、毎日運動したほうがいいですよ。

明日は雨が降るかもしれません。

应用课文：生活の変化（生活的变化）
场景：小李和小野在办公室谈论科技、生活变化以及健康。

人物：

李秀麗（り しゅうれい）

小野緑（おの みどり）

森健太郎（もり けんたろう）

对话正文：

小野：李さん、最近の携帯電話は本当に小さくなりましたね。
李：そうですね。昔のは大きくて重かったですが、今のは軽くて持ちやすいです。
森：それに、機能も便利になりました。インターネットもできるし、写真もきれいです。
小野：でも、そのせいで目が疲れやすくなった気がします。
李：分かります。私もスマホを見る時間が長くて、目が悪くなりました。
森：それなら、あまり長く見ないほうがいいですよ。
小野：そうですね。たまには外で運動したほうがいいですね。
李：最近、近くの公園を走るようにしています。
森：いい習慣ですね。続けたほうがいいです。
小野：ところで、明日の天気はどうでしょう。
森：天気予報によると、曇りでしょう。
李：雨が降るかもしれませんね。傘を持って行ったほうがいいですよ。
小野：そうします。

後で
李：健康のために、食事も野菜を多くしたほうがいいですね。
小野：そうですね。私も最近、甘いものを少なくするようにしています。
森：お互い、気をつけましょう。

会話のキーポイント
「小さくなりました」 客观变化。

「便利になりました」 ナ形容詞变化。

「目が疲れやすくなった」 「～やすい」表示容易～（第20课后）。

「目が悪くなりました」 イ形容詞变化。

「見ないほうがいいです」 否定建议。

「走るようにしています」 「ようにしている」表示努力养成习惯（中高级）。

「続けたほうがいいです」 肯定建议。

「曇りでしょう」 推测。

「降るかもしれません」 可能性推测。

「野菜を多くしたほうがいい」 「多くする」人为使～变多。

ロールプレイ課題
课题1：两人一组，谈论最近自己或周围的变化（天气、科技、个人技能等），使用「～くなった」「～になった」。

课题2：针对对方的健康或学习问题，给出建议，使用「～たほうがいい」「～ないほうがいい」。

课题3：讨论明天的天气或未来的计划，使用「～でしょう」「～かもしれません」。

【作业】
1. 発音練習
朗读变化表达：小さくなる→小さくなりました、便利になる→便利になりました。

朗读建议：行ったほうがいいです、行かないほうがいいです。

朗读推测：晴れるでしょう、雨が降るかもしれません。

2. 活用ドリル
将下列句子改为「～くなる」「～になる」的过去形式：

寒い → 寒くなりました

便利 → 便利になりました

医者 → 医者になりました

静か → 静かになりました

将下列句子改为「～くする」「～にする」的意志形式：

音を（小さい）→ 音を小さくします

部屋を（きれい）→ 部屋をきれいにします

コーヒー（选择）→ コーヒーにします

3. 選択問題
Q1：携帯電話はとても＿＿＿なりました。
A. 小さく　B. 小さい　C. 小さな　D. 小さくて → A

Q2：息子は医者＿＿＿なりました。
A. に　B. が　C. を　D. で → A

Q3：健康のために、たばこを＿＿＿ほうがいいです。
A. 吸う　B. 吸った　C. 吸わない　D. 吸って → C（否定建议用ない形）

Q4：明日は雨が＿＿＿かもしれません。
A. 降る　B. 降り　C. 降って　D. 降った → A（辞書形）

4. 穴埋め問題
Q5：最近、日本語が＿＿＿なりました。（上手） → 上手に
Q6：テレビの音をもう少し＿＿＿してください。（大きい） → 大きく
Q7：無理をしない＿＿＿がいいです。 → ほう

5. 翻訳（日→中）
Q8：最近、生活が便利になりましたが、その反面、忙しくなりました。
→ 最近生活变方便了，但另一方面，变忙了。
Q9：明日、試験があるから、早く寝たほうがいいですよ。
→ 明天有考试，最好早点睡哦。

6. 翻訳（中→日）
Q10：请把空调温度调高一点。
→ エアコンの温度を高くしてください。
Q11：明天也许会下雪。
→ 明日は雪が降るかもしれません。

7. 間違い探し
Q12：私は医者になりました。
错误：无错误。但可确认主语是“我”时，成为医生是客观变化。句子正确。
Q13：たばこを吸ったほうがいいです。
错误：如果是建议戒烟，应使用否定「吸わないほうがいい」。但句子本身语法正确，只是语义不健康。如果是建议吸烟则奇怪。需根据语境。

8. 作文ミニタスク
課題：写5-6句话，描述你生活中最近发生的变化（至少一个客观变化、一个人为选择）、对未来的一个推测、以及一个健康建议。
要求：使用「～くなった」「～になった」「～する」「～でしょう」「～かもしれません」「～ほうがいい」。
例：
最近、忙しくなりました。でも、パソコンのスキルが上手になりました。健康のために、毎日早く寝ることにしました。明日は晴れるでしょう。でも、夕方から雨が降るかもしれません。傘を持って行ったほうがいいです。

9. 聴解問題（预留，需音频）
听录音，判断是客观变化还是主观改变。

听对话，回答说话人的建议和推测。

【复习与扩展】
