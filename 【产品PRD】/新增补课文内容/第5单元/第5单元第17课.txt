第17課：わたしは新しい洋服がほしいです
本课定位：第5单元（迎新春）的第一课。学习愿望表达「～がほしい」「～たい」，这是日常会话中非常高频的表达。注意对象用「が」标记，以及第三人称愿望的间接表达（「ほしがっている」「たがっている」）。同时对比「ほしい」（名词愿望）与「たい」（动作愿望）的区别。

発音ポイント
「ほしい」 声调②，注意「ほ」高，「し」低，「い」低。否定「ほしくない」中「く」轻读。

「ほかの」 中的「ほか」声调②，「ほ」高，「か」低。

「ガイドブック」 外来语，注意「ガイ」长音的感觉，「ド」和「ブ」连续。

「なかなか」 声调⓪，要读出节奏感。

語源メモ
「ほしい」 源自古典日语「欲し」，表示内心渴望的对象。

「たい」 是古日语「たし」的变体，接在动词连用形后表示愿望。

「～たがる」「～ほしがる 通过「がる」将感情客观化，用于第三人称。

活用予告
本课学习「ほしい」（イ形容词）和「たい」（动词愿望形）。「たい」的活用与イ形容词相同（～たい、～たくない、～たかった等）。第三人称愿望要用「ほしがっている」「たがっている」（第31课左右正式学，本课仅预告）。

【语法】
ノード1：～がほしい（名詞の願望）
句型：名詞 がほしいです
意义：想要某物（名词性愿望）。

例：

わたしは新しい洋服がほしいです。 → 我想要新衣服。

今、何がほしいですか。 → 现在想要什么？

新しいスマホがほしいです。 → 想要新手机。

否定形：～がほしくないです／～がほしくありません
過去形：～がほしかったです／～がほしかったです（丁寧）

例（否定）：

何もほしくないです。 → 什么都不想要。

注意：想要的东西用「が」标记，不用「を」。（与「好き」「嫌い」相同）

ノード2：～たい（動作の願望）
句型：动词ます形去ます ＋ たい
意义：想要做某事（动作性愿望）。

例：

日本へ行きたいです。 → 想去日本。

映画を見たいです。 → 想看电影。

何を食べたいですか。 → 想吃什么？

否定形：～たくないです／～たくありません
過去形：～たかったです／～たかったです

例（否定）：

今日は何もしたくないです。 → 今天什么都不想做。

助詞注意：

他动词的宾语通常用「が」也可以（～が食べたい），但传统语法用「を」也可。实际使用中两者都常见。「が」强调愿望对象，「を」中性。

例：コーヒーを飲みたい／コーヒーが飲みたい。

ノード3：第三人称愿望的表达（予告）
日语中不能直接说「彼は新しい車がほしいです」「彼は行きたいです」，因为推测他人内心情感不礼貌。应该用：

～たがっています（想要做，第三人称）

～ほしがっています（想要某物，第三人称）

例：

彼は新しい車をほしがっています。 → 他想要新车。

子供たちは遊びたがっています。 → 孩子们想玩。

注意：本课只做了解，正式学习在中级。考试不要求产出，但要能理解。

ノード4：～たいんです（気持ちの強調）
句型：～たい＋ん＋です
意义：比「～たいです」更强调愿望的理由或强烈的内心想法。

例：

日本へ行きたいんです。 → （因为某些原因）我想去日本。

どうしてその仕事がほしいんですか。 → 为什么想要那份工作？

对比：

「日本へ行きたいです」 → 单纯陈述事实

「日本へ行きたいんです」 → 说明理由或强烈意愿

【助詞カード】第17课：願望の対象「が」
句型	助詞	例
～がほしい	が	車がほしい
～たい（他動詞）	を／が	ラーメンを食べたい／ラーメンが食べたい
～たい（自動詞）	に	日本に行きたい
【対比表】ほしい vs たい
項目	ほしい	たい
願望対象	名詞（もの）	動作（すること）
例	車がほしい	車を買いたい
否定	ほしくない	たくない
過去	ほしかった	たかった
品詞	イ形容詞	イ形容詞型活用
【対比表】～たいんですが（前置き・依頼）
句型：～たいんですが、～
意义：先说自己的愿望，然后委婉引出请求或询问。

例：

日本へ行きたいんですが、おすすめの場所を教えてくださいませんか。 → 我想去日本，能推荐一下地方吗？

この本を読みたいんですが、貸してもらえますか。 → 我想读这本书，能借给我吗？

【敬語レベル表示】
「たい」「ほしい」本身是普通表达。对长辈或上级直接用「～たいですか」不礼貌，应用「～たいと思われますか」或避免直接问。本课不要求。

【会话】
基本课文（4句型）
わたしは新しい洋服がほしいです。

日本へ行きたいです。

今、何がほしいですか。
— 新しいスマホがほしいです。

何を食べたいですか。
— すしを食べたいです。

应用课文：欲しいものの話（谈论想要的东西）
场景：小李和小野午休时聊天，谈论各自想要的东西和想去的地方。

人物：

李秀麗（り しゅうれい）

小野緑（おの みどり）

对话正文：

小野：李さん、今、何かほしいものがありますか。
李：そうですね…新しいパソコンがほしいです。今使っているのはちょっと古くて、動作が遅いんです。
小野：ああ、わかります。私も最近、スマホを新しくしようかなと思っています。
李：どのスマホがほしいですか。
小野：カメラがきれいなのがほしいです。旅行の写真をよく撮るので。
李：そうですか。小野さんはどこかへ行きたいですか。
小野****：うーん、そうですね。今年の夏は北海道へ行きたいです。
李：北海道は涼しいですからね。
小野：そうなんです。それに、海鮮がとてもおいしいと聞きました。
李：いいですね。私も行ってみたいです。
小野：じゃあ、一緒に行きませんか。
李：ぜひ行きたいです。でも、まずはお金を貯めないと…
小野：それは私も同じです（笑）
李：ところで、小野さんは何か習いたいことはありますか。
小野：ピアノを習いたいです。子どものときから興味があって。
李：すごいですね。私は日本語をもっと勉強したい**です。
小野：李さんはもう十分上手ですよ。
李：いいえ、まだまだです。もっと話せるようになりたいです。

会話のキーポイント
「何かほしいものがありますか」 询问想要的东西。

「古くて」 て形表原因。

「～しようかなと思っています」 委婉表达自己的打算（正在考虑做～）。

「カメラがきれいなのがほしい」 「の」名词化，指代手机。

「北海道へ行きたいです」 动作愿望。

「行ってみたいです」 「～てみる」＋「たい」表示“想试试做～”。

「お金を貯めないと…」 「～ないと」省略「だめ」，表示“必须～”。

「習いたい」「勉強したい」 其他动词的愿望形。

ロールプレイ課題
课题1：两人一组，互相询问“现在想要什么”“想去哪里”“想做什么”，使用「～がほしい」「～たい」。

课题2：A告诉B自己想去某地旅游，并说明理由，B表示也想一起去。

课题3：谈论最近想买的东西（电子产品、衣服、书等），说明原因（旧的坏了、性能更好等）。

【作业】
1. 発音練習
朗读「ほしい」的各种形式：ほしい、ほしくない、ほしかった、ほしくなかった。

朗读「たい」的变化：行きたい、行きたくない、行きたかった、行きたくなかった。

句子：新しいスマホがほしいです。日本へ行きたいです。

2. 活用ドリル
将下列「たい」「ほしい」改为否定形和过去形：

ほしい → ほしくない、ほしかった

食べたい → 食べたくない、食べたかった

行きたい → 行きたくない、行きたかった

買いたい → 買いたくない、買いたかった

将下列愿望句改为第三人称（预告，用「ほしがっている」「たがっている」）：

妹は新しい服がほしいです。 → 妹は新しい服をほしがっています。

子供は遊びたいです。 → 子供は遊びたがっています。

3. 選択問題
Q1：わたしは新しい靴＿＿＿ほしいです。
A. を　B. が　C. に　D. で → B

Q2：日本へ＿＿＿です。
A. 行く　B. 行きます　C. 行きたい　D. 行った → C

Q3：第三者の願望を言うときは？
A. ほしい　B. ほしがっている　C. ほしかった　D. ほしいだ → B

Q4：コーヒー＿＿＿飲みたいです。
A. を　B. が　C. AもBも正しい　D. に → C

4. 穴埋め問題
Q5：私は時間＿＿＿ほしいです。 → が
Q6：何＿＿＿食べたいですか。 → を/が
Q7：子供たちは早く＿＿＿たがっています。（寝る） → 寝

5. 翻訳（日→中）
Q8：今、新しいパソコンがほしいんですが、お金が足りません。
→ 现在想要新电脑，但是钱不够。
Q9：いつか富士山に登りたいです。とても美しいと聞きました。
→ 想什么时候去爬富士山。听说非常美。

6. 翻訳（中→日）
Q10：你想吃什么？——想吃寿司。
→ 何を食べたいですか。 — すしを食べたいです。
Q11：小李想要一辆新车。
→ 李さんは新しい車をほしがっています。（第三人称用ほしがる）

7. 間違い探し
Q12：私は新しいスマホをほしいです。
错误：「ほしい」的对象应用「が」，不是「を」。
正解：私は新しいスマホがほしいです。
Q13：彼は日本へ行きたいです。
错误：第三人称愿望不能用「たい」，要用「たがっている」。
正解：彼は日本へ行きたがっています。

8. 作文ミニタスク
課題：写5-6句话，介绍你想要的三样东西、想做的两件事，以及想去的两个地方。至少说明一个理由。
要求：使用「～がほしい」「～たい」「～たいんです」。
例：
私は新しいカバンがほしいです。今使っているのは古くて、壊れかけています。それから、北海道へ行きたいです。海鮮が食べたいんです。温泉にも入りたいです。夏休みに旅行の計画を立てようと思っています。

9. 聴解問題（预留，需音频）
听录音，判断说话人想要什么（～がほしい）。

听对话，回答“对方想做什么”（～たい）。

【复习与扩展】
📖 これまでのまとめ
第17课的核心是愿望表达「～がほしい」「～たい」。注意对象用「が」标记，动词愿望形的活用与イ形容词相同。第三人称愿望要用「ほしがっている」「たがっている」。
跨课链接：

第11课：好恶表达（～が好き／嫌い）对象也用「が」，与本课一致。

第18课：变化表达「～くなる」「～にする」可与愿望结合（新しくなりたい）。

第20课：可能表达「～ことができる」可与「たい」结合（～たいことができる？可以想做就做吗？）。

第24课：「～たいんです」的强调用法进一步展开。
