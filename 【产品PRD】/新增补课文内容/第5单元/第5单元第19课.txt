第19課：部屋のかぎを忘れないでください
本课定位：第5单元（迎新春）的第三课。学习动词的「ない形」以及基于ない形的三种重要表达：禁止请求「～ないでください」、义务「～なければなりません」、允许不必要「～なくてもいいです」。这是日常交际中非常实用的表达，与第15课的许可「～てもいいです」形成完整对照体系。

発音ポイント
「ない形」 的变化：1类动词词尾变あ段＋ない（例：書く→書かない），2类动词去る＋ない（食べる→食べない），3类动词「する→しない」「来る→こない」。

「なければなりません」 口语常缩略为「なきゃなりません」或「なきゃ」。注意「なければ」的「なけれ」部分。

「なくてもいいです」 注意「なくて」的促音感。

「うっかり」 是拟态词，注意促音和「か」的轻读。

語源メモ
「ない形」 是动词的否定连用形，用于接续「ないで」「なければ」「なくて」等。

「～なければならない」 源于「～ない＋ければ＋ならない」，“如果不～就不行”。

「～なくてもいい」 源于「～なくても＋いい」，“即使不～也好”。

「～ないでください」 中的「ないで」是「ない」的连用形＋「て」。

活用予告
本课核心是动词ない形的构成，以及三种常用句型。ない形是学习否定命令、义务、许可不必要的基础。第20课将学习可能表达（～ことができる）和辞書形。

【语法】
ノード1：動詞のない形（作り方）
动词ない形是各种否定表达的基础。必须先掌握变化规则。

1グループ（五段動詞）：词尾う段→あ段＋ない

词尾	变化	例
う→わ	わない	買う→買わない
く→か	かない	書く→書かない
ぐ→が	がない	泳ぐ→泳がない
す→さ	さない	話す→話さない
つ→た	たない	待つ→待たない
ぬ→な	なない	死ぬ→死なない
ぶ→ば	ばない	呼ぶ→呼ばない
む→ま	まない	飲む→飲まない
る→ら	らない	帰る→帰らない（注意：帰る是1类）
2グループ（一段動詞）：去掉「る」＋ない
例：食べる→食べない、見る→見ない、起きる→起きない

3グループ：

する→しない

来る（くる）→来ない（こない）

特例：ある→ない（注意「あらない」是错误的！）

ノード2：～ないでください（禁止の依頼）
句型：動詞ない形＋でください
意义：请不要做某事（委婉的禁止请求）。

例：

部屋のかぎを忘れないでください。 → 请不要忘记房间钥匙。

ここでタバコを吸わないでください。 → 请不要在这里吸烟。

明日遅れないでくださいね。 → 请不要迟到哦。

对比：

「～てください」：请做～

「～ないでください」：请不要做～

注意：比「～てはいけません」更礼貌、更柔和。

ノード3：～なければなりません（義務）
句型：動詞ない形（去ない）＋なければなりません
意义：必须做某事（义务、必要）。

例：

明日、レポートを出さなければなりません。 → 明天必须交报告。

毎日、薬を飲まなければなりません。 → 每天必须吃药。

電車の中で電話をしてはいけませんが、しなければならないこともありますね。 → 电车里不能打电话，但也有必须打的时候。

口语缩略：

～なければならない → ～なきゃならない／～なきゃ

～なければなりません → ～なきゃなりません

例：もう行かなきゃ。 → 必须得走了。

ノード4：～なくてもいいです（許可・不必要）
句型：動詞ない形（去ない）＋なくてもいいです
意义：可以不～（不必要）。

例：

明日は来なくてもいいです。 → 明天可以不来。

全部書かなくてもいいです。 → 不用全写也可以。

高いものを買わなくてもいいです。 → 不用买贵的东西也行。

注意：这是「～てもいいです」的否定版，表示允许不做某事。

【助詞カード】第19课：ない形＋助詞
接続	意味	例
ないで＋ください	请不要	忘れないでください
なければ＋なりません	必须	行かなければなりません
なくても＋いいです	可以不	来なくてもいいです
【対比表】許可・禁止・義務の強さレベル
強さ	禁止	許可	義務（必須）	不必要
強い	～てはいけません	—	～なければならない	—
普通	～ないでください	～てもいいです	～なければなりません	～なくてもいいです
やわらかい	～ないほうがいい	～てもかまいません	～ほうがいい	～なくてもかまいません
カジュアル	～ちゃダメ	～ていいよ	～なきゃ	～なくていいよ
【対比表】「～ないで」vs「～なくて」
項目	～ないで	～なくて
用法	伴随状态、否定请求	原因、并列
例	何も食べないで寝た	食べなくてお腹が空いた
本课	～ないでください	（第20课以后接触）
【敬語レベル表示】
「～ないでください」已是丁寧体。对长辈可用「～ないでいただけますか」更礼貌。「～なければなりません」足够正式。

【会话】
基本课文（4句型）
部屋のかぎを忘れないでください。

ここで写真を撮らないでください。

明日、レポートを出さなければなりません。

明日は来なくてもいいです。

应用课文：注意する・される（提醒与被提醒）
场景：小李即将外出旅行，森和小野叮嘱她注意事项。

人物：

李秀麗（り しゅうれい）

小野緑（おの みどり）

森健太郎（もり けんたろう）

对话正文：

森：李さん、明日から旅行ですね。気をつけて行ってきてください。
李：はい、ありがとうございます。
小野：ところで、ホテルの部屋のかぎは忘れないでくださいね。
李：はい、気をつけます。
森：それから、貴重品はなくさないでください。
李：分かりました。
小野：あと、出かけるときは電気やエアコンを消すのを忘れないでください。
李：はい、しっかり確認します。
森：旅行中にレポートを出さなければなりませんか。
李：いいえ、今週中に出せばいいので、旅行から帰ってからで大丈夫です。
小野：じゃあ、急がなくてもいいですね。
李：そうです。ゆっくり楽しんできます。
森：そうそう、無理をしないでくださいね。体調を崩さないように。
李：はい、気をつけます。
小野：楽しい旅にしてください。お土産楽しみにしています。
李：ありがとうございます。何か買ってきてほしいものはありますか。
森：私は特にありません。気を使わないでください。
小野：そうそう、遠慮しないでくださいね。
李：分かりました。それでは、行ってきます。
森・小野：いってらっしゃい。

会話のキーポイント
「気をつけて行ってきてください」 「行ってらっしゃい」的丁寧请求形。

「忘れないでください」 禁止请求（钥匙）。

「なくさないでください」 禁止请求（贵重品）。

「消すのを忘れないでください」 「消すの」名词化（忘记关掉这件事）。

「出さなければなりませんか」 疑问句询问是否必须。

「急がなくてもいい」 可以不急。

「無理をしないでください」 请不要勉强。

「気をつけます」 我会注意的。

「気を使わないでください」 请不要客气/不要费心。

「遠慮しないでください」 请不要顾虑/请别客气。

ロールプレイ課題
课题1：两人一组，A要出远门或参加重要活动，B给出注意事项（不要忘带东西、不要迟到、必须做什么、可以不做什么）。使用「～ないでください」「～なければなりません」「～なくてもいいです」。

课题2：模拟工作场景，上司对下属下达任务（明天必须交报告、不要忘记会议等）。

课题3：朋友之间互相提醒（不要忘记买礼物、不必着急等）。

【作业】
1. 発音練習
朗读下列动词的ない形：

書く→書かない、話す→話さない、待つ→待たない、死ぬ→死なない、飲む→飲まない、帰る→帰らない

食べる→食べない、見る→見ない

する→しない、来る→来ない

ある→ない（特例）

朗读句子：忘れないでください。行かなければなりません。来なくてもいいです。

2. 活用ドリル
将下列动词改为ない形，并分别造一个「～ないでください」「～なければなりません」「～なくてもいいです」的句子（仅练习变化，不造句也可）：

書く → 書かない

食べる → 食べない

する → しない

来る → 来ない

ある → ない

3. 選択問題
Q1：ここでタバコを＿＿＿ください。
A. 吸う　B. 吸って　C. 吸わないで　D. 吸わない → C

Q2：明日、レポートを出さ＿＿＿なりません。
A. ない　B. なくて　C. なければ　D. なかった → C

Q3：今日は残業し＿＿＿いいですか。
A. ないで　B. なくても　C. なければ　D. ない → B

Q4：危ないから、ここに＿＿＿ないでください。
A. 入る　B. 入り　C. 入ら　D. 入って → C（入る→入らない）

4. 穴埋め問題
Q5：約束を＿＿＿ないでください。（忘れる） → 忘れ
Q6：明日までに申し込ま＿＿＿なりません。 → なければ
Q7：全部＿＿＿なくてもいいです。（書く） → 書か

5. 翻訳（日→中）
Q8：無理をしないでください。自分のペースでやってください。
→ 请不要勉强。请按自己的节奏做。
Q9：明日は早く起きなければなりませんから、今夜は早く寝てください。
→ 明天必须早起，所以今晚请早点睡。

6. 翻訳（中→日）
Q10：请不要在这里拍照。
→ ここで写真を撮らないでください。
Q11：明天可以不来。
→ 明日は来なくてもいいです。

7. 間違い探し
Q12：ここに車を止めないでください。
错误：无错误，句子正确。
Q13：毎日、薬を飲むなければなりません。
错误：应是「飲まなければなりません」（ない形去ない＋なければ）。
正解：毎日、薬を飲まなければなりません。

8. 作文ミニタスク
課題：写5-6句话，列出你在学校或公司必须遵守的规则（至少2条必须做的事、2条禁止做的事、1条可以不做的事）。
要求：使用「～なければなりません」「～ないでください」「～なくてもいいです」。
例：
学校の規則です。毎日、授業に遅れないでください。宿題を出さなければなりません。教室で携帯電話を使ってはいけません（也可用「使わないでください」）。テストの時、カンニングをしないでください。全部の漢字を覚えなくてもいいです。大切なのは努力することです。

9. 聴解問題（预留，需音频）
听录音，判断说话人说的是“禁止”“义务”还是“不必要”。

听对话，回答“对方被告知不要做什么”“必须做什么”。
