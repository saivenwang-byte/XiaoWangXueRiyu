発音ポイント
「飲みます」 的「み」母音无声化（轻微），但可先清晰发音。

「ときどき」 的「き」有促音化感觉，但实际没有促音，是「とき」+「どき」的重复，注意声调。

「あまり」 重音在「あ」后下降，后接否定时需注意。

「ぜんぜん」 的「ぜ」要清晰，不要发成「じぇんじぇん」。

語源メモ
「お茶」 的「お」是美化语（ご美化語），用于表示礼貌。

「いつも・よく・ときどき・あまり・ぜんぜん」 频率副词的程度差异：100%→80%→50%→20%→0%。

「一緒に」 来自「一绪」，表示“同一个状态”。

活用予告
本课出现的动词均为他动词（动作对象需用「を」）。大部分是1类动词（五段），2类动词（一段）有「食べる」「見る」等。3类动词有「する」及其复合词（掃除する）。

动词分类（本课只需先了解，第14课て形时系统学）：

1类：飲む、買う、撮る、書く、読む、聞く

2类：食べる、見る

3类：する、掃除する

【语法】
ノード1：～を（動作の対象）
句型：名 を 動（他動詞）
意义：表示动作的直接对象（宾语）。
例：

李さんは毎日コーヒーを飲みます。 → 小李每天喝咖啡。

私は映画を見ます。 → 我看电影。

手紙を書きました。 → 写了信。

注意：

「を」只用于他动词（需要宾语的动作），自动词（不需要宾语）不用。

自动词的第4课「あります・います」和第5课「起きます・寝ます」等不接「を」。

ノード2：頻度副詞
表示动作发生的频率，位置一般在动词前、主语后。

副詞	頻度	例句
いつも	100%	私はいつも７時に起きます。
よく	80%	よく映画を見ます。
ときどき	50%	ときどきテニスをします。
あまり～ない	20%	あまり飲みません。
ぜんぜん～ない	0%	ぜんぜん勉強しません。
注意：「あまり」「ぜんぜん」必须与否定形式呼应。

ノード3：～ませんか / ～ましょう（誘い・提案）
～ませんか：邀请对方一起做某事（疑问句，语气较委婉，给对方选择空间）。
～ましょう：提议一起做某事（肯定句，语气更主动，包含自己也做）。

例：

一緒に映画を見ませんか。 → 一起看电影好吗？

一緒に映画を見ましょう。 → 一起看电影吧。

回答：

肯定：はい、いいですね。／はい、行きましょう。

否定：すみません、ちょっと…（省略式拒绝）／いいえ、ちょっと…。

注意：「～ませんか」更礼貌，「～ましょう」更积极。

ノード4：～をください（請求）
句型：名 を ください
意义：请给我～（购物或请求）。

例：

コーヒーをください。 → 请给我咖啡。

この本をください。 → 请给我这本书。

注意：购物时也可以省略「を」，只说「コーヒーください」，但标准语法保留「を」。

【助詞カード】第7课：格助詞「を」
用法	説明	例文
动作对象	他动词的直接宾语	コーヒーを飲む
移动的起点/经过点	（第14课补充）	家を出る、橋を渡る
本课只学第一种。

【対比表】「～ませんか」vs「～ましょう」
項目	～ませんか	～ましょう
形态	疑问句	肯定句
语感	委婉邀请，尊重对方意愿	主动提议，包含自己
回答	可回答はい/いいえ	一般回答はい
例	行きませんか	行きましょう
【対比表】頻度副詞の位置
主语	频率	动词	中文
私は	いつも	コーヒーを飲みます	我总是喝咖啡。
私は	よく	映画を見ます	我常看电影。
私は	ときどき	テニスをします	我有时打网球。
私は	あまり	飲みません	我不太喝。
私は	ぜんぜん	飲みません	我完全不喝。
【敬語レベル表示】
「～ませんか」比「～ましょう」更礼貌。对长辈或上级建议时，用「～ませんか」更合适。

【会话】
基本课文（4句型）
李さんは毎日コーヒーを飲みます。

私は時々テニスをします。

一緒に映画を見ませんか。
— いいですね。

昨日、何を食べましたか。
— カレーを食べました。

应用课文：カフェで注文（在咖啡店点单）
场景：小李和小野在咖啡店休息，点饮料和点心。

人物：

李秀麗（り しゅうれい）

小野緑（おの みどり）

对话正文：

店員：いらっしゃいませ。ご注文はお決まりですか。
小野：すみません、ちょっとまだ… 李さん、何にしますか。
李：私はコーヒーにします。小野さんは？
小野：私は紅茶にします。
店員：コーヒーと紅茶ですね。
小野：はい。それと、ケーキもありますか。
店員：はい、あります。今日はチーズケーキとショートケーキです。
小野：じゃあ、チーズケーキを一つください。
李：私は結構です。
店員：かしこまりました。少々お待ちください。
（数分後）
小野：李さんはよくカフェへ来ますか。
李：いいえ、あまり来ません。小野さんは？
小野：私はときどき来ます。ここは静かでいいですね。
李：そうですね。また一緒に来ましょう。
小野：はい、ぜひ。

会話のキーポイント
「ご注文はお決まりですか」：店员常用的礼貌询问。“您决定好点单了吗？”

「何にしますか」：决定选什么时用「～にします」。“我要咖啡。”

「それと」：表示“还有”，用于追加点单。

「～を一つください」：数量词+「ください」，本课未完全展开，第13课学数量词。

「私は結構です」：礼貌拒绝，意为“我不需要了”。

「かしこまりました」：店员接受订单时的标准用语。

「少々お待ちください」：请稍等。

「よく～ますか」：询问频率。回答用频率副词。

「静かでいいですね」：形容动词「静か」的て形连接原因/状态，可理解为“安静，很好”。

「また一緒に来ましょう」：邀请对方将来一起再来。

ロールプレイ課題
课题1：两人一组，扮演顾客和店员，进行点单对话。顾客至少点两样东西，使用「～をください」「～にします」。

课题2：互相询问对方平时做什么（看电影、听音乐、运动等），使用频率副词。

课题3：邀请同桌一起去看电影，使用「～ませんか」和「～ましょう」。

【作业】
1. 発音練習
朗读带「を」的句子：コーヒーを飲みます。映画を見ます。

朗读频率副词：いつも、よく、ときどき、あまり、ぜんぜん。

练习邀约句子：一緒にテニスをしませんか。いいですね。行きましょう。

2. 活用ドリル
将下列动词改为「ます」「ません」「ました」「ませんでした」形式：

飲む → 飲みます、飲みません、飲みました、飲みませんでした

食べる → 食べます、食べません、食べました、食べませんでした

する → します、しません、しました、しませんでした

掃除する → 掃除します、掃除しません、掃除しました、掃除しませんでした

3. 選択問題
Q1：毎朝、牛乳＿＿＿飲みます。
A. を　B. が　C. に　D. で → A

Q2：私は＿＿＿映画を見ます。（週に１回くらい）
A. いつも　B. よく　C. ときどき　D. あまり → C

Q3：一緒に映画を見＿＿＿。
A. ます　B. ませんか　C. ました　D. ない → B

Q4：コーヒーを＿＿＿ください。
A. を　B. が　C. に　D. は → A（但日常省略可）

4. 穴埋め問題
Q5：私は日曜日＿＿＿テニスをします。 → に（时间点）
Q6：昨日、何＿＿＿食べましたか。 → を
Q7：私はあまりテレビ＿＿＿見ません。 → を

5. 翻訳（日→中）
Q8：李さんは毎日コーヒーを飲みます。
→ 小李每天喝咖啡。
Q9：一緒に京都へ行きませんか。
→ 一起去京都好吗？

6. 翻訳（中→日）
Q10：我有时看报纸。
→ 私はときどき新聞を読みます。
Q11：昨晚你喝了什么？——喝了啤酒。
→ 昨夜、何を飲みましたか。 — ビールを飲みました。

7. 間違い探し
Q12：私は毎日映画を見る。
错误：应用丁寧体「見ます」。
正解：私は毎日映画を見ます。
Q13：一緒に食べませんか。 — はい、食べません。
错误：肯定回答不能用否定形。
正解：はい、食べましょう。／はい、いいですね。

8. 作文ミニタスク
課題：写4-5句话，介绍你平日经常做的活动（至少3项），并询问对方是否一起做其中一项。
要求：使用「を」「よく/ときどき」「～ませんか」。
例：
私はよく音楽を聞きます。ときどき映画も見ます。週末はいつもテニスをします。一緒にテニスをしませんか。

9. 聴解問題（预留，需音频）
听录音，选择听到的频率副词。

听对话，判断“对方邀请做什么”，以及回答是肯定还是否定。
