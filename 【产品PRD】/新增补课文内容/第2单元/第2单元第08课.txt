発音ポイント
「あげます」「くれます」「もらいます」 三个授受动词的语调差异注意：「あげます」⓪，「くれます」②（「く」高），「もらいます」④。

「はし」 筷子，声调①。与「橋」（はし①/⓪地域差）区分，但初学者不必深究。

「プレゼント」 是外来语，注意长音「プレー」实际非长音，但日本人常读成「プレゼント」无长音。

「にもつ」 不在本课，但提及。

語源メモ
「あげる」 原意是“举起”，引申为“给对方”恩惠。

「もらう」 原意是“领取”，表示接受。

「くれる」 是“给”的敬体，但主语必须是给予者（长辈、别人）。

「日本語で」 的「で」表示手段，原来自「にて」。

活用予告
本课出现更多的他动词，以及重要的授受动词（あげる、くれる、もらう）。三类动词「電話する」是サ変。另外「書く」「貸す」「借りる」等是1类或2类，为后续て形做准备。

【语法】
ノード1：～で（手段・方法）
句型：名 で 動詞
意义：表示做某事所用的工具、语言、方法等。

例：

李さんは日本語で手紙を書きます。 → 小李用日语写信。

はしで食べます。 → 用筷子吃。

テープで音楽を聞きます。 → 用磁带听音乐。

注意：

第6课的「で」表示交通手段（電車で行く），本课扩展为语言、工具等。

与「を使って」意思相近，但「で」更简洁。

ノード2：あげます（授受・给予）
句型：私は（人）に（物）をあげます
意义：我给某人某物。（主语是给予者，对象用「に」）

例：

私は友達にプレゼントをあげました。 → 我给了朋友礼物。

田中さんは李さんに本をあげます。 → 田中给小李书。

注意：

不能对自己说「私は私にあげる」。

对长辈或上级不能用「あげる」，改用「差し上げる」（中高级）。

ノード3：くれます（授受・别人给我）
句型：人が（私に）物をくれます
意义：别人给我（或我方）某物。主语是给予者，对象（我）常省略。

例：

友達がプレゼントをくれました。 → 朋友给了我礼物。

先生が本をくれました。 → 老师给了我书。

注意：

「くれます」不能用于“我给他人”。

常省略「私に」，因为默认是“给我”。

ノード4：もらいます（授受・接受）
句型：私は（人）に／から（物）をもらいます
意义：我从某人那里得到某物。

例：

私は友達にプレゼントをもらいました。 → 我从朋友那里得到了礼物。

李さんは先生から本をもらいました。 → 小李从老师那里得到了书。

注意：

「に」和「から」都可，但「から」更强调来源。

视点：主语是接受者。

【助詞カード】第8课：格助詞「で」（手段・方法）

【対比表】授受动词一览
动词	方向	主语	对象	例句
あげる	我→别人	我/我方	に	私は友達にあげる。
くれる	别人→我	别人	（に）	友達がくれる。
もらう	我←别人	我	に/から	私は友達にもらう。
视点原则：日语授受必须以说话人视点为中心。不能混用。

【敬語レベル表示】
授受动词有敬语形式（差し上げる、くださる、いただく），但本课只学基本形。对长辈用「あげる」「くれる」「もらう」不礼貌。

【会话】
基本课文（4句型）
李さんは日本語で手紙を書きます。

私は妹にプレゼントをあげます。

友達から CD をもらいました。

先生がお菓子をくれました。

应用课文：プレゼントを渡す（送礼物）
场景：小李要回国了，她给同事们送礼物，也收到同事的礼物。

人物：

李秀麗（り しゅうれい）

小野緑（おの みどり）

森健太郎（もり けんたろう）

对话正文：

李：小野さん、これ、プレゼントです。中国のお菓子です。
小野：まあ、ありがとうございます。開けてもいいですか。
李：はい、どうぞ。
小野：わあ、きれいな箱ですね。何が入っていますか。
李：中にクッキーやキャンディなどが入っています。
小野：ありがとうございます。とても嬉しいです。
李：いいえ、どういたしまして。
森：李さん、先月はありがとうございました。私からもプレゼントです。
李：えっ、いいんですか。開けてもいいですか。
森：もちろん。
李：あっ、日本の文房具ですね。とても便利そうです。
森：どうぞ使ってください。
李：森さん、小野さん、本当にありがとうございました。

（後日、李からお礼の手紙を書くシーン）
李（ナレーション）：小野さんと森さんに、日本語でお礼の手紙を書きます。

小野：李さんから手紙が来ました。日本語で書いてありますね。
森：読みましょう。『お世話になりました。日本でとても楽しかったです。また会いましょう。』
小野：李さんは日本語が上手ですね。

会話のキーポイント
「これ、プレゼントです」 口语中常用逗号省略助词。

「開けてもいいですか」 询问能否打开，礼貌。

「きれいな箱ですね」 ナ形容詞「きれい」修饰名词用「な」。

「何が入っていますか」 「入っています」是「入る」的て形+いる，表示状态（里面放着）。

「～が入っています」 存在句的扩展。

「私からもプレゼントです」 「から」表示来源，与「あげる」语感不同，这里是“从我这边的礼物”。

「使ってください」 て形+ください，请求对方做某事（第14课正式学）。

「お世話になりました」 感谢照顾的固定表达，很重要。

「日本語で書いてあります」 て形+ある表示有意图的状态（第16课）。

「日本語が上手ですね」 表扬对方日语好。

ロールプレイ課題
课题1：两人一组，A给B送礼物（书、零食等），使用「これ、プレゼントです」「あげます」「開けてもいいですか」。

课题2：收到礼物后，表达感谢，并询问里面是什么。

课题3：写信（或明信片）给朋友，用日语写一句话，再念出来。

【作业】
1. 発音練習
朗读三个授受动词完整句子：

私は友達に本をあげます。

友達が私に本をくれました。

私は友達から本をもらいました。

注意「もらいます」的「ら」要清晰。

读句子「日本語で手紙を書きます」。

2. 活用ドリル
将下列动词改为「ます」「ません」「ました」「ませんでした」：

書く → 書きます、書きません、書きました、書きませんでした

教える → 教えます、教えません、教えました、教えませんでした

もらう → もらいます、もらいません、もらいました、もらいませんでした

あげる → あげます、あげません、あげました、あげませんでした

くれる → くれます、くれません、くれました、くれませんでした

電話する → 電話します、電話しません、電話しました、電話しませんでした

3. 選択問題
Q1：日本語＿＿＿話しましょう。
A. を　B. が　C. で　D. に → C（手段）

Q2：私は友だちに本を＿＿＿。
A. もらいました　B. あげました　C. くれました　D. くださいました → B

Q3：私は先生＿＿＿辞書をもらいました。
A. に　B. から　C. を　D. AもBも正しい → D

Q4：友達＿＿＿プレゼントをくれました。
A. は　B. に　C. が　D. の → C（主语用「が」）

4. 穴埋め問題
Q5：この手紙は日本語＿＿＿書きました。 → で
Q6：私は母に花を＿＿＿（あげる/もらう/くれる の適切な形）。 → あげました
Q7：昨日、李さん＿＿＿お菓子をもらいました。 → に／から

5. 翻訳（日→中）
Q8：私は日本から友達に中国のお茶をあげました。
→ 我从日本给朋友送了中国茶。
Q9：先生が私に本をくれました。とても嬉しかったです。
→ 老师给了我书。我很高兴。

6. 翻訳（中→日）
Q10：我用日语写信给母亲。
→ 私は日本語で母に手紙を書きます。
Q11：你从谁那里得到这个的？——从森先生那里。
→ 誰からそれをもらいましたか。 — 森さんからです。

7. 間違い探し
Q12：私は田中さんに本をくれました。
错误：「くれます」主语必须是给予者（别人），此处主语是我，应用「あげました」。
正解：私は田中さんに本をあげました。
Q13：友達が私にプレゼントをもらいました。
错误：「もらいます」主语是接受者，此处主语是友達，但动作是“给”，应改为「くれました」。
正解：友達が私にプレゼントをくれました。

8. 作文ミニタスク
課題：写一段话（4-5句），描述你和朋友之间交换礼物的经历（你给了什么，收到了什么，什么时候，用什么语言写卡片等）。
要求：使用「あげる」「くれる」「もらう」「～で」。
例：
先週、私は友達に中国の切手をあげました。友達は私に日本のマンガをくれました。嬉しかったです。お礼の手紙を日本語で書きました。

9. 聴解問題（预留，需音频）
听录音，判断对话中谁给了谁什么（画出授受方向）。

听录音，写出听到的「あげる/くれる/もらう」及其时态。

【复习与扩展】
📖 これまでのまとめ
第8课的核心是授受动词（あげる・くれる・もらう）和手段「で」的扩展（语言、工具）。授受方向是难点，需反复练习。
跨课链接：

第6课：交通手段「で」与此同形异义。

第7课：他动词「を」与授受动词连用（～をあげる）。

第24课：授受动词的て形（～てあげる/～てくれる/～てもらう），表示恩惠的授受。

第14课：て形用于请求（～てください）。
