第23課：休みの日、散歩したり買い物に行ったりします
本课定位：第6单元（再见日本）的第三课。学习动作的示例性列举「～たり～たりする」，条件表达「～場合」，不确定性疑问「～かどうか」，间接疑问句「疑問詞＋か」，以及变化·依据表达「～によって」。这些语法点在中级阶段会频繁出现，是本单元从中级过渡的重要桥梁。

発音ポイント
「～たり～たりする」 注意「たり」中的「た」和「り」要清晰，不能连成「たり」的模糊音。整个句型由两个以上「～たり」构成，最后加「する」。

「散歩」 读「さんぽ」，注意「んぽ」的鼻音+半浊音。

「場合」 读「ばあい」，注意「ば」和「あい」的长音感。

「かどうか」 中的「ど」要浊化清晰。

「によって」 中的「よっ」促音，「て」轻读。

語源メモ
「～たり」 来源于古代日语连用形「～て」+「あり」的反复动作表现，后固定为「たり」。

「場合」 原意“那个场合”，引申为条件性的情况。

「かどうか」 中的「どうか」是“怎样”，合起来表示“是否”。

「によって」 由「に」+「よる」（依靠） + 「て」构成。

活用予告
本课核心是「～たり～たりする」（基于た形），以及条件表达「場合」、间接疑问句。た形（第21课）和简体（第22课）是基础。本课内容完成后，第24课将学习引用、说明、意见表达等核心语法。

【语法】
ノード1：～たり～たりする（動作の例示）
句型：動詞た形 ＋ り、動詞た形 ＋ り する
意义：列举若干有代表性的动作，表示“有时做A，有时做B”或“做A啦B啦之类的”。不暗示全部动作，只是举例。

例：

休みの日、散歩したり買い物に行ったりします。 → 休息日，散散步、去买买东西之类的。

週末はいつも映画を見たり、本を読んだりして過ごします。 → 周末总是看看电影、读读书什么的。

昨日は友達と飲んだり、カラオケに行ったりしました。 → 昨天和朋友喝了酒，去了KTV之类的。

变化规则：

动词た形（買った・行った・食べた・した）＋り

最后一定要用「する」（过去时「した」，否定「しない」等）

形容词/名词也可用たり（了解即可）：

イ形容詞：高かったり、安かったり

ナ形容詞・名詞：静かだったり、学生だったり

ノード2：～場合（条件）
句型：名詞＋の／動詞簡体 ＋ 場合
意义：表示“在…的情况下”，常用于说明书、注意事项等书面语或正式口语。

例：

雨の場合は、試合は中止します。 → 下雨的情况下，比赛中止。

何か問題が起きた場合は、すぐに連絡してください。 → 如果发生了问题，请马上联系。

駐車する場合は、この駐車場を使ってください。 → 需要停车的情况下，请使用这个停车场。

注意：

相比「～たら」「～ば」，「場合」更客观、书面化，侧重于“事态情况”。

「の場合」用于名词，「～た場合」用于动词过去形表示假定。

ノード3：～かどうか（不確かな疑問）
句型：簡体文 ＋ かどうか
意义：表示“是否…”。用于将疑问句内容嵌入大句中（间接疑问）。

例：

明日、晴れるかどうか分かりません。 → 不知道明天是否放晴。

彼が来るかどうか聞いてみてください。 → 请去问一下他来不来。

この料理が美味しいかどうか、実際に食べてみないと分からない。 → 这个菜是否好吃，不实际尝一下不知道。

注意：

接续时使用简体形（动词/形容词/名词简体）。

名词/ナ形容词的肯定简体形为「～だ」，但「かどうか」前通常用「～か」也可以，但标准是简体句尾直接接「かどうか」。

例：学生かどうか（○）／学生だかどうか（△，但可用）

ノード4：疑問詞＋か（間接疑問）
句型：疑問詞 ＋ 簡体文（含疑問詞）＋ か
意义：将包含疑问词的疑问句嵌入大句中（间接疑问句）。

例：

鍵がどこにあるか教えてください。 → 请告诉我钥匙在哪里。

誰が来るか知っていますか。 → 你知道谁来吗？

なぜ彼が怒ったか説明できますか。 → 你能解释他为什么生气吗？

注意：

嵌入的疑问句句尾不能加「か」之外的语气（不能「どこにありますか」）。

间接疑问句是简体，不能用敬体。

ノード5：～によって（根拠・相違）
句型：名詞 ＋ によって
意义：表示“根据…（不同而不同）”或“因…而异”。

例：

航空券の値段は季節によって違います。 → 机票价格因季节而异。

人によって考え方が違う。 → 不同的人想法不同。

この規則は場合によって変わることがあります。 → 这条规则有时会根据情况改变。

注意：另外用法“由…造成”（火事によって焼けた），本课只学“因…而异”。

【助詞カード】第23课：例示・条件・疑問嵌入
接続	用法	例
た形＋り＋する	动作例示	行ったり来たりする
簡体＋場合	条件	雨の場合
簡体＋かどうか	是否	行くかどうか
疑問詞＋簡体＋か	间接疑问	どこにあるか
名＋によって	因…而异	季節によって
【対比表】例示：～たり～たりする vs ～や～など
項目	～たり～たりする	～や～など
接続	動詞/形容詞	名詞
例	飲んだり食べたりする	お茶やコーヒーなど
语感	动作/状态的例示	物品的例示
【対比表】条件：～場合 vs ～たら vs ～ば
表現	特徴	使用场景
～場合	书面、客观、郑重	说明书、规则
～たら	口语、强烈假设	日常会话
～ば	一般条件	书面、口语皆可
【対比表】間接疑問の語順
直接疑問	間接疑問
どこにありますか。	どこにあるか 分かりません。
彼は来ますか。	彼が来るかどうか 知りません。
誰が来ましたか。	誰が来たか 教えてください。
【敬語レベル表示】
「～場合」「～によって」本身是书面/正式用语，敬語度中。间接疑问句中用了简体，但整体句子可用敬体结尾（～ませんか、～ください）。

【会话】
基本课文（4句型）
休みの日は、散歩したり、買い物に行ったりします。

雨の場合は、運動会は中止します。

李さんが来るかどうか、知っていますか。
— いいえ、知りません。

鍵がどこにあるか、教えてください。

应用课文：送別会（欢送会）
场景：小李即将回国，公司同事为她举办欢送会。大家讨论送别会的活动安排以及周末的生活。

人物：

李秀麗（り しゅうれい）

小野緑（おの みどり）

森健太郎（もり けんたろう）

田中（たなか）

对话正文：

小野：今週の金曜日、李さんの送別会をします。みんな、何をしますか。
森：食事をしたり、話をしたり、写真を撮ったりしましょう。
田中：いいですね。カラオケも行ったりしますか。
小野：そうですね。時間があったら、行ってもいいですね。
李：みんな、ありがとうございます。とても楽しみです。

小野：ところで、当日の集合時間はどうしますか。
森：遅れる場合は、連絡するように言っておきます。
田中：もし雨の場合は、お店は大丈夫ですか。
小野：大丈夫です。全部屋内ですから。

李：小野さんは週末、いつも何をしていますか。
小野：そうですね。家でゆっくりしたり、買い物に行ったりしています。
李：森さんは？
森：私は友達と遊んだり、スポーツをしたりしています。
李：いいですね。日本では週末の過ごし方は人によって違いますね。
小野：そうですね。季節によっても変わります。夏はよく海へ行きます。

李：ところで、当日は誰が何を準備するか、もう決まりましたか。
小野：まだです。でも、森さんに聞いてみます。
森：私は飲み物を買うかどうかまだ分からない。みんなで決めましょう。
田中：じゃあ、アンケートを取ってみるのはどうですか。
小野：いいアイデアですね。それでは、早速準備を始めましょう。

会話のキーポイント
「～したり、～したりしましょう」 提议活动。

「遅れる場合は、連絡するように言っておきます」 条件表达+准备（～ておく）。

「人によって違います」 因人而异。

「季節によっても変わります」 也因季节变化。

「誰が何を準備するか、もう決まりましたか」 间接疑问（谁准备什么）+自动词决定。

「買うかどうかまだ分からない」 是否购买。

「アンケートを取ってみる」 尝试做。

「いいアイデアですね」 好主意。

「早速」 马上。

ロールプレイ課題
课题1：两人一组，谈论周末常做的事，使用「～たり～たりする」列举3-4种活动。

课题2：设定一个条件（下雨、迟到、生病等），讨论应对措施，使用「～場合」。

课题3：A问B关于某件事是否确定（会来吗？准备好了吗？），B用「かどうか分からない」回答。然后A问“谁做了什么”，用间接疑问句。

【作业】
1. 発音練習
朗读「～たり」的句子：飲んだり、食べたり、行ったり、来たり、見たり、読んだり。

朗读「～場合」的句子：雨の場合、遅れた場合、問題があった場合。

朗读间接疑问句：どこにあるか教えてください。誰が来るか知っていますか。

2. 活用ドリル
用「～たり～たりする」造句（将下列动作各选两个组合）：

例：散歩する／買い物する → 散歩したり、買い物に行ったりします。

読む／書く →

映画を見る／音楽を聞く →

掃除する／洗濯する →

用「～場合」改写下列句子：

雨が降る → 雨の場合

電車が遅れる → 電車が遅れた場合

パスポートを忘れる → パスポートを忘れた場合

将下列直接疑问句改为间接疑问句：

鍵はどこにありますか。 → 鍵がどこにあるか教えてください。

彼は来ますか。 → 彼が来るかどうか知りません。

3. 選択問題
Q1：休みの日は＿＿＿して過ごします。
A. 散歩する　B. 散歩し　C. 散歩したり　D. 散歩した → C

Q2：雨の＿＿＿は、運動会がありません。
A. とき　B. こと　C. 場合　D. の → C

Q3：昨日は映画を＿＿＿、買い物を＿＿＿しました。（見る・する）
A. 見たり／したり　B. 見て／して　C. 見た／した　D. 見る／する → A

Q4：李さんが来る＿＿＿分かりません。
A. か　B. かどうか　C. が　D. を → B

Q5：鍵が＿＿＿教えてください。
A. どこだ　B. どこにあるか　C. どこにありますか　D. どこです → B

4. 穴埋め問題
Q6：週末は友達と遊ん＿＿＿、映画を見た＿＿＿します。 → だり／り
Q7：駐車する＿＿＿は、この駐車場を使ってください。 → 場合
Q8：航空券の値段は季節＿＿＿違います。 → によって

5. 翻訳（日→中）
Q9：休みの日は、料理したり、ピアノを練習したりして過ごします。
→ 休息日做做饭、练练钢琴之类的。
Q10：もし事故があった場合は、すぐに警察に連絡してください。
→ 如果发生事故，请马上联系警察。

6. 翻訳（中→日）
Q11：周日你通常做什么？——看电影或读书。
→ 日曜日はたいてい何をしますか。 — 映画を見たり、本を読んだりします。
Q12：请告诉我这个汉字怎么读。
→ この漢字の読み方を教えてください。（或 この漢字が何と読むか教えてください。）

7. 間違い探し
Q13：雨の場合は、試合を中止する。
错误：无语法错误，但书面语气，如要更口语可用「～たら」。此处可接受。
Q14：彼が来るかどうかを知っていますか。
错误：「かどうか」后面不用「を」，直接用「かどうか知っていますか」。
正解：彼が来るかどうか知っていますか。

8. 作文ミニタスク
課題：写5-6句话，介绍你周末通常做什么（使用「～たり～たりする」），然后写一个条件情况下的应对措施（使用「～場合」），最后写一个你不知道别人是否来的疑问句（使用「～かどうか」）。
例：
週末は、友達と遊んだり、音楽を聴いたりして過ごします。雨の場合は、家で映画を見ることが多いです。次の週末にコンサートがあるんですけど、友達が来るかどうかまだ分かりません。誰が一緒に行くか、後で確認します。

9. 聴解問題（预留，需音频）
听录音，判断说话人列举了哪些动作（～たり～たり）。

听对话，回答“根据什么不同而变化（～によって）”。

【复习与扩展】
📖 これまでのまとめ
第23课学习了动作的示例性列举「～たり～たりする」，条件「～場合」，不确定疑问「～かどうか」，间接疑问句「疑問詞＋か」，以及依据变化「～によって」。这些语法使表达更加丰富，能应对更复杂的交际需求。
跨课链接：

第21课：た形是「～たり～たりする」的基础。

第22课：简体是间接疑问句和「かどうか」的基础。

第24课：「～と思う」「～と言う」需要接简体，与本课形成衔接。

中级：「～によって」还有被动、原因等多种用法。
