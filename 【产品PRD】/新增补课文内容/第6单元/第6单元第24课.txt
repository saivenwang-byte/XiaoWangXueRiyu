第24課：李さんはもうすぐ来ると思います
本课定位：第6单元（再见日本）的最后一课，也是《标日初级上册》的最终课。学习意见表达「～と思う」、引用「～と言う」、说明强调「～のです/～んです」、以及前置依頼「～んですが/～んだけど」。同时总结告别时的寒暄语。掌握本课后，学习者能表达自己的观点、引用他人话语、说明理由、委婉请求，为中级学习打下坚实基础。

発音ポイント
「～と思う」 中的「と」要清晰，接简体句尾。口语中常说「～と思うよ」。

「～と言う」 中的「と」同样重要，引用内容用简体。

「～のです」 口语缩略为「～んです」，注意「ん」的鼻音清晰。

「～んですが」 中的「ん」与「ですが」连读，用于铺垫请求。

「そろそろ」 是拟态词，表示“差不多该…” ，读时有节奏感。

「さようなら」 长音「よ」和「う」注意拉长，但现代口语常说「さよなら」或更短的「じゃね」。

語源メモ
「～と思う」 的「と」是引用助词，来源于古日语表示引用内容。

「～と言う」 中的「言う」是“说”，引用别人说的话。

「～のです」 是「の」＋「です」，名词化后加判断，表示说明/强调。

「お元気で」「お気をつけて」 中的「お」是美化语，告别时用于表达祝愿。

活用予告
本课是上册的总结。至此，学习者应掌握：名词句、动词句、形容词句、简体敬体转换、各类助词、て形、た形、ない形、辞書形、可能表达、经验表达、义务禁止、愿望、变化、推测、列举、条件、间接疑问、引用、意见表达、说明强调等。下册将进入更复杂的句型和敬语体系。

【语法】
ノード1：～と思う（意見・推量）
句型：簡体文 ＋ と思います
意义：表示说话人的想法、意见、推测。“我觉得…”“我想…”。

例：

李さんはもうすぐ来ると思います。 → 我想小李马上就来。

明日は晴れると思います。 → 我觉得明天会晴。

この料理は美味しいと思います。 → 我认为这道菜好吃。

彼は来ないと思います。 → 我觉得他不会来。

注意：

前面必须用简体句（普通形）。

否定意见时，否定在简体部分：来ないと思います（不是「来ると思いません」，但后者也偶见，不标准）。

第三人称的「～と思っている」表示“那个人认为”，可用于客观叙述。

ノード2：～と言う（引用）
句型：引用内容（简体）＋ と言います
意义：表示引用某人说的话、写的文字等。引用内容通常用简体，直接引用可用「」括起来。

例：

彼は「行きます」と言いました。 → 他说“我去”。（直接引用）

李さんは明日来ると言っていました。 → 小李说明天来。（间接引用）

先生は試験があると言いました。 → 老师说有考试。

注意：

引用内容用简体（包括「です/ます」也可，但通常用于直接引用的句尾）。

「と言っていました」表示转述某人说过的话。

口语中「って」代替「と」：彼は行くって言ってた。

ノード3：～のです／～んです（強調・説明）
句型：簡体文 ＋ のです（书面/正式）／んです（口语）
意义：用于说明原因、解释情况、强调事实、引出话题等。

例：

すみません、頭が痛いんです。 → 对不起，我头疼。（说明理由）

今日は早く帰りたいんです。 → 今天想早点回去。（强调愿望）

どうしたんですか。 → 怎么了？（询问原因）

注意：

名词/ナ形容词时：～なのです／～なんです（学生なんです）。

疑问句、否定句也可使用：行かないんです。

「～のですか」用于询问理由或要求说明。

ノード4：～んですが／～んだけど（前置き＋依頼）
句型：簡体文 ＋ んですが／んだけど
意义：用于引出话题、铺垫请求、委婉询问。相当于“那个…我想…”。

例：

東京タワーへ行きたいんですが、どうやって行きますか。 → 我想去东京塔，怎么去？

ちょっとお聞きしたいんですが、お時間よろしいですか。 → 我想问一下，您有时间吗？

パソコンが動かないんだけど、見てくれない？ → 电脑不动了，能帮我看一下吗？

注意：

在请求或提问前使用，使语气更委婉、自然。

对长辈用「～んですが」，对朋友用「～んだけど」。

【助詞カード】第24课：引用・説明・前置き
接続	用法	例
簡体＋と思う	意见	来ると思う
簡体＋と言う	引用	行くと言った
簡体＋のです/んです	说明/强调	病気なんです
簡体＋んですが	前置请求	聞きたいんですが
【対比表】「～と思う」vs「～でしょう」
項目	～と思う	～でしょう
主観性	高（个人意见）	低（一般推测）
使用场俥	日常意见表达	天气、新闻等
例	私は彼が来ると思う	明日は晴れるでしょう
【対比表】「～のです」のいろいろ
形	意味	例
～のです	说明/强调	そういうことなのです
～んですか	询问理由	どうしたんですか
～のですね	确认/感叹	そうなんですね
～のですよ	告知/强调	大丈夫なんですよ
【敬語レベル表示】
本课中，「～と思います」「～と言います」已经是丁寧体，可对长辈使用。请求铺垫「～んですが」足够礼貌。告别用「お元気で」「お気をつけて」也是敬语表达。

【会话】
基本课文（4句型）
李さんはもうすぐ来ると思います。

彼は「行きます」と言いました。

すみません、頭が痛いんです。

東京タワーへ行きたいんですが、どうやって行きますか。

应用课文：見送り（送行）
场景：在机场，小李结束研修即将回中国，小野和森前来送行。大家表达感谢、不舍、祝愿。

人物：

李秀麗（り しゅうれい）

小野緑（おの みどり）

森健太郎（もり けんたろう）

对话正文：

小野：李さん、もうすぐ出発の時間ですね。
李：はい。本当にありがとうございました。
森：こちらこそ。李さんのおかげで、楽しい時間を過ごせました。
李：私もです。皆さんにとても親切にしていただいて、感謝しています。
小野：李さんはまた日本に来ると思いますか。
李：ぜひ来たいと思います。そのときはまたよろしくお願いします。
森：待っていますよ。
小野：さっき、田中さんが「李さんによろしくと言っていました」。
李：そうですか。田中さんにもよろしくと伝えてください。

李：ところで、帰国したら何をしたいんですか。
小野：まずはゆっくり休みたいんです。
李：そうなんですね。
森：お土産、買いましたか。
李：はい、たくさん買ったんです。日本の友達にあげるつもりです。

アナウンス：「日本航空〇〇便のご搭乗は…」
小野：そろそろ時間ですね。
李：本当にありがとうございました。
森：お元気で。
小野：気をつけて帰ってください。
李：皆さんもお元気で。また会いましょう。
全員：さようなら。

（李が保安検査場へ向かう）
小野：李さん、頑張ってください！
李：ありがとう！日本での思い出は一生忘れません。

会話のキーポイント
「～していただいて」 敬语（～てもらう的谦让），表示“承蒙您…”。

「～と思います」 意见表达。

「～と言っていました」 转述他人话语。

「～と伝えてください」 请求转达。

「何をしたいんですか」 询问意愿+说明语气。

「ゆっくり休みたいんです」 说明理由。

「そうなんですね」 接受对方说明。

「たくさん買ったんです」 强调/说明。

「そろそろ時間ですね」 “差不多到时间了”。

「お元気で」「お気をつけて」「さようなら」 告别语。

「頑張ってください」 加油。

ロールプレイ課題
课题1：两人一组，A即将去旅行/回国，B送行。使用「～と思います」「～と言っていました」「～んです」「お元気で」等。

课题2：A向B询问某个地方怎么去，先用「～んですが」引出话题，再问具体路线。

课题3：互相说出自己对某部电影/某本书的看法，使用「～と思います」。

【作业】
1. 発音練習
朗读「～と思います」的句子：来ると思います。美味しいと思います。行かないと思います。

朗读「～と言います」：行くと言いました。来ると言っていました。

朗读「～んです」：どうしたんですか。頭が痛いんです。

朗读告别语：お元気で。お気をつけて。さようなら。

2. 活用ドリル
将下列简体句改为「～と思います」：

明日、晴れる → 明日、晴れると思います。

この本は面白い → この本は面白いと思います。

彼は来ない → 彼は来ないと思います。

ここは静かだ → ここは静かだと思います。

将下列直接引用改为间接引用（～と言う）：

「私は行きます。」 → 彼は行くと言いました。

「試験は難しいです。」 → 先生は試験が難しいと言いました。

「今日は疲れた。」 → 友達は今日は疲れたと言っていました。

将下列句子改为「～んです」说明理由：

なぜ遅れたの？ → 事故があったんです。

どうして来なかった？ → 病気だったんです。

3. 選択問題
Q1：明日は晴れる＿＿＿。
A. と思う　B. と思った　C. と言う　D. と言った → A

Q2：彼は「さようなら」＿＿＿言いました。
A. を　B. と　C. に　D. で → B

Q3：すみません、頭が痛い＿＿＿。
A. です　B. んだ　C. んです　D. ですか → C

Q4：先生は来週テストがある＿＿＿言っていました。
A. と　B. を　C. に　D. で → A

Q5：日本へ行きたい＿＿＿ですが、ビザはどうやって取りますか。
A. ん　B. の　C. こと　D. もの → A（〜んですが）

4. 穴埋め問題
Q6：もうすぐバスが来る＿＿＿思います。 → と
Q7：李さんは「また来たい」＿＿＿言っていました。 → と
Q8：どうして遅れた＿＿＿ですか。 — 事故があった＿＿＿です。 → ん、ん

5. 翻訳（日→中）
Q9：私は来年、日本へもう一度行きたいと思います。
→ 我想明年再去一次日本。
Q10：田中さんが「李さんにお土産をあげてください」と言っていました。
→ 田中先生说“请把礼物给小李”。

6. 翻訳（中→日）
Q11：我认为这道菜不太辣。
→ この料理はあまり辛くないと思います。
Q12：我想问一下，这附近有邮局吗？
→ ちょっとお聞きしたいんですが、この近くに郵便局はありますか。

7. 間違い探し
Q13：彼は来ると思います。
错误：无错误，但注意主语是第三人称时，用「～と思っている」也可以。本句没问题。
Q14：すみません、お腹が痛いんです、トイレはどこですか。
错误：「痛いんです」后应用句号或断句。分句连接应用「から」或「が」。但此处可视为口语中直接转换话题，可接受。严谨写法：すみません、お腹が痛いんです。トイレはどこですか。

8. 作文ミニタスク
課題：写一段话（5-6句），内容为：表达一个你对未来的看法（使用「～と思う」）、转述一个朋友说过的话（使用「～と言っていました」）、说明一个你现在的身体或情绪状况（使用「～んです」）、然后提出一个请求或询问（使用「～んですが」）。
例：
私は来週の旅行がとても楽しみだと思います。友達が「京都は紅葉がきれいだよ」と言っていました。今、ちょっと緊張しているんです。お土産を何を買ったらいいか、アドバイスしてほしいんですが、もしよろしければ教えてください。

9. 聴解問題（预留，需音频）
听录音，判断说话人的意见（～と思う）。

听录音，转述某人说的话（～と言っていた）。

听录音，回答“对方请求什么”（〜んですが）。
