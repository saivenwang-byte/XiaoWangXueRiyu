発音ポイント
「きれい」 虽然是「い」结尾，但是ナ形容词。注意发音「きれー」，长音感，不是「き・れ・い」分开。

「ゆうめい」 注意长音「ゆう」，不要读成「ゆめい」。

「おだいじに」 的「だい」有长音，「じ」母音稍轻。

「どんな」 与「どう」的区别：前者连体词后接名词，后者副词单独用。

語源メモ
「きれい」 来自汉语「綺麗」，原意是丝绸的美丽，后泛化。

「にぎやか」 拟态词，来源于「にぎにぎしい」，后变成「にぎやか」。

「お大事に」 是对病人的祝福，“请保重身体”。

「紅葉」 读「こうよう」指秋季红叶，也读「もみじ」指枫叶。

活用予告
本课系统学习ナ形容詞（二类形容词/形容動詞）的时态变化（现在/过去/否定），以及名词谓语句的过去式（～でした）。ナ形容詞在修饰名词时要用「な」，在第16课学て形时进一步扩展。

【语法】
ノード1：ナ形容詞（現在肯定・現在否定・過去肯定・過去否定）
ナ形容詞的词尾是「だ」（辞書形），但丁寧体用「です」结句。活用时本身不变，只变「です」部分。

時制	肯定	否定
現在	～です	～ではありません／～じゃないです
過去	～でした	～ではありませんでした／～じゃなかったです
例（有名）：

現在肯定：京都の紅葉は有名です。

現在否定：この町は有名ではありません／有名じゃないです。

過去肯定：昔はこの寺は有名でした。

過去否定：あの時はまだ有名ではありませんでした。

注意：

否定形式有两种：「ではありません」正式，「じゃないです」口语（更常说「じゃありません」）。

ナ形容詞没有イ形容詞那样的词尾变化，变化都在「です」上。

ノード2：～でした（名詞・ナ形容詞の過去）
句型：名詞／ナ形容詞語幹 でした
意义：表示过去的状态或过去的名词判断。

例：

昨日はいい天気でした。 → 昨天是好天气。

昔、ここは静かでした。 → 从前这里很安静。

彼は学生でした。 → 他曾是学生。

对比：イ形容詞的过去是「～かったです」，而名词/ナ形容詞是「～でした」。

ノード3：～にします（選択・決定）
句型：名詞 にします
意义：决定选择某物（点餐、购物等）。

例：

コーヒーにします。 → 我决定要咖啡。

旅行は京都にします。 → 旅行就定京都吧。

注意：本课只做了解，正式学习在第18课「～ことにします」。

ノード4：いかがですか（勧誘・提案の敬語）
句型：いかがですか／いかがでしょうか
意义：比「どうですか」更礼貌的询问方式。

例：

コーヒーはいかがですか。 → 咖啡怎么样？（招待客人）

ご意見はいかがでしょうか。 → 您的意见如何？

注意：「どうですか」对同级或朋友，「いかがですか」对长辈/顾客。

【助詞カード】第10课：ナ形容詞の連体修飾（予告）
ナ形容詞修饰名词时要用「な」：有名な寺、きれいな町。第16课正式学。

【対比表】イ形容詞 vs ナ形容詞（本课总结）
項目	イ形容詞	ナ形容詞
辞書形	辛い	有名だ
丁寧現在肯定	辛いです	有名です
丁寧現在否定	辛くないです／辛くありません	有名ではありません／じゃないです
丁寧過去肯定	辛かったです	有名でした
丁寧過去否定	辛くなかったです／辛くありませんでした	有名ではありませんでした
名詞修飾	辛い料理	有名な寺
見分け方（简易）：

词尾是「い」且不是「きれい」「嫌い」等例外 → イ形容詞

其他（包括「きれい」「嫌い」） → ナ形容詞

【対比表】「どうですか」vs「いかがですか」
項目	どうですか	いかがですか
敬語度	普通	丁寧
使用对象	朋友、同辈	长辈、顾客、上级
例	体調はどうですか	お加減はいかがですか
【敬語レベル表示】
「いかがですか」是「どうですか」的敬语。本课接触。

【会话】
基本课文（4句型）
京都の紅葉は有名です。

この町は静かではありません。

あの寺は有名でした。

昔、ここはにぎやかでした。

应用课文：観光地の感想（对观光地的感想）
场景：小李和小野从京都旅行回来，在办公室谈论感想。

人物：

李秀麗（り しゅうれい）

小野緑（おの みどり）

森健太郎（もり けんたろう）

对话正文：

森：李さん、京都はどうでしたか。
李：とてもよかったです。紅葉が本当にきれいでした。
森：そうですか。どこへ行きましたか。
李：清水寺と金閣寺へ行きました。
小野：清水寺はどうでしたか。
李：とてもすばらしかったです。でも、人がたくさんいて、ちょっとにぎやかすぎました。
森：金閣寺は？
李：金閣寺もきれいでした。池の中に建っていて、とても優雅でした。
小野：温泉には入りましたか。
李：はい、旅館でゆっくり温泉に入りました。とても気持ちよかったです。
森：それはよかったですね。
李：京都の料理もおいしかったです。特に湯豆腐が有名だと聞きました。
小野：食べましたか。
李：はい、いただきました。とてもあっさりしていて、ヘルシーでした。
森：李さんは歴史に興味がありますか。
李：はい、あります。日本の歴史はとても面白いです。
小野：じゃあ、今度は奈良もいいですね。
李：はい、ぜひ行きたいです。

会話のキーポイント
「京都はどうでしたか」 询问过去的经历，用「でした」。

「とてもよかったです」 「いい」的过去形「よかった」。

「紅葉が本当にきれいでした」 ナ形容詞过去形「きれいでした」。

「人がたくさんいて」 动词て形表示原因/状态，「いる」的て形「いて」。

「にぎやかすぎました」 「すぎる」表示过度。

「池の中に建っていて」 「建っている」是「建つ」的进行/状态（第15课）。

「とても気持ちよかったです」 「気持ちいい」的过去形。

「あっさりしていて」 「あっさりする」表示清淡，て形连接。

「歴史に興味があります」 「興味がある」表示有兴趣。

「ぜひ行きたいです」 强烈愿望。

ロールプレイ課題
课题1：两人一组，A扮演去过某地旅行的人，B询问当地情况（怎么样？去了哪里？吃了什么？），使用「どうでしたか」「～でした」「～かったです」。

课题2：介绍自己的家乡（或喜欢的城市），使用「～は有名です」「～は静かです」「～はきれいです」。

课题3：对某人表达关心（生病或劳累），使用「お大事に」「お元気ですか」。

【作业】
1. 発音練習
朗读ナ形容詞的四形式（以「有名」为例）：

有名です、有名ではありません、有名でした、有名ではありませんでした

朗读「きれい」的长音：きれいな町、きれいでした。

句子：京都の紅葉は有名です。とてもきれいでした。

2. 活用ドリル
将下列ナ形容詞改成「現在否定」「過去肯定」「過去否定」（丁寧形）：

有名 → 有名ではありません、有名でした、有名ではありませんでした

きれい → きれいではありません、きれいでした、きれいではありませんでした

静か → 静かではありません、静かでした、静かではありませんでした

にぎやか → にぎやかではありません、にぎやかでした、にぎやかではありませんでした

将下列名词改成过去肯定：

晴れ → 晴れでした

学生 → 学生でした

3. 選択問題
Q1：京都の紅葉は＿＿＿です。
A. 有名　B. 有名な　C. 有名に　D. 有名だ → A（終止形用「有名です」）

Q2：昨日はいい天気＿＿＿。
A. です　B. でした　C. ます　D. ました → B

Q3：＿＿＿な町ですね。
A. きれい　B. きれいだ　C. きれく　D. きれいに → A（ナ形容詞修饰名词用「な」，但此题是「な町」前填词干「きれい」）

Q4：この辺は＿＿＿ではありません。
A. 静か　B. 静かな　C. 静かに　D. 静かだ → A

4. 穴埋め問題
Q5：昔、この寺はとても＿＿＿でした。（有名） → 有名
Q6：あの人は＿＿＿じゃないです。（元気） → 元気
Q7：京都の町は＿＿＿です。（きれい） → きれい

5. 翻訳（日→中）
Q8：京都の紅葉はとても有名です。
→ 京都的红叶非常有名。
Q9：昔、ここはにぎやかな町でした。
→ 从前，这里是很热闹的城镇。

6. 翻訳（中→日）
Q10：那个寺庙不安静。
→ あの寺は静かではありません。
Q11：昨天天气怎么样？——很好。
→ 昨日の天気はどうでしたか。 — よかったです。

7. 間違い探し
Q12：この部屋はきれいかったです。
错误：ナ形容詞过去不能加「かった」。
正解：この部屋はきれいでした。
Q13：この町は静かなです。
错误：ナ形容詞终止形用「静かです」，修饰名词才是「静かな」。
正解：この町は静かです。

8. 作文ミニタスク
課題：写5-6句话，介绍你最近去过的一个地方（或你想去的地方），描述那里怎么样（景色、氛围、设施），使用イ形容詞和ナ形容詞的过去/现在形式。
要求：使用「～でした」「～かったです」「～くないです」「～じゃないです」。
例：
先週、箱根へ行きました。箱根はとても有名な観光地です。温泉がとても気持ちよかったです。紅葉はまだあまりきれいじゃなかったですが、景色はすばらしかったです。また行きたいです。

9. 聴解問題（预留，需音频）
听录音，判断是イ形容詞还是ナ形容詞，以及时态。

听对话，回答「京都はどうでしたか」「何がきれいでしたか」。
