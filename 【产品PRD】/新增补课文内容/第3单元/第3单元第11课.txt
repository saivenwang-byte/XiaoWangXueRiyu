発音ポイント
「すき」 和 「きらい」 声调注意：「すき」②（低高低），「きらい」⓪（平板但「ら」稍高）。

「じょうず」 ③型（低高低低），「へた」 ②型（高低低），区别开。

「にがて」 ⓪型（低高低低）。

「ピアノ」「バイオリン」 外来语，注意母音清晰。

語源メモ
「好き」 来自动词「好く」的名词化，对象用「が」是古日语残留。

「上手」「下手」 原是书法术语（上手的字、下手的字），后泛化为技能。

「得意」「苦手」 比「上手」「下手」更委婉、主观。“擅长/不擅长”但「得意」可用于自己，「上手」用于他人。

「どう」 询问方式或状态，「どんな」询问性质。

活用予告
本课出现的「すき」「きらい」「じょうず」「へた」「とくい」「にがて」都是ナ形容詞。它们的时态变化和名词相同（～です、～ではありません、～でした等）。第9、10课已经学过。

【语法】
ノード1：～が好き・嫌い（好悪の対象）
句型：名1 は 名2 が 好き／嫌い です
意义：名1喜欢/讨厌名2。名2是感情的对象，用「が」标记。

例：

小野さんは歌が好きです。 → 小野女士喜欢唱歌。

私は犬が嫌いです。 → 我讨厌狗。

森さんはお酒が好きじゃないです。 → 森先生不喜欢喝酒。

注意：

感情形容词（好き、嫌い、怖い等）的对象用「が」不用「を」。

「好き」的程度可用「大好き」「とても好き」「ちょっと好き」等。

ノード2：～が上手・下手（技能の評価）
句型：名1 は 名2 が 上手／下手 です
意义：名1擅长/不擅长名2。

例：

森さんはテニスが上手です。 → 森先生擅长网球。

私は料理が下手です。 → 我做饭不好。

李さんは日本語が上手ですね。 → 小李日语很好呢。

注意：

用「上手」评价他人是赞美；评价自己或家人时用「得意」（以免自夸）。

例：自己「私は料理が得意です」；别人「お母さんは料理が上手ですね」。

「下手」可用于自己或他人。

ノード3：疑問詞＋が（強調）
句型：疑問詞 が ～ですか
意义：询问“哪个/谁”时，用「が」提示新信息。

例：

どれがあなたの傘ですか。 → 哪个是你的伞？

誰が来ましたか。 → 谁来了？

注意：与本课的感情对象「が」不同，但都是焦点标记。

ノード4：どうですか（意見・感想を求める）
句型：～は どうですか
意义：询问对方的意见、感想或状态。可译为“……怎么样？”

例：

この料理はどうですか。 → 这个菜怎么样？

新しい仕事はどうですか。 → 新工作怎么样？

注意：对长辈用「いかがですか」（第10课）。

【助詞カード】第11课：格助詞「が」（感情・能力の対象）
用法	説明	例文
感情的对象	好き・嫌い等	猫が好きです
能力的对象	上手・下手等	テニスが上手です
愿望的对象	第17課「ほしい」	車がほしいです
可能的对象	第20課「できる」	ピアノが弾けます
【対比表】「上手」vs「得意」
項目	上手	得意
使用对象	他人（褒めるとき）	自己、他人（可）
主观/客观	客观评价	主观信心
自慢感	说自己会自夸	说自己没问题
例	李さんは上手ですね。	私は料理が得意です。
【対比表】「下手」vs「苦手」
項目	下手	苦手
意义	技术不好	不喜欢、不擅长（心理）
感情色彩	客观事实	包含回避感
例	絵が下手です	数学が苦手です
【敬語レベル表示】
本课无特殊敬语。但评价他人时「上手」已是礼貌用词。自己不用「上手」。

【会话】
基本课文（4句型）
小野さんは歌が好きです。

森さんはテニスが上手です。

私は料理が下手です。

李さんは日本語が上手ですね。

应用课文：趣味の話（谈论兴趣）
场景：小李和小野在办公室休息时间聊天，谈论各自的兴趣和擅长的事。

人物：

李秀麗（り しゅうれい）

小野緑（おの みどり）

对话正文：

小野：李さん、趣味は何ですか。
李：音楽を聞くことです。特に日本のポップスが好きです。
小野：そうですか。私は歌を歌うことが好きです。でも、あまり上手じゃありません。
**李：そんなことないですよ。小野さんの歌、上手だと思います。
小野：ありがとうございます。李さんは何か楽器ができますか。
李：いいえ、私はピアノが少しできますが、とても下手です。
小野：得意なことは何ですか。
**李：料理ですかね。中国料理なら、いくつか作れます。
小野：すごいですね。私は料理が苦手で、いつも簡単なものしか作りません。
李：小野さんはスポーツはどうですか。
小野：テニスが好きで、週に一回ぐらいやります。でも、あまり上手じゃありません。
**李****：私もテニスをやってみたいです。
小野：じゃあ、今度一緒にやりませんか。
李：はい、ぜひお願いします。
小野：楽しみにしています。

会話のキーポイント
「趣味は何ですか」 询问兴趣的标准问法。

「音楽を聞くことです」 动词短语名词化（～こと）表示“听音乐这件事”。第20课系统学。

「特に」 尤其，特别喜欢某类。

「歌を歌うことが好きです」 动词短语作主语。

「そんなことないですよ」 否定对方的谦虚（哪里哪里，没有那回事）。

「上手だと思います」 「～と思う」表示认为（第24课）。

「何か楽器ができますか」 「できる」表示会做。

「少しできます」 会一点。

「得意なことは何ですか」 询问擅长的事。

「いくつか作れます」 能做一些（可能形第20课）。

「～が苦手で」 て形表示原因或并列。

「週に一回ぐらい」 频率表达（第13课）。

「楽しみにしています」 期待。

ロールプレイ課題
课题1：两人一组，互相询问兴趣（趣味は何ですか）和擅长的技能（何が得意ですか），使用「～が好きです」「～が上手です」「～が苦手です」。

课题2：向对方推荐一项运动或爱好，邀请一起做（～ませんか）。

课题3：介绍自己最喜欢的明星或作品（歌手、运动员等），说明为什么喜欢。

【作业】
1. 発音練習
朗读「好き」「嫌い」「上手」「下手」「得意」「苦手」的声调。

句子：小野さんは歌が好きです。森さんはテニスが上手です。

注意「上手じゃありません」的口语连读。

2. 活用ドリル
将下列ナ形容詞改成現在否定、過去肯定、過去否定：

好き → 好きではありません／じゃないです、好きでした、好きではありませんでした

上手 → 上手ではありません、上手でした、上手ではありませんでした

得意 → 得意ではありません、得意でした、得意ではありませんでした

3. 選択問題
Q1：わたしは猫＿＿＿好きです。
A. を　B. が　C. に　D. で → B

Q2：李さんは料理＿＿＿上手ですね。
A. を　B. が　C. に　D. で → B

Q3：自分のことを言うときは？
A. 上手です　B. 得意です　C. 上手いです　D. 上手でした → B

Q4：私はピアノが＿＿＿。
A. 下手　B. 苦手　C. どちらも可（意味違い） → C（根据语境，但两者都正确）

4. 穴埋め問題
Q5：私は犬が＿＿＿です。 → 嫌い
Q6：小野さんは英語が＿＿＿じゃないです。 → 上手
Q7：趣味は＿＿＿です。（読書） → 読書

5. 翻訳（日→中）
Q8：私はテニスが好きですが、あまり上手じゃありません。
→ 我喜欢网球，但不太擅长。
Q9：森さんはお酒が大好きです。
→ 森先生非常喜欢喝酒。

6. 翻訳（中→日）
Q10：小李擅长日语吗？——是的，非常擅长。
→ 李さんは日本語が上手ですか。 — はい、とても上手です。
Q11：我讨厌早起。
→ 私は早起きが嫌いです。

7. 間違い探し
Q12：私は料理を上手です。
错误：对象应用「が」，不是「を」。
正解：私は料理が上手です。
Q13：この本はとても好きでした。
错误：无错误，但注意「好きでした」过去式用于过去喜欢的状态。可用于回忆。句子正确。

8. 作文ミニタスク
課題：写5-6句话，介绍你的兴趣、你擅长的事情和不擅长的事情，并邀请朋友一起做某件事。
要求：使用「～が好き」「～が上手/得意」「～が下手/苦手」「～ませんか」。
例：
私の趣味は映画を見ることです。特にアニメが好きです。私は料理が得意です。でも、スポーツはあまり上手じゃありません。週末、一緒に映画を見ませんか。

9. 聴解問題（预留，需音频）
听录音，判断说话人对某事物的好恶（好き／嫌い／上手／下手）。

听对话，回答「趣味は何ですか」「何が得意ですか」。

【复习与扩展】
📖 これまでのまとめ
第11课的核心是感情和能力评价的表达：好恶（好き・嫌い）和能力评价（上手・下手・得意・苦手），它们的对象都用「が」。注意「上手」与「得意」的使用场合区别。
跨课链接：

第9、10课：ナ形容詞的活用（～です、～でした等），本课是具体应用。

第17课：愿望表达「～がほしい」「～たい」，对象也用「が」。

第20课：可能表达「～ことができる」「～ができる」，同样用「が」。
