発音ポイント
イ形容詞 的声调：多数为⓪或②型，注意「おいしい」③型、「あつい」②型。

「しょっぱい」 含有促音，注意停顿。

「すっぱい」 同样促音。

「いい」 的活用变化：否定「よくない」、过去「よかった」等特殊。

「とても」 与「すごく」的区别：「とても」书面一些，「すごく」更口语。

語源メモ
「辛い」 有两个读音：「からい」（辣）和「つらい」（痛苦），汉字相同但意思不同。

「おいしい」 据说源自「お～（美化語）」+「いし」（美味），语言学家有争议，但记住是常用形容词。

「便利」 是ナ形容詞，虽然词尾不是「い」，但有“方便”的意思。

活用予告
本课首次系统学习イ形容词（一类形容词）的活用（现在肯定/否定/过去/过去否定）。第10课学ナ形容词（二类形容词）。第16课学形容词て形。

【语法】
ノード1：イ形容詞（現在肯定・現在否定・過去・過去否定）
イ形容詞的词尾都是「い」，活用时去掉「い」加后续。

時制	肯定	否定
現在	～いです	～くないです／～くありません
過去	～かったです	～くなかったです／～くありませんでした
例（辛い）：

現在肯定：四川料理は辛いです。

現在否定：この料理は辛くないです／辛くありません。

過去肯定：昨日のカレーは辛かったです。

過去否定：このスープは辛くなかったです／辛くありませんでした。

注意：

「いい」的特殊变化：否定「よくないです／よくありません」、過去「よかったです」、過去否定「よくなかったです／よくありませんでした」。

否定有两种形式：「～くないです」更口语，「～くありません」更正式。

ノード2：あまり～ない（程度の否定）
句型：あまり ＋ イ形容詞(否定形)
意义：“不太～”，表示程度不高。

例：

この料理はあまり辛くないです。 → 这个菜不太辣。

彼はあまり優しくないです。 → 他不太温柔。

注意：「あまり」必须与否定呼应，不能单独用（除了极少数情况「あまりにも」等，但初学不涉及）。

ノード3：～が、～（逆接の前置き）
句型：句子A が、句子B
意义：表示转折，相当于“虽然…但是…”。也可以用于铺垫（不翻译）。

例：

この料理は辛いが、美味しいです。 → 这个菜虽然辣，但是好吃。

すみませんが、お名前は何ですか。 → 不好意思，请问您叫什么名字？（铺垫）

注意：口语中常省略「ですが」，说「すみません、お名前は？」。但书面或正式对话保留。

ノード4：～でしょう（推量・確認）
句型：名詞／形容詞 ＋ でしょう
意义：表示推测或确认（～吧）。相当于「～だろう」的丁寧体。

例：

明日はいい天気でしょう。 → 明天是好天气吧。

この料理は辛いでしょう。 → 这个菜辣吧。

注意：本课只做了解，第18课正式学推量。

【助詞カード】第9课：イ形容詞の活用助動詞（です・ありません）
形式	説明	例
～いです	現在肯定	辛いです
～くないです	現在否定（口語）	辛くないです
～くありません	現在否定（正式）	辛くありません
～かったです	過去肯定	辛かったです
～くなかったです	過去否定（口語）	辛くなかったです
～くありませんでした	過去否定（正式）	辛くありませんでした
【対比表】イ形容詞 vs ナ形容詞（予告）
項目	イ形容詞	ナ形容詞（第10課）
詞尾	い	だ（辞書形）
現在肯定	辛いです	便利です

修飾名詞	辛い料理	便利な道具
【対比表】「とても」vs「あまり」
副詞	意味	後続	例

【敬語レベル表示】
イ形容詞本身无敬语，但「です」已经是丁寧体。对长辈不用简体「辛い」。

【会话】
基本课文（4句型）
四川料理は辛いです。

このスープはあまり辛くないです。

このりんごは甘くておいしいです。（て形第16課）

昨日は寒かったです。

应用课文：料理の感想（对菜品的感想）
场景：小李和小野在四川餐厅吃饭，边吃边聊。

人物：

李秀麗（り しゅうれい）

小野緑（おの みどり）

对话正文：

小野：李さん、この四川料理はどうですか。
李：とても辛いですが、美味しいです。
小野：そうですか。私はちょっと辛すぎます。
李：じゃあ、これはどうですか。麻婆豆腐です。
小野：これも辛いですか。
李：いいえ、あまり辛くないです。ちょっと甘いです。
小野：そうですか。いただきます。…本当ですね。甘くておいしいです。
李：小野さんは辛いものが好きですか。
小野：いいえ、あまり好きじゃないです。李さんは？
李：私は大好きです。でも、あまり辛すぎるのはちょっと…
小野：そうですね。ちょうどいい味が一番ですね。
李：その通りです。
小野：今度、私も四川料理を作ってみたいです。
李：ぜひ作ってください。楽しみにしています。

会話のキーポイント
「どうですか」 询问对某事物的印象或状态。

「辛いですが、美味しいです」 转折句「が」的使用。

「辛すぎます」 「すぎる」表示“过于～”，接形容词词干（第11课左右会学）。

「麻婆豆腐」 （マーボードウフ）常见中国菜名。

「いただきます」 吃饭前的寒暄，表示“那我开动了”。

「辛いものが好きですか」 「もの」表示“东西”，指辣的东西。

「あまり好きじゃないです」 「じゃないです」是「ではないです」的口语否定。

「大好きです」 表示“非常喜欢”，比「好き」程度高。

「ちょうどいい味が一番ですね」 “恰到好处的味道最好呢”。

「楽しみにしています」 “我期待着”，常用表达。

ロールプレイ課題
课题1：两人一组，拿出某种食物（图片或想象），用イ形容词描述其味道（辣、甜、咸等），并使用「とても」「あまり～ない」。

课题2：评价某次就餐经历（餐厅的菜怎么样，环境如何），使用「～が、～」「～かったです」。

课题3：介绍自己最喜欢的食物和不太喜欢的食物，给出理由。

【作业】
1. 発音練習
朗读イ形容词的四种形式（以「辛い」为例）：

辛いです、辛くないです、辛かったです、辛くなかったです

朗读句子：「この料理はとても辛いです。でも、美味しいです。」

注意「おいしい」的声调③。

2. 活用ドリル
将下列イ形容词改成「現在否定」「過去肯定」「過去否定」：

辛い → 辛くない（です）、辛かった（です）、辛くなかった（です）

おいしい → おいしくない、おいしかった、おいしくなかった

高い → 高くない、高かった、高くなかった

いい → よくない、よかった、よくなかった（注意「いい」的特殊变化）

3. 選択問題
Q1：この料理は＿＿＿です。
A. 辛い　B. 辛く　C. 辛　D. 辛さ → A

Q2：このスープは＿＿＿熱くないです。
A. とても　B. あまり　C. いつも　D. よく → B

Q3：昨日のテストは＿＿＿です。（難しかった）
A. 難しい　B. 難しくない　C. 難しかった　D. 難しいでした → C

Q4：このりんごは甘い＿＿＿、おいしいです。
A. から　B. が　C. で　D. に → B

4. 穴埋め問題
Q5：このカレーは＿＿＿辛くないです。 → あまり
Q6：富士山は＿＿＿ですね。（高い） → 高い
Q7：昨日の天気は＿＿＿です。（寒かった） → 寒かった

5. 翻訳（日→中）
Q8：四川料理は辛いですが、とても美味しいです。
→ 四川菜虽然辣，但很好吃。
Q9：このスープはあまり熱くないです。
→ 这个汤不太热。

6. 翻訳（中→日）
Q10：这件衣服很贵。
→ この服は高いです。
Q11：昨天的电影有趣吗？——不，没意思。
→ 昨日の映画は面白かったですか。 — いいえ、面白くなかったです。

7. 間違い探し
Q12：この部屋は広いでした。
错误：イ形容詞过去式不能直接加「でした」。
正解：この部屋は広かったです。
Q13：私はあまりテレビを見ます。
错误：「あまり」必须与否定呼应。
正解：私はあまりテレビを見ません。

8. 作文ミニタスク
課題：写4-5句话，描述你吃过的一道菜（味道如何，价格如何，是否喜欢）。使用「とても」「あまり～ない」「～が、～」。
例：
昨日、四川料理を食べました。とても辛かったです。でも、美味しかったです。値段はあまり高くなかったです。また食べたいです。

9. 聴解問題（预留，需音频）
听录音，选择听到的形容词及其时态。

听对话，判断说话人对食物的评价（肯定/否定、程度）。
