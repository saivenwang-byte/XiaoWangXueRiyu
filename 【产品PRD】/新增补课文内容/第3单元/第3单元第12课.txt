発音ポイント
「せがたかい」 是「背が高い」的连语，注意「が」的鼻浊音倾向（可不强调）。

「おおい」 与 「おおきい」 的区别：前者是数量多，后者是体积大。

「どちら」 的「ち」清音、「ら」稍卷舌但日语不用卷舌，注意发音不要太重。

「やっぱり」 口语中常说「やっぱ」，注意促音。

語源メモ
「より」 源自古典日语，表示比较的基准。

「一番」 源自「一」+「番」（顺序），意为“第一、最”。

「大きな」「小さな」 是连体词，只能修饰名词，不能作谓语。区别于「大きい」「小さい」。

活用予告
本课主要使用イ形容詞（若い、古い、新しい等）进行比较。比较句型本身不改变形容词活用，只需掌握「より」「のほうが」「一番」的用法。

【语法】
ノード1：比較表現「～より」（比较基准）
句型：A は B より 形容詞（肯定）
意义：A比B更～。B是比较的基准。

例：

李さんは森さんより若いです。 → 小李比森先生年轻。

中国は日本より広いです。 → 中国比日本辽阔。

今日は昨日より暑いです。 → 今天比昨天热。

注意：「より」前面的名词是比较的基准，形容词必须用肯定形式。

ノード2：比較表現「～のほうが」（选择偏好）
句型：A のほうが B より 形容詞
意义：比起B，A更～（强调A更优选）。「のほうが」可以单独使用（省略B）。

例：

日本より中国のほうが広いです。 → 比起日本，中国更辽阔。

紅茶よりコーヒーのほうが好きです。 → 比起红茶，更喜欢咖啡。

どちらのほうが安いですか。 → 哪一个更便宜？

注意：「のほうが」中的「が」不能省略。单独问“哪个更好”时用「どちらが～ですか」。

ノード3：最上級「一番」（いちばん）
句型：＜範囲＞ で A が一番 形容詞
意义：在某个范围内，A是最～的。

例：

クラスで李さんが一番背が高いです。 → 班里小李个子最高。

日本料理の中で、すしが一番好きです。 → 在日本料理中，最喜欢寿司。

この中でどれが一番安いですか。 → 这里面哪个最便宜？

注意：范围用「で」表示；比较的主体用「が」提示。

ノード4：同じ・違う（同一・不同）
句型：A は B と同じです／と違います
意义：A和B相同/不同。

例：

私の車は森さんのと同じです。 → 我的车和森先生的一样。

この料理はあの料理と違います。 → 这个菜和那个菜不同。

注意：「同じ」是ナ形容詞，但修饰名词时用「同じ＋名詞」，不加「な」（特殊）。

【助詞カード】第12课：比較の助詞「より」「で」
助詞	用法	例
より	比较基准（比～）	私より若い人
で	比较范围（在～之中）	クラスで一番背が高い
が	比较主体	李さんが一番若い
【対比表】「より」vs「のほうが」
項目	より	のほうが
强调点	基准（B）	优选（A）
语序	AはBより～	AのほうがBより～
省略可能	B可省略？ 不可	可单独使用（どちらのほうが）
例	私より若い	私のほうが若い
【対比表】「大きい」vs「大きな」
項目	大きい	大きな
词类	イ形容詞	連体詞
作谓语	可（この部屋は大きい）	不可
修饰名词	可（大きい部屋）	可（大きな部屋）
语感	客观大小	主观感觉（更大的）
例	大きい犬（体型大的狗）	大きな犬（给人的感觉很大）
【敬語レベル表示】
比较表达本身无敬语，但询问对方意见时可用「どちらが～ですか」或更礼貌的「どちらが～でございますか」（不要求）。

【会话】
基本课文（4句型）
李さんは森さんより若いです。

中国は日本より広いです。

クラスで李さんが一番背が高いです。

日本料理の中で、すしが一番好きです。

应用课文：どっちがいい？（哪个更好？）
场景：小李和小野在百货商店挑选礼物，比较两种商品。

人物：

李秀麗（り しゅうれい）

小野緑（おの みどり）

对话正文：

小野：李さん、これとこれとどっちがいいですか。
李：そうですね…値段はどちらも同じぐらいですか。
小野：いいえ、左の方がちょっと安いです。右は少し高いです。
李：じゃあ、デザインはどうですか。
小野：私は右の方が好きです。色がきれいですから。
李：私も右の方がいいと思います。でも、少し大きすぎますね。
小野：そうですね。でも、実用的には右の方が使いやすいです。
李：じゃあ、右のにしましょうか。
小野：はい。小野さんも「左の方が軽い」と言っていましたが、私はやっぱり右がいいです。
李：小野さんは右の方が好みなんですね。
小野：はい。じゃあ、これにします。
李：プレゼントですから、もらう人が喜ぶ方が一番ですよね。

会話のキーポイント
「これとこれとどっちがいいですか」 询问两个物品哪个更好。「どっち」是「どちら」的口语。

「どちらも同じぐらい」 两者差不多。

「左の方がちょっと安い」 比较句。

「右の方が好きです」 比较喜好。

「～から」 表示原因（后置）。

「私も右の方がいいと思います」 「～と思う」表示意见。

「大きすぎます」 「～すぎる」表示过度。

「実用的には」 从实用角度来说。

「～にしましょうか」 提议决定。

「やっぱり」 果然，还是。

「～の方が好みなんです」 「好み」是名词，“偏好”。「なんです」是「なのです」的口语，强调说明。

「もらう人が喜ぶ方が一番です」 最高级「一番」＋抽象事情。

ロールプレイ課題
课题1：两人一组，拿出两个实际物品（手机、书、饮料等），比较价格、大小、颜色、喜好等，使用「～より」「～のほうが」「どちらが～」。

课题2：询问对方“班里谁个子最高”“哪种运动最受欢迎”，使用「～で一番～」。

课题3：介绍自己所在城市与其他城市的不同（气候、物价等），使用「～と同じ」「～と違う」。

【作业】
1. 発音練習
朗读比较句：李さんは森さんより若いです。中国は日本より広いです。

朗读最上级：クラスで李さんが一番背が高いです。

注意「どっち」的促音。

2. 活用ドリル
将下列形容词改为比较级句子（使用「より」）：

若い（私／あなた）→ 私はあなたより若いです。

高い（これ／それ）→ これはそれより高いです。

面白い（この本／あの本）→ この本はあの本より面白いです。

将下列句子改为最上级：

クラス／高い／森さん → クラスで森さんが一番高いです。

果物／好き／いちご → 果物の中で、いちごが一番好きです。

3. 選択問題
Q1：李さんは森さん＿＿＿若いです。
A. より　B. ほう　C. が　D. は → A

Q2：日本より中国の＿＿＿広いです。
A. ほう　B. ほうが　C. こと　D. もの → B

Q3：クラスで李さんが＿＿＿背が高いです。
A. 一番　B. とても　C. あまり　D. よく → A

Q4：夏より冬のほう＿＿＿好きです。
A. を　B. が　C. に　D. で → B

4. 穴埋め問題
Q5：この中で、どれ＿＿＿一番安いですか。 → が
Q6：私の車は森さん＿＿＿同じです。 → のと／と（「と同じ」）
Q7：昨日＿＿＿今日のほうが暑いです。 → より

5. 翻訳（日→中）
Q8：李さんは森さんより若いですが、背は森さんより高くないです。
→ 小李比森先生年轻，但个子没有森先生高。
Q9：日本料理の中で、私はすしが一番好きです。
→ 在日本料理中，我最喜欢寿司。

6. 翻訳（中→日）
Q10：这个和那个哪个便宜？
→ これとそれとどっちが安いですか。
Q11：在中国，哪个城市最热闹？
→ 中国で、どの町が一番にぎやかですか。

7. 間違い探し
Q12：中国は日本より広い。
错误：本课程要求用丁寧体。
正解：中国は日本より広いです。
Q13：私のよりあなたのほうが新しいです。
错误：比较对象不完整。应为「私のよりあなたのほうが新しいです」其实正确？但「より」前应为基准名词。更自然：「私のよりあなたのほうが新しいです」可接受，但标准说法「私の車よりあなたの車のほうが新しいです」或省略。此处无错，但可优化。

8. 作文ミニタスク
課題：写5-6句话，比较两个你熟悉的事物（两部电影、两个城市、两位朋友等），说明哪个更好及理由。使用「～より」「～のほうが」「一番」。
例：
私は東京と大阪に行ったことがあります。東京は大阪より人が多いです。でも、大阪のほうが食べ物が安いです。私は大阪のほうが好きです。大阪で一番好きな食べ物はたこ焼きです。

9. 聴解問題（预留，需音频）
听录音，判断A比B更怎么样（年轻、高、便宜等）。

听对话，回答“哪一方被选择”以及理由。
