第16課：ホテルの部屋は広くて明るいです
本课定位：第4单元（公司生活②）的最后一课，也是标日初级上册前半部分的重要总结。本课学习形容词的て形（并列、原因），以及自动词＋ている表示结果状态（与第15课的进行形形成对比）。同时补充逆接「が」和「～という」等表达。本课之后将进入第5单元（迎新春）。

発音ポイント
形容词て形：イ形容詞「広い→広くて」注意「くて」的「く」轻读；ナ形容詞「静か→静かで」的「で」要清晰。

自動詞＋ている：口语中常缩略为「～てる」（例：割れてる、閉まってる）。

「知っている」 否定是「知らない」，注意「知っていますか」的回答：肯定「はい、知っています」，否定「いいえ、知りません」。

「～という」 中的「う」发音清晰。

語源メモ
形容词て形 来源与动词て形相似，表示并列或原因。

自動詞＋ている 表示结果状态的持续，源于古日语「居る」表示存在。

「知る」 是瞬间动词，其て形＋いる表示“知道了并一直知道”的状态。

「という」 由「と」＋「言う」构成，用于引用名称或内容。

活用予告
本课核心是形容词て形（第9、10课已学形容词，现在学并列/原因）和自动词＋ている（第15课已学进行形，本课学结果状态）。这是て形体系的关键一环。第18课将学自他动词对应。

【语法】
ノード1：イ形容詞のて形（并列・原因）
变化规则：去掉词尾「い」＋「くて」

例：

広い → 広くて

明るい → 明るくて

新しい → 新しくて

用法A：并列（两个以上性质）
この部屋は広くて明るいです。 → 这个房间又宽敞又明亮。
このりんごは甘くておいしいです。 → 这个苹果又甜又好吃。

用法B：原因・理由
忙しくて、友達に会えません。 → 因为忙，没法见朋友。
このカレーは辛くて食べられません。 → 这个咖喱太辣了，吃不了。

ノード2：ナ形容詞のて形（并列・原因）
变化规则：词干 + で

例：

静か → 静かで

きれい → きれいで

便利 → 便利で

用法A：并列
この町は静かできれいです。 → 这个小镇又安静又漂亮。
私の部屋は便利で明るいです。 → 我的房间又方便又明亮。（ナ形＋イ形可混用）

用法B：原因・理由
静かで、よく眠れました。 → 因为安静，睡得很好。
便利で、このホテルが好きです。 → 因为方便，喜欢这家酒店。

ノード3：～ています（結果の状態）
句型：自動詞て形 ＋ います
意义：表示动作发生后，其结果状态持续存在。

与第15课进行形的区别：

动词类型	～ている	示例	意义
持续动词（他動詞/自動詞）	正在进行	本を読んでいる	正在读书
瞬间动词（自動詞）	结果状态	窓が割れている	窗户碎了（状态）
常见自动词＋ている（结果状态）：

窓が割れています。 → 窗户碎了。

電気がついています。 → 灯亮着。

ドアが閉まっています。 → 门关着。

友達の名前を忘れています。 → 忘了朋友的名字。

車が壊れています。 → 车坏了。

注意：他动词＋ている一般表示进行（他動詞＋ている：正在进行），自动词＋ている一般表示结果状态（自動詞＋ている：结果持续）。这是难点，需要大量例句理解。

ノード4：～が（逆接・前置き）
句型：文A が、文B
意义：表示转折（虽然…但是…）或引出话题（铺垫）。

例（逆接）：

この部屋は狭いですが、きれいです。 → 这个房间虽然小，但是干净。

日本料理は美味しいですが、高いです。 → 日本料理虽然好吃，但是贵。

例（前置き）：

失礼ですが、お名前は何ですか。 → 不好意思，请问您贵姓？

すみませんが、トイレはどこですか。 → 劳驾，厕所在哪里？

ノード5：～という（名称・内容の引用）
句型：名 という 名
意义：叫做～的～。

例：

「富士山」という山があります。 → 有座叫“富士山”的山。

「さくら」という歌が好きです。 → 我喜欢叫“樱花”的歌。

田中という人を知っていますか。 → 你认识叫田中的人吗？

注意：口语中常省略「いう」，只说「～って」（例：富士山って山）。

【助詞カード】第16课：て形の用法まとめ
用法	形式	例
并列（イ形）	～くて	広くて明るい
并列（ナ形）	～で	静かできれい
原因（イ形）	～くて	辛くて食べられない
原因（ナ形）	～で	便利で好き
逆接	～が	狭いがきれい
【対比表】自動詞＋ている vs 他動詞＋ている
動詞	他/自	～ている	意味
開ける	他	開けている	正在打开
開く	自	開いている	开着（状态）
閉める	他	閉めている	正在关
閉まる	自	閉まっている	关着（状态）
割る	他	割っている	正在打碎
割れる	自	割れている	碎了（状态）
つける	他	つけている	正在开灯
つく	自	ついている	灯亮着
核心规律：他动词＋ている → 动作进行；自动词＋ている → 结果状态。

【対比表】「知る」vs「知っている」
形態	意味	例
知る	瞬间动作（知道→状态）	その名前を知りました。
知っている	知道（状态持续）	その名前を知っています。
知らない	不知道	その名前を知りません。
注意：日语中问“你知道～吗？”用「知っていますか」，肯定回答「知っています」，否定「知りません」（不用「知っていません」）。

【敬語レベル表示】
「～てもいいですか」已学（第15课），本课「～が」用于前置时属于礼貌表达。「～という」无特殊敬语。

【会话】
基本课文（4句型）
この部屋は広くて明るいです。

この町は静かできれいです。

窓が割れています。

ドアが閉まっています。

应用课文：ホテルの感想（酒店感想）
场景：小李和小野入住京都的日式酒店，讨论房间和周围环境。

人物：

李秀麗（り しゅうれい）

小野緑（おの みどり）

对话正文：

小野：李さん、ホテルの部屋はどうですか。
李：とても広くて明るいです。窓から庭が見えます。
小野：いいですね。私はこの静かで落ち着く雰囲気が好きです。
李：そうですね。ホテルのスタッフも親切で、とても感じがいいです。
小野：ところで、エアコンはついていますか。
李：はい、ついています。でも、リモコンがありません。
小野：あっ、本当ですね。窓も閉まっていますし、ちょっと暑いですね。
李：フロントに電話して聞いてみましょう。
小野：そうですね。
李（電話で）：もしもし、すみませんが、エアコンのリモコンが見つからないんですが。
フロント：かしこまりました。すぐにお持ちします。
小野：あっ、窓の外を見てください。雨が降っています。
李：本当だ。でも、雨音が心地よくて、逆にリラックスできます。
小野：そうですね。この静けさがいいですね。
李：では、少し休みましょう。おやすみなさい。
小野：おやすみなさい。

会話のキーポイント
「広くて明るい」 イ形容詞て形并列。

「静かで落ち着く」 ナ形容詞て形＋动词。

「親切で感じがいい」 て形原因/并列。

「エアコンはついていますか」 自动词＋ている（结果状态：亮着/开着）。

「窓も閉まっていますし」 「～し」表示列举原因（本课不要求，但出现）。

「リモコンが見つからないんですが」 「～んですが」前置き。

「雨が降っています」 自动词进行（雨は降る本身是持续动词，表示正在下雨）。

「心地よくて」 イ形容詞て形原因。

ロールプレイ課題
课题1：两人一组，描述自己的房间或酒店房间（大小、明亮度、安静与否、设备状态），使用形容词て形和「～ています」（结果状态）。

课题2：A问B“窗户关着吗？”“灯亮着吗？”等，B根据实际情况回答。

课题3：介绍一个你喜欢的城镇（或场所），使用「～くて」「～で」并列形容词。

【作业】
1. 発音練習
朗读形容词て形：広くて、明るくて、新しくて、静かで、きれいで、便利で。

朗读自动词＋ている句子：窓が割れています。ドアが閉まっています。電気がついています。

注意「～ています」的口语缩略「～てます」。

2. 活用ドリル
将下列形容词改为て形（并列形式）：

広い→

明るい→

新しい→

静か→

きれい→

にぎやか→

解答：広くて、明るくて、新しくて、静かで、きれいで、にぎやかで

将下列动词改为「～ている」并写明是进行还是状态：

読む（本を）→

割れる（窓が）→

降る（雨が）→

閉まる（ドアが）→

勉強する→

解答：読んでいる（进行）、割れている（状态）、降っている（进行）、閉まっている（状态）、勉強している（习惯/进行）

3. 選択問題
Q1：この部屋は＿＿＿明るいです。
A. 広い　B. 広くて　C. 広く　D. 広かった → B

Q2：この町は＿＿＿きれいです。
A. 静か　B. 静かな　C. 静かに　D. 静かで → D

Q3：窓が＿＿＿います。
A. 割れて　B. 割り　C. 割る　D. 割った → A

Q4：ドアが＿＿＿。
A. 閉まっています　B. 閉めています　C. 閉まっている　D. AとC両方正しい → D（A丁寧、C普通）

4. 穴埋め問題
Q5：この店は＿＿＿やすいです。（安い） → 安くて
Q6：彼は親切で、＿＿＿いい人です。 → 感じが
Q7：友達の電話番号を＿＿＿いました。 → 忘れて

5. 翻訳（日→中）
Q8：このホテルの部屋は広くて明るいですが、ちょっと高いです。
→ 这家酒店的房间又宽敞又明亮，但是有点贵。
Q9：窓が開いていますから、ちょっと寒いです。
→ 因为窗户开着，所以有点冷。

6. 翻訳（中→日）
Q10：这个城市又安静又漂亮。
→ この町は静かできれいです。
Q11：灯关着吗？——是的，关着。
→ 電気が消えていますか。 — はい、消えています。（消える是自动词）

7. 間違い探し
Q12：この部屋は広いで明るいです。
错误：イ形容詞て形应是「広くて」。
正解：この部屋は広くて明るいです。
Q13：窓が割れます。
错误：表示“窗户碎了”的状态应使用「割れている」。
正解：窓が割れています。

8. 作文ミニタスク
課題：写5-6句话，描述你的房间（或教室、图书馆），使用形容词て形并列2-3个特征，再描述两样东西的状态（窗户开/关、灯亮/灭、空调开/关等）。
例：
私の部屋は狭いですが、明るくてきれいです。机の上に本がたくさんあります。窓が開いていて、風が入ってきます。電気は今ついています。エアコンは消えていますが、ちょうどいい温度です。

9. 聴解問題（预留，需音频）
听录音，判断是进行还是状态（～ている）。

听对话，选出对房间的描述（形容词）。
