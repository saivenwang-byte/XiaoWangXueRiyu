# 服务类目选择对比表（工具向）

> **产品**：日语初级课后练习  
> **主体**：个人 / 个体工商户（按实际）  
> **填写位置**：微信公众平台 → 设置 → **基本设置 → 服务类目** → 添加类目

---

## 一、拍板结论（2026-05-30）

**仅选：工具 → 信息查询**（不再并列教育信息展示，除非审核要求补充）。

审核备注：

> 服务类目：工具-信息查询。本产品为《标准日本语》初级上册课后练习辅助，提供课次练习与进度查询，非在线授课、非培训机构、暂无付费；非人民教育出版社官方产品。学员须自备正版教材。

---

## 一（原）、推荐方案（参考）

| 优先级 | 一级类目 | 二级类目 | 是否推荐 | 理由 |
|--------|----------|----------|----------|------|
| **★ 首选** | **工具** | **信息查询** | ✅ 强烈推荐 | 展示课序、语法点、学习进度与练习结果；无直播、无排课、无招生 |
| **★ 辅选** | **教育** | **教育信息展示** | ✅ 推荐（与工具并列） | 课后练习与复习信息展示；**非**在线培训、非学科辅导直播 |
| 备选 | 教育 | 在线教育 | ❌ 不推荐 | 易被理解为授课/培训，个人主体常需额外资质 |
| 备选 | 教育 | 培训机构 | ❌ 不推荐 | 需办学许可等，与本产品形态不符 |
| 备选 | 文娱 | 资讯 | ❌ 不推荐 | 语义偏内容资讯，与练习工具不符 |
| 备选 | 商家自营 | 图书 | ❌ 不推荐 | 不销售教材 |

**实操建议**：先添加 **工具 → 信息查询**；若审核提示需教育类目，再补充 **教育 → 教育信息展示**，并在备注中强调「仅课后练习展示、无授课」。

## 二、「工具 · 信息查询」适配说明（审核用）

| 维度 | 本产品情况 | 是否符合 |
|------|------------|----------|
| 核心功能 | 按教材课次查询练习、查看进度 | ✅ |
| 是否销售商品 | 否 | ✅ |
| 是否提供付费咨询/授课 | 否 | ✅ |
| 是否 UGC 社区 | 否（仅本地笔记） | ✅ |
| 是否医疗/金融查询 | 否 | ✅ |

**一句话论证**：  
用户打开小程序是为了**查询并完成与自己教材课次对应的练习信息**，属于工具类信息查询，不是在线学校。

## 三、「教育 · 教育信息展示」适配说明（若双类目）

| 维度 | 本产品情况 | 是否符合 |
|------|------------|----------|
| 内容形态 | 课后练习题、语法提示、会話示范 | ✅ |
| 是否直播上课 | 否 | ✅ |
| 是否发放学历/证书 | 否 | ✅ |
| 是否招聘教师 | 否 | ✅ |
| 是否替代教材 | 否，明确课后辅助 | ✅ |

**一句话论证**：  
仅展示与初级上册课次相关的**教育信息与实践题目**，不开展在线培训业务。

## 四、类目 vs 资质对照

| 类目 | 个人主体常见要求 | 本产品是否需要 |
|------|------------------|----------------|
| 工具 · 信息查询 | 通常无额外资质 | 否 |
| 教育 · 教育信息展示 | 一般无培训许可证（以平台当期规则为准） | 否 |
| 教育 · 在线教育 | 常需ICP、培训备案或企业资质 | **不适用** |
| 教育 · 培训机构 | 民办学校许可证等 | **不适用** |
| 出版物 / 图书销售 | 出版物经营许可证 | **不适用**（不售书） |
| 音像制品 / 在线课程销售 | 相关许可 | **不适用**（无付费课） |

> 平台规则会变，提审前以 [微信公众平台](https://mp.weixin.qq.com) 类目资质页面为准。

## 五、不推荐类目及风险

| 类目 | 若误选的风险 |
|------|----------------|
| 在线教育 / 校外培训 | 要求培训资质、易拒审 |
| 社交 / 社区 | 需内容审核机制，本产品无社交 |
| 医疗、金融、政务 | 完全无关，必拒 |
| 使用「标日」「官方教材」作小程序名称 | 商标与官方关联风险，与类目无关但会导致合规失败 |

> 本产品为《标准日本语》初级上册**课后练习辅助**，提供课次练习与进度查询，**非在线授课、非培训机构、无付费**；非人民教育出版社官方产品。学员须自备正版教材。

## 七、决策流程图

```
是否在线直播/排课/招生？
  ├─ 是 → 不适用本产品
  └─ 否 → 是否主要做「查课次、做练习、看进度」？
            ├─ 是 → 首选【工具 · 信息查询】
            └─ 是否强调「教育内容展示」？
                  ├─ 是 → 可加【教育 · 教育信息展示】
                  └─ 否 → 仅工具类目即可
```
