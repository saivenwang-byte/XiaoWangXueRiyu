# 标准课文模块 · 第1課「李さんは中国人です」

> 版本：v220 · 2026-05-25  
> 用途：所有后续课次按此模板对齐映射  
> 数据源：`js/data/lessons-data.js`（lessonId: 1）

---

## 一、页面入口

```
首页「第1単元」→ 点第1課 → 进入五关页面
```

顶部显示：
- ← 首页（返回按钮）
- 第1課 · 自己紹介（课号+主题）
- 五个 Tab：単語 / 会話 / 文法 / 作業 / 拡張

---

## 二、Tab 栏规范

### Tab 定义（第1課专用，在 `lesson-1-flow.js`）

| k值 | 标签 | 副标题 | 颜色 |
|-----|------|--------|------|
| 0 | 単語 | フラッシュ | 蓝 #2196F3 |
| 2 | 会話 | 多场景 | 绿 #4CAF50 |
| 1 | 文法 | ネット | 橙 #FF9800 |
| 3 | 作業 | 練習 | 紫 #9C27B0 |
| 4 | 拡張 | まとめ | 青绿 #009688 |

### Tab 样式

- 当前选中：对应颜色背景 + 白色文字
- 已完成：淡绿背景 `#e8f5e9` + 深绿文字 `#2e7d32`
- 未选中：灰色背景 `#f5f5f5` + 灰色文字 `#666`
- 无 ✅ 打勾图标
- 图标：无（纯文字标签）
- 字号：主 13px bold，副 9px
- 排列：横向一行，等宽分布

### 3.4 发音/语源/预告/敬语区块

```
<details class="l1-vocab-supp" open>
  <summary class="l1-vocab-supp-head">🗣 発音ポイント</summary>
  <ul class="l1-vocab-supp-list">
    <li>助词「は」读「わ」</li>
    ...
  </ul>
</details>
```

- 发音区块默认展开（`open`）
- 其余区块默认折叠
- 箭头：浏览器默认 ▶
- 数据来源：`L.summaryBlocks[].lines`

### 3.5 词变卡片

- 触发：点击 📐 按钮
- 弹窗：`<div class="kcard-overlay">` 蒙层 + 白色卡片
- 内容：活用类型 + 变形表（否定/礼貌/て形/た形）
- 关闭：右上角 ✕（28x28px 圆形灰底）或点击蒙层

### 3.6 延展知识卡片

- 触发：点击 💡 按钮
- 弹窗：`SenseiTipCard.fromTipPayload(tipData)`
- 数据来源：`L1KnowledgeTips.vocab(v)`
- 含跳转链接到关联的语法/会话关

### 4.2 关键规范

- 会话要点：折叠在顶部，默认展开
- 底部按钮：「本场景听完 ✓」+「下一场景 →」

## 五、文法关（gate 1）

### 5.1 整体布局

```
┌──────────────────────────────────┐
│  必学 1/4 · 全体 1/4              │ ← 进度
│                                  │
│  ┌────────────────────────────┐  │
│  │ 〜は〜です                  │  │ ← 日文标题 + 🔊
│  │ 名1是名2（名词谓语句・肯定） │  │ ← 中文标题
│  └────────────────────────────┘  │
│                                  │
│  日文说明                        │
│  「AはBです」で「A＝B」を表す    │
│                                  │
│  中文说明 · 怎么用                │
│  名词谓语句肯定形。「は」提示主题 │
│                                  │
│  例句                            │
│  わたしは学生です。 🔊           │
│  我是学生。                      │
│                                  │
│  📖 前：ます形（第5課）           │ ← 关联链接
│  🔗 後：形容詞のて形（第16課）    │
│                                  │
│  #動詞活用 #接続の基本            │ ← 标签
│                                  │
│  [前へ] [次へ]                   │ ← 导航
└──────────────────────────────────┘
```

### 5.2 关键规范

- 进度条：`必学 N/M · 全体 N/M`
- 例句兼容：`exampleZh` 可为字符串或数组
- `RubyRender.nodeExample(node)` 渲染例句注音
- 关联链接以不同颜色按钮显示（蓝=前置，绿=扩展，红=对比）
- 标签以灰色小胶囊显示

## 六、作業关（gate 3）

### 6.1 整体布局

```
┌──────────────────────────────────┐
│  [← 上一题] 問題 1/12 [下一题 →] │ ← 导航
│                                  │
│  わたし＿＿＿学生です。   🔊    │ ← 题干 + 🔊
│  我＿＿＿是学生。                 │ ← 中文翻译
│                                  │
│  题干 🔊 读完整日文句            │ ← 提示
│                                  │
│  ┌────────────────────────────┐  │
│  │ ○ は  🔊                  │  │ ← 选项（每项带 🔊）
│  │ ○ が  🔊                  │  │
│  │ ○ の  🔊                  │  │
│  │ ○ に  🔊                  │  │
│  └────────────────────────────┘  │
│                                  │
│  [送信] （填空题时显示）          │
└──────────────────────────────────┘

回答后：
┌──────────────────────────────────┐
│  ✅ せいかい！                    │
│  提示：助词「は」提示主题         │ ← SenseiTipCard
│  [次へ]                          │
└──────────────────────────────────┘
```

### 6.2 关键规范

- 题型：`choice`（选择）或 `fill`（填空）
- 每题带 🔊 喇叭（读题干完整句）
- 选项带 🔊 喇叭（读选项）
- 回答后显示 SenseiTipCard 提示卡片
- 错误题目自动加入复习列表
