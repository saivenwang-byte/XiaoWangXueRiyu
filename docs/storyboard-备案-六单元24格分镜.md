## 1. 画幅与格位（每页统一 · 美工按此切格）

| 项 | 规格 |
|----|------|
| 成品文件 | `assets/story/unit-{1..6}-strip.webp` |
| 画布比例 | **横 4:1**（建议 1600×400px，@2x 可 3200×800） |
| 分格 | **等宽 4 格**，格间 8px 留白或细白线；**阅读顺序左→右** |
| 安全区 | 每格内缩 12px；四角圆角 8px（与 App 占位一致） |
| 主人公 | **グルミ ＝ 小李（李さん）** 一体；每格出镜；造型见绘本，**不另画人类脸** |
| 配角 | 同事/路人 = 绘本简笔或剪影 |
| 台词 | **日漫对白泡** + **场景云**；课文会話原文；泡旁标角色；见 [storyboard-漫画表现规范.md](./storyboard-漫画表现规范.md) |
| 背景 | 季节纯色/双色渐变 + 卡通抽象建筑（绘本规范） |

**定格原则**：一格 = 该课课文**最有画面感的一瞬**，不是四课剧情拼贴在一格。
