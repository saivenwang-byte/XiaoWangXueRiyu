## 三、侵权 / 授权风险清单

> **声明**：以下为产品合规自查，非法律意见。正式上架前建议咨询律师或版权方。

| 风险级 | 序号 | 内容 | 风险说明 | 建议 |
|--------|------|------|----------|------|
| 🔴 高 | R1 | 产品名 **「标日 あと学習」「标日课后巩固」** | 「标日」为《新版中日交流标准日本语》简称，与人教社等权利人关联强；用作 App/小程序名易被认定为**商标性使用**或**官方关联暗示** | 对外改为中性名（如「日语初级上·学习の道」）；副标注明「非官方·课后辅助」 |
| 🔴 高 | R2 | 分享文案 **「标日初级上·24课」** | 暗示覆盖官方教材全册，且未授权 | 改为「初级上册课序辅助」；加免责声明链接 |
| 🔴 高 | R3 | **课文摘录 / 会話原句**（全日文对话、基本课文锚点） | 教材正文受著作权保护；即便标注【标日课文】仍可能构成**复制传播** | 获授权，或仅保留自建例句 + 页码指引；勿全文摘录 |
| 🟠 中 | R4 | **作業题「【题源】标日第 N 课」** | 题型若源自练习册，存在衍生内容风险 | 标注「自编题，语法点与教材对应」；避免照抄原题 |
| 🟠 中 | R5 | **单词表 / 语音包** 与教材生词对齐 | 生词列表与官方词表高度重合时需注意 | 注明「学员自备正版教材」；不打包官方 PDF/扫描图 |
| 🟠 中 | R6 | **角色名「小李」「森」「小野」** 等 | 标日课文固定人设，商业使用可能涉教材 IP | 漫画用原创グルミ；课文区仅作第三人称讲述（已符合 Soul 锁） |
| 🟡 低 | R7 | 课内标注 **「与标日课文一致」**（会話 noteZh） | 描述性对照，风险低于直接转载 | 可改为「与教材课文一致（请自备课本）」 |
| 🟡 低 | R8 | **「标日 P.34」** 等页码引用 | 事实性引用，一般风险较低 | 保留即可 |
| ⚪ 合规 | R9 | **无登录、无支付、无上传服务器** | 个人信息风险较低 | 仍需隐私政策（本地存储、麦克风） |
| ⚪ 合规 | R10 | **GitHub 域名无 ICP** | 微信小程序业务域名 / 国内合规问题，非版权 | 企业成立后备案或迁国内 CDN |

**仓库现状**：未见与人教社/版权方书面授权文件。
