## 1. 三个底栏入口（只看图标应能懂）

> INTEPOINT「人才全景」用于**拆领域、定顺序、定验收**；底栏下表是**产品 IA**，不是岗位人设表。

| 位置（左→右） | 产品语义 | 图标 | 禁止 |
|---------------|----------|------|------|
| 左 | **课程 / 课表** | 打开的书本 | 地图 emoji、折线图 |
| 中 | **我的 / 进度** | 头像轮廓（人形） | 齿轮、设置堆砌 |
| 右 | **五十音查阅** | 方框 + あ | 键盘 ABC、🔤 |

文案可辅助（课程 / 我的 / 五十音），但**图例与导航必须用同一套 SVG**。

## 2. 四层图标体系

| 层级 | 用途 | 造型规则 |
|------|------|----------|
| L0 | 底栏、地图图例「底栏」行 | 24px 线性 SVG，`currentColor` |
| L1 | SK-xx、单元色条、U1–U6 | 色条 + 胶囊字，不用 emoji |
| L2 | 课号圆、四颗星、地图站点 | 几何+数字，海星 ○☆★ |
| L3 | 🔊 🎤 等 | 小尺寸操作，灰色次要 |
| L0 顶栏 | **回到课文首页** | 房子线性 SVG（`NavIcons.home`）· 与底栏「课程」书本区分 · 四模块（入門/课文/书写/笔记）首行同型 |
| L0 顶栏 | **转发学习链接** | 与二维码成对：`icons/share-forward.svg` · 圆角卡片 + **向上外链箭头**（系统「分享」语义）· `stroke-width:2` · `currentColor` = 海军蓝，**不用**微信绿底 |

## 3. 首页一屏高度预算（移动端）

```text
100dvh
 − 顶栏 (~48px)
 − 底栏 (~56px)
= 主内容区（地图 + 00入門 + 折叠课表）
```

- 地图区：`~22dvh` 上限，可再压
- 课表：仅 1 个单元 `open`，其余 `<summary>` 单行
- 状态徽章（未開啟/進行中/START）在 **summary 右侧**，不占第三行
