# 微信小程序 · 你只需要做一次（之后同学「点开就用」）

届时做成 **真正的微信小程序**：  
同学在微信里 **点小程序卡片 → 直接进学习页**（和点「美团」「滴滴」一样）。

---

## 同学怎么用（发布后）

- 你发 **小程序卡片** 到群 / 好友，或  
- 他们 **搜索小程序名称**，或  
- **扫小程序码**  

**不需要** 复制网址，**不需要** 同一 WiFi。

---

## 你只需要做 4 步（一次性）

### 1. 注册小程序

电脑打开：https://mp.weixin.qq.com/  
用邮箱注册「小程序」（个人或企业均可）。  
记下你的 **AppID**（开发 → 开发管理 → 开发设置）。

### 2. 配置业务域名（让小程序能打开学习内容）

登录 mp.weixin.qq.com → **开发** → **开发管理** → **开发设置** → **业务域名**

添加：

```text
saivenwang-byte.github.io
```

按页面提示下载校验文件（一个 `.txt`），把该文件放到本仓库 **根目录**，并 `git push`，等 GitHub Pages 生效后再点「保存」。

### 3. 用微信开发者工具上传

1. 下载安装：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html  
2. 打开开发者工具 → **导入项目**  
3. 目录选：`日语学习\japanese_learning_miniapp`  
4. AppID 填你在第 1 步记下的（不要选「测试号」才能正式发布）  
5. 点 **预览**：用手机微信扫二维码，先自己试一遍  
6. 确认无误后点 **上传** → 填写版本号（如 1.0.0）

### 4. 提交审核（或先设体验版）

回到 mp.weixin.qq.com → **管理** → **版本管理**  
- 开发版：可先加「体验成员」扫码用  
- 正式版：点 **提交审核** → 通过后 **发布**

发布后，在后台 **设置 → 基本设置** 里可下载 **小程序码**，发到微信群即可。

---

## 学习内容改哪里？

小程序里加载的是线上 H5，地址在：

`japanese_learning_miniapp/config/h5-url.js`

你改 H5、推 GitHub 后，同学下次打开小程序就是新版本（与小程序「上传」无关，除非改小程序壳本身）。

**壳内刷新**：学习页右下角「刷新」按钮（`cover-view`）；加载失败会显示重试页（非白屏）。`CACHE_VER` 须与 `js/share-wechat.js` 同步。

## 和之前 bat / 链接的区别

| 以前（网页链接） | 现在（小程序） |
|------------------|----------------|
| 复制一长串 https | 点小程序卡片 |
| 像打开外部网页 | 像打开正常小程序 |
| 要自己发链接 | 可搜索、可分享卡片、可小程序码 |

技术细节（仓库、Netlify、语音包）由开发维护；**你只需完成上面 4 步一次。**
