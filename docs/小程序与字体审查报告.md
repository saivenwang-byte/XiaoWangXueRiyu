## 三、美工 · 字体与乱码

### 3.1 字体规范（当前）

| 区域 | 字体栈 | 评价 |
|------|--------|------|
| 正文中文 | PingFang SC / Hiragino Sans GB / system | ✓ 微信可用 |
| `.jp` 日文 | Noto Sans JP → Hiragino Kaku Gothic ProN → sans | ✓ 无离线字体包；缺字时走系统日文 |
| 书写 `.write-l0-*` | Hiragino Sans / Yu Gothic UI / Noto Sans JP | ✓ |
| 底栏图标 | SVG + 系统 sans（「あ」字） | ✓ 非图片字 |

**未引入** `.woff` / `.ttf`，避免包体膨胀与 WebView 闪屏。
