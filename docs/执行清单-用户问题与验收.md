## 三、分项验收命令

### 3.1 语音包（缺失必须为 0）

```bat
python scripts\audit-tts-registry.py --write
python scripts\report-tts-audit.py
```

通过标准：输出 `缺失 MP3 0`；见 `docs\TTS-对账报告-最新.md`。

### 3.2 本地预览（禁止 file://）

```bat
打开本地预览.bat
```

或：

```bat
开发者竖屏验收.bat
```

URL：`http://127.0.0.1:8765/index.html?v=` + `js/share-wechat.js` 内 `CACHE_VER`

探测：

```bat
python scripts\start-local-server.py --probe
```

### 3.3 我的 · 笔记

| 步骤 | 操作 | 预期 |
|------|------|------|
| 1 | 底栏「笔记」→ 展开自学笔记 | 高编辑区、全册/单元/课次 tab |
| 2 | 写几句 → 点页面空白 | 收起且内容仍在 |
| 3 | 单元下点 📝 | 切到该课/单元 scope |
| 4 | 点课次进学习 → 返回 | 回到「笔记」，不是课文首页 |

### 3.4 五十音 / 入門

| 步骤 | 操作 | 预期 |
|------|------|------|
| 1 | 课文 → 00 入門 | 不白屏、不 3 秒卡死 |
| 2 | 滑假名表 | 不自动连弹多层 |
| 3 |  intentional 点一字 | 才出先生卡 |

### 3.5 会話 · 纵向折叠

| 步骤 | 操作 | 预期 |
|------|------|------|
| 1 | 第1课 → 会話 | 顶栏无 ★単語→まとめ 横条 |
| 2 | 见 1～N 纵向编号行 | 默认全折叠 |
| 3 | 点开第 1 行 | 显示该句完整会話+🔊 |
| 4 | 点第 2 行 | 第 1 行收起 |

### 3.6 会話 🔊

| 步骤 | 操作 | 预期 |
|------|------|------|
| 1 | 第1课 → 会話 → 点 🔊 | 灰色喇叭；约 1～2 秒有声 |
| 2 | 同句再点 | 不应长期「朗读失败」 |
