# 链接转发 · 现阶段正式上线方式

## 正式链接

- 学习：`https://saivenwang-byte.github.io/XiaoWangXueRiyu/index.html?v=290`
- 分享：`https://saivenwang-byte.github.io/XiaoWangXueRiyu/share.html`

## 作者操作

1. 改课程内容、语音等
2. `git push` 到 GitHub（仓库 `XiaoWangXueRiyu`）
3. 等 1～2 分钟
4. 双击 **`帮你发布好了.bat`** 复制链接发微信

学员始终用**同一条链接**；内容随 push 更新。

## 学员体验

- 微信收到链接 → 点开学习
- 可再转发给好友（人传人）
- 跟读录音：微信内若失败 → 右上角「在浏览器中打开」
