# 项目知识库：标日日文书写与对齐（首要）

> **适用范围**：`js/data/` 内所有会出现在界面、TTS、跟读评分中的日文。  
> **权威参照**：《标准日本语》初级上册第 14·16·18 课课文、对话、语法例句（`lessons-detailed.js` 已摘录，改稿时先对照）。  
> **原则**：按日语发音与排版习惯写，不按中文标点或中文语序硬套。

---

## 1. 字段分工（禁止混写）

| 字段 | 语言 | 标点 |
|------|------|------|
| `japanese` / `jp` / `example` / `meaningJa` / `question` / `questionTts` / `noteJa` | 日文 | 日文规则 |
| `chinese` / `meaningZh` / `noteZh` / `explanationZh` / `zh` | 中文 | 中文规则 |
| `kana` / `ruby` | 读音 | 不用中文逗号；并列读音用 `・` |

语法条目（て形、ください等）只出现在 `grammarNodes`，**不得**进入 `lesson-vocab-biaori.js` 单词表。

---

## 2. 标点对照（日文区必查）

| 错误（中文习惯） | 正确（日文/标日） | 说明 |
|------------------|-------------------|------|
| `，` 中文逗号 | `、` 读点 | 并列、停顿 |
| `；` | `。` 或另起一句 | 日文少用分号 |
| `：` 中文冒号 | `：` 全角亦可，或改用「」 | UI 标签里的「第5課」可保留 |
| `（）` 在注释里混用 | 日语注释可用（），引用用「」 | 例句对比优先 `bad`/`good` 分字段 |
| 半角空格 ` ` 夹在假名/汉字间 | **禁止** | 影响 TTS 分词 |
| 全角空格 `　` | **禁止**（课文、例句） | 标日正文不用 |
| `／` 或 ` / ` 并列 | 见下节 | |
| `？` `！` 单独来自中文输入法 | `？` `！` 全角即可；疑问句也可用「…か。」 | 与标日课文一致即可 |
| `……` | `…` 或 `〜` | 省略、拖音 |

---

## 3. 并列与列举（常见错误源）

- **多个变形/例句在一行**：用 **`、`** 连接  
  - ✅ `書く→書いて、待つ→待って、読む→読んで`  
  - ❌ `書く→書いて / 待つ→待って`
- **两个完整句子**：用 **`。`** 分句，或拆成两条 `example`  
  - ✅ `寒くなりました。元気になりました。`  
  - ❌ `寒くなりました。 / 元気になりました。`
- **自动词／他动词对照**：用 **`／`**（全角）或 **`・`**，不用半角 ` / `  
  - ✅ `開く／開ける`、`あく・あける`（读音字段）
- **读音多个**：`なに・なん`  
  - ❌ `なに／なん`、`なに/なん`

---

## 4. 标日课文对齐（第 14·16·18 课）

改对话、标题、例句前，**先查** `lessons-detailed.js` 同课 `basicText` / `dialogues` / `grammar.examples`。

| 课 | 易错点 | 标日写法 |
|----|--------|----------|
| 14 | 购物句宾语 | `買い物**を**しました`（要有を） |
| 14 | て形连接 | `行って、買い物をします` |
| 14 | 疑问 | `何を買いましたか。`（句末。或か。） |
| 16 | 形容词并列 | `広くて明るいです`（て形连接） |
| 16 | 转折 | `…ですが、…`（不是中文「但是」语序） |
| 18 | 变化 | `小さくなりました`（い→くなる） |
| 18 | 人为变化 | `きれいにします`（にする） |

MVP 会话（`lessons-mvp.js`）可在标日对话基础上做**扩展场景**，但**同一语法点的例句**应与标日一致，勿自造错误宾语、助词。

## 5. 中文灰字（课外讲题 / 单词 / 语法）

- 单词：`meaningJa`（日文说明）+ 下一行 `meaningZh` → UI 用 `zh-annotation`
- 语法卡片：`explanationZh`、`exampleZh`（`GRAMMAR_EXAMPLE_ZH`）
- 课外讲题 `depthSections`：每条 `text` 必有 `zh`；`list` 项用 `{ jp, zh }` 对象，**禁止**仅日文字符串
- 会話：`noteZh` 可选，有则灰字显示
