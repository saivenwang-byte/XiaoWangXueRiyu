# 作業 · 拡張 — 标日真源规范（PM 定案）

> **真源顺序**：标日初级上册课文（会話 + 基本课文）→ `js/data/lessons-data.js` 同课 `basicText` / `dialogues` → `js/data/lessons-detailed.js` 摘要 → 练习册思路（不另造句义）  
> **L1 冻结**：第1课数据与 `lesson-1-flow.js` 不改；本规范从 **第2课起** 适用。

---

## 一、作業（`homeworkSections` + `quizQuestions`）

### 1.1 必须

| 要求 | 说明 |
|------|------|
| **课文为根** | Q1–Q12 的日文句、场景、词汇均须能在当课 **标日课文**（`basicText` + 会話 A 轨）中找到或由其直接改写 |
| **小测同源** | `quizQuestions` 与作業 Q1–Q12 **一一对应**；第12题为 **综合题**，综合当课 **2 个以上** 课文语法点 |
| **九段结构** | 発音・活用・選択・穴埋め・日译中・中译日・総合（原間違い位）・作文・聴解 — 与 L1 MVP 一致 |
| **禁止** | 未出课的语法、与课文无关的「泛化刷题句」、机翻腔中文解析 |

### 1.2 每段建议写法

```text
【标日课文】基本课文：……（列出本课 basicText 原句）
【标日课文】会話摘录：……（1–2 句 A 轨原句，可选）
Q1：……（改写/挖空仍保持课文词汇）
→ 答案 + 一句「对应课文：……」
```

### 1.3 综合题（Q12）口径

- 选项均为 **课文句型变体**（正确项 = 课文或课文直接组合）
- `explanation` 须写清 **依据哪句课文 / 哪条语法**

## 二、拡張（`summaryBlocks` + `reviewExtension` + 要点/扮演）

### 2.1 两层内容

| 标签 | 含义 | 允许程度 |
|------|------|----------|
| **【课文锚点】** | 复述、表格化、对比当课标日已出表达 | 必须与课文一致 |
| **【知识延展】** | 易错、模板、跨课预告、文化注 | 可 **略超纲**（标日当课未展开但初级上册内、或礼貌/文化补充）；须标注「延展」 |

### 2.2 五块 `reviewExtension`（对齐 L1）

1. 📖 迄今总结 — **先列本课课文核心句 2–4 条**，再写跨课链接  
2. 🗂️ 参考表 — 本课词汇/语法 **课文已出现** 的归纳表  
3. 📋 模板 — 由 **课文句型** 抽槽位 `[名詞]`，例句用课文原句或课文变体  
4. ⚠️ 常见误用 — 针对课文句型的 ×/○；可含 **【知识延展】** 1 条  
5. 📝 单元/复习预告 — 标日 **单元练习范围** + 下单元入口（略超纲仅在此块末尾 1–2 行）

### 2.3 禁止

- 与当课课文无关的「百科式」大段
- 未标注的难点语法（应写「延展：第 N 课正式学」）
- 扩展模块里整段未译日文（中文区须可读）
