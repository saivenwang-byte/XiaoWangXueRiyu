# 四格漫画 · 画幅与版式定案（2026-05-22）

> **勿与**「探索型学习地图」封面混淆：地图稿只借 **布局/信息结构**，**不借**其中人物造型画四格。

---

## 1. 你要的形态（最终）

| 项 | 定案 |
|----|------|
| 观看 | 手机 **横屏** |
| 整页 | **一张图** = 一个单元的四格漫画 |
| 分法 | 横屏画面 **1 分 4**（**2×2 等大田字格**） |
| 整图画布 | **16:9**（例 1920×1080） |
| 单格比例 | 四格等分时，**每一格也是 16:9**（因 16/2 : 9/2 = 16:9） |
| 阅读顺序 | **左上 → 右上 → 左下 → 右下** = 第1課 → 第2課 → 第3課 → 第4課 |
| 一格一课 | 每格 = **一幅完整插画** + 圆泡台词（不是聊天列表 UI） |
| 文件 | `assets/story/unit-{n}-strip.webp` **单文件** |

**不是**：竖屏纵滚、四张分离卡片、微信聊天气泡列表、4:1 细长横条四连。
