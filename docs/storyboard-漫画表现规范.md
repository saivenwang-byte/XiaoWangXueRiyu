# 单元四格漫画 · 日漫表现规范（全册）

> **用途**：美工、条带彩蛋、L1 单课插画统一。  
> **原则**：**对话用课文真实会話**；**主人公 グルミ＝小李**；绘本线条。  
> **分镜**：按单元见 `storyboard-P1-第1单元-定稿.md` … P6（逐页追加）。

---

## 1. 初衷（产品）

彩带彩蛋 = **把课文会話还原成真实场景的定格漫画**：谁在什么场合说哪句话、怎么应答。表现手法同日本漫画：

- 角色头上 **对白泡** 说话  
- 画面一角 **场景云** 说明时间地点  
- 必要时 **旁白云** 交代转场  

不是语法图解，不是第三人称解说漫画。

---

## 2. 符号表（分镜稿标注）

| 标注 | 形状 | 指向 | 内容来源 |
|------|------|------|----------|
| `【泡·上】` | 椭圆对白泡 | 说话者嘴 | `lesson-dialogues.js` / 课内会話 **原文** |
| `【泡·下】` | 略小对白泡 | 另一人 | 应答句 |
| `【云·场景】` | 无尾方框 | 格角 | 场景名（教室/デパート/宿舍…） |
| `【云·说明】` | 旁白方框 | 不指向人 | 时间跳跃、省略 |
| `【思考】` | 云朵泡 | 头部 | 少用 |

泡旁 **必须标角色名**（田中 / 李 / 店员）。**李 一律画为 グルミ**。

---

## 3. 对话取材顺序

1. `js/data/lesson-dialogues.js` → `getLessonDialogues(课号)` 主会話  
2. `js/data/lessons-detailed.js` → `basicText` / `exercises.read` 补句  
3. `js/data/curriculum-catalog.js` → `CURRICULUM_LESSON_META.headline` 作标题/小字  
4. 仍缺则标 **待补课文** 再画，不编造语法例句

---

## 4. 每格构图

- **1 格 = 1 课 = 1 个场景定格**  
- 1–3 组泡（超过则删次要轮次，保留核心问答）  
- 背景：季节色渐变 + 卡通抽象（绘本）  
- 文件：`assets/story/unit-{n}-strip.webp`，横 4 等分
