## 1. 产品定位（策划必须先统一的一句话）

### 1.1 我们是什么 / 不是什么

| 是 | 不是 |
|----|------|
| 《标日·初级上册》课后的 **10 分钟轻量巩固** | 电子课本、完整课程体系 |
| 课本 **教**，我们 **串 + 练 + 记** | 替代课堂、AI 发音评分、SRS、登录体系 |
| **学习の道**：沿日本列岛示意路径，从南向北「一站站」学完 24 课 | 剧情 RPG、体力、角色养成、开放世界地图 |
| 小朋友能玩懂，家长能 **在マイ 看见进度** | 给成人自学的重度语言 App |

**PRD 原话**（`产品PRD.txt` §1.5）：做轻不做重——不做剧情地图/角色养成，做 **一课四关 + 完成打勾**（产品愿景已升级为 **海星通关 + 漫画收藏**，但仍 **轻**）。

### 1.2 核心用户与场景

| 角色 | 是谁 | 看到封面时在干什么 |
|------|------|-------------------|
| **直接使用者** | 学标日的小朋友（**初中生为主**，小学高年级/高中生亦可） | 微信里点开链接 / 打开小程序 **第一眼** |
| **监督者** | 家长（多半不懂日语） | 偶尔看孩子屏幕；需要感到「正规、清晰、不像乱糟糟的游戏」 |

**场景**：学校/机构上完某一课 → 回家打开 → **封面 3 秒内** 应感受到「这是标日课后助手，有一条清楚的路要走」，而不是「满屏菜单不知道从哪点」。

### 1.3 品牌与气质关键词（给视觉）

- **标日 あと学習**（课后巩固，不是另起炉灶的教材名）
- **学习の道 · 新幹線 24 駅**（路径隐喻：一站一课，从南向北）
- 气质：**轻松、干净、可信、略游戏感**（站点、虚线、海星）——**不是** 暗黑 RPG、不是幼幼园卡通糊墙
- 日语学习 App：**日文主、中文辅**（封面可仅一行中文副标或全无长段中文）

## 2. 课程结构 · 单元与课的关系（地图语义的基础）

### 2.1 标日上册骨架（与 PRD / 愿景一致）

```text
00 入門（五十音 · 发音与文字）     ← 课前能力关，不锁正课
────────────────────────────────
第1单元  01–04  小李赴日
第2单元  05–08  公司生活①
第3单元  09–12  箱根旅行
第4单元  13–16  公司生活②
第5单元  17–20  迎新春
第6单元  21–24  再见日本
────────────────────────────────
= 正课 24 課 + 入門 = 25 个「站」心智（全册满星 → 隐藏彩蛋）
```

### 2.2 两层导航（产品逻辑）

| 层级 | 用户心智 | 封面要不要表达 |
|------|----------|----------------|
| **6 单元** | 标日课本的 **6 大块**（每块 4 课） | **要** — 用地图上 **6 个站点圆**（编号 1–6）即可 |
| **24 课** | 每单元内 4 课，沿铁轨/虚线分布 | **封面不要** — 进 P1 学习地图再显示课名/pill |
| **00 入門** | 五十音前置，建议先完成 | **要** — 独立入口条，**不等于** 第 1 单元 |

### 2.3 地理隐喻（愿景 §2 · 07 线框）

- **四岛示意图**：北海道 · 本州 · 四国 · 九州（**不含**冲绳以南）
- **不是真实地图**：异形剪影 + **虚线通路** = 游戏关卡图
- **方向**：**从南向北** — 起点在 **四国**（南端）→ 九州 → 本州 → 北海道
- **新幹線**：金色/淡金 **虚线** 串联站点（工程与 `journey-home` 铁轨一致）

### 2.4 单元与「七色」的关系（`docs/封面配色说明.md`）

| 区块 | 谱色 | 用途 |
|------|------|------|
| 00 入門 | 紫 | 五十音条、课前必过 |
| 第1–6单元 | 赤→橙→黄→绿→青→蓝 | P1 目录区、通关后地图上色 |

**封面（P0）建议**：六站 **统一灰/未启程**（未登录进度），**不要** 六色单元 POP 卡——避免「还没学就满屏彩色课表」的噪音。单元色在 **进入学习地图后** 再随通关显现。

## 3. 地图在产品里要达成的目的

### 3.1 开机页（P0）上的地图 — 只答三件事

1. **隐喻**：「学标日 = 沿日本走一条有终点的路（24 站 + 入門）」  
2. **范围**：一眼知 **一共 6 个大段**（6 个圆），不是 24 个按钮挤在一屏  
3. **情绪**：**邀请出发**（主按钮「开始学习」），不是 **要求选课**

### 3.2 学习地图页（P1）上的地图 — 另做一稿（勿与封面混）

1. **导航**：点站 / 点课 → 进四关  
2. **进度**：灰 → 弱彩 → 金边通关（`dormant / active / cleared`）  
3. **目录**：下方或两侧 **单元色块 + 课名 pill**（探索地图稿）

### 3.3 明确不做的事（PRD + 用户拍板）

- 不做 **开放世界**、不做 **真实地理标注**（东京大阪文字堆在开机屏）  
- 不做 **单元手风琴 + 24 课列表** 在开机页  
- 不做 **HERO / PPT / SPLASH 开发水印**  
- 不用 **带 TRAVEL JAPAN 等水印的素材图** 当终稿  
- **漫画グルミ、小李、海星细则** — 属于课内奖励与 P1，**不是** 封面主视觉（最多品牌小图标）

## 4. 开机封面 · 必须体现的内容（清单）

按 **上 → 中 → 下** 三区；与 `过程讨论内容/19` 用户标注对齐。

| 区 | 元素 | 说明 |
|----|------|------|
| **顶 · 品牌** | App 图标 + **标日 あと学習** | 图标：可与 L0 品牌菱形一致；标题日文为主、字重清晰 |
| **顶 · 副标** | **一行**即可：`学习の道 · 新幹線 24 駅` | 可选更小字「标日初级上册 · 课后巩固」— **不超过两行** |
| **上中 · 前置** | **注音 · 五十音** 入口 | 紫系条/卡片 +「先修」或「建议先完成」；链到 `intro.html`；**不**与第 1 单元圆抢主视觉 |
| **中 · 主视觉** | 日本列岛 **示意剪影** + **虚线铁轨** + **6 个编号圆（1–6）** | 地图 **垂直居中**（屏中偏上亦可）；列岛 **浅灰填充**；六圆 **灰/未通关**；**无** 单元名、课名、POP 卡 |
| **下 · 行动** | 主按钮 **开始学习** | 珊瑚橙/品牌暖色、圆角、全宽约 85% |
| **下 · 辅助** | 一行灰字：`进入学习地图 · 选课续学` | 说明点按钮后去哪，**不是**第二套导航 |

**可选（拍板后）**：极淡底纹（纸感/和纹 **5% 透明度**）；**不要** 底栏三 Tab（进 P1 再出现）。

## 5. 开机封面 · 禁止出现的内容

| 禁止 | 原因 |
|------|------|
| 24 课课名、第 N 課 列表 | 信息过载；属 P1 |
| 六单元展开卡、4/4 课、单元通関文案 | 属 P1 目录 |
| 六色单元大色块 + 中文场景长文 | 像课表不是封面 |
| 探索地图整图（带彩色 POP） | 用户明确：那是 **点学习之后** 的页 |
| 海星/海胆/金星统计、进度条 | 属课内与マイ |
| グルミ 大图、单元四格漫画 | 属通关奖励，非开机 |
| 小李漫画人设 | 全景治理：小李仅叙述，不当画面主角 |
| `HERO` `PPT` `SPLASH` `TRAVEL JAPAN` | 开发/素材水印 |
| 英文 Slogan 占主视觉 | 日语学习产品首屏日文优先 |
| 示意图灰块/SVG 线框当交付终稿 | 用户要 **390×844 可审稿的真稿**（插画或精修合成） |

## 6. 美工 / AI 出图 · 提示词提纲（模块化复制）

> 使用时按 **§6.1 全局** + **§6.2 分区** 拼接；出 **竖屏 390×844**（或 9:19.5）安全区，留顶底各 ~8% 给系统栏。

### 6.1 全局设定（每条提示词都带）

```text
【产品】中国初中生使用的《标准日本语·初级上册》课后巩固小程序开机页，微信内竖屏第一眼。
【气质】干净、温暖、可信的教育 App；轻微游戏化（站点+虚线路径），非 RPG、非幼幼园贴纸风。
【布局】上品牌、中地图、下单一主按钮；大量留白；奶油白/暖白背景 #FFF8E1 附近。
【禁止】课名列表、单元折叠菜单、英文旅游水印、开发角标、写实东京地图标注、角色养成 UI。
【尺寸】390×844 px，安全区居中，适合手机全屏截图审稿。
```

### 6.2 分区模块

#### A · 品牌区

```text
顶部居中：圆角 App 图标（暖橙渐变底+深蓝菱形标记），下方大字「标日 あと学習」现代哥特/无衬线日文，
副标题一行小字「学习の道 · 新幹線 24 駅」灰色 #666，字间距舒适，不拥挤。
```

#### B · 五十音前置条

```text
品牌与地图之间：横向圆角卡片，淡紫底 #F3E5F5，左侧白底「あ」字标，中间「注音·五十音」紫色粗体，
副文「学习前置·发音与文字（建议先完成）」小灰字，右侧小按钮「先修」实心紫。精致、扁、不抢主图。
```

#### C · 地图主视觉（核心）

```text
画面正中：日本四岛抽象剪影（北海道·本州·四国·九州），浅灰蓝填充 #CFD8DC，柔和投影，
略卡通化示意而非真实地理。一条金色虚线「新干线」从南端四国向北蜿蜒至北海道。
仅 6 个圆形站点编号 1–6，沿铁路分布，全部为「未启程」状态：浅灰圆、白字编号、无彩色填充。
不要任何文字标签（不要「第1单元」「小李赴日」）。不要 24 个小点。不要彩色信息卡片。
地图占屏高约 38–42%，水平居中，视觉重心在屏幕几何中心偏上 5%。
```

#### D · 行动区

```text
底部：宽大圆角主按钮「开始学习」，珊瑚橙到暖红渐变 #FF8A80，白字加粗，轻微阴影。
按钮下 12px 一行说明灰字「进入学习地图 · 选课续学」。无底部 Tab 栏。
```

### 6.3 负面提示词（Negative prompt · 通用）

```text
 cluttered UI, lesson list, 24 buttons, accordion menu, anime character full body,
 photorealistic map, city names, English travel poster, watermark, logo mockup text,
 dark RPG, neon cyberpunk, childish clipart overload, three column textbook layout,
 progress bar, star rating dashboard, comic strip, duplicate titles
```

### 6.4 风格参考方向（文字描述，非抄图）

| 可参考 | 不要抄成 |
|--------|----------|
| 日本教育 App 柔和渐变 + 纸感留白 | 多邻国重度游戏化 |
| 新干线/铁路旅图 **示意** | JR 官方时刻表写实 |
| `探索地图.png` 的 **铁轨+列岛构图**（P1） | 其彩色 POP 与课名（仅 P1） |
| `日本地图框架.pptx` 列岛剪影 **形态** | PPT 水印与 TRAVEL 字样 |
| 品牌七色在 **P1 目录** 使用 | 开机六站全彩 |

## 7. 验收标准（产品策划拍板用）

| # | 标准 | 通过 |
|---|------|------|
| M1 | 3 秒内能回答：这是什么 App、下一步点哪 | ☐ |
| M2 | 小朋友不读长文也能找到「开始学习」 | ☐ |
| M3 | 家长觉得「正规标日辅助」而非野游游戏 | ☐ |
| M4 | 地图只表达 **6 单元 + 一条路**，无课名噪音 | ☐ |
| M5 | 五十音入口可见且像「建议先修」而非隐藏设置 | ☐ |
| M6 | 与 P1 探索地图 **一眼可区分**（封面更空、更品牌） | ☐ |
| M7 | 无禁止项（§5） | ☐ |
| M8 | 交付为 **可审稿高清稿**（非工程线框） | ☐ |

**待拍板项**（纪要 19）：S1 是否每次打开都显示封面；S2 底栏「课文」是否永远回 P1；S3 五十音条是否可收起。
