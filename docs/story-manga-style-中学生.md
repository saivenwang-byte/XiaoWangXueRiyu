# 单元四格 · 画风备选（中学生漫画）— 暂缓

> **2026-05-22**：你更偏好 **上一版**（水彩绘本感 + 三视图一致）。本文档保留作备选，**当前出图不走这条**。

## 目标读者

标日自学 / 中学段 → 画面像 **杂志连载的轻松系日漫**（《ゆるキャン△》《日常》一档），不是 3–6 岁绘本。

## 要 / 不要

| ✅ 要 | ❌ 不要 |
|--------|--------|
| 清晰墨线、适度赛璐璐明暗 | 水彩晕染、蜡笔质感 |
| 角色比例偏少年漫画（头身约 1:3–1:4） | Q版婴儿比例、巨大婴儿眼 |
| 场景透视清楚、物件可读 | 过度粉嫩、满屏星星闪粉 |
| グルミ 仍用三视图造型 | 换成真人少女或 losing 纸垂绳 |

## 负面提示词（每格必带）

```text
children's picture book, kindergarten, preschool, chibi baby, super deformed,
watercolor storybook, soft pastel blur, sparkly stars everywhere, crayon texture,
anime moe blob, dot eyes only, infant proportions
```

## 正面风格词（每格必带）

```text
Japanese manga panel for junior high readers, clean ink outlines, soft cel shading,
slice-of-life anime background, natural lighting, readable props, 16:9 landscape
```
