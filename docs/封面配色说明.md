# 封面配色说明 · 学習の道

> 真源：`js/data/curriculum-catalog.js` → `CURRICULUM_PART0_THEME` + `CURRICULUM_STAGE_THEMES`  
> 课内 UI 与单元块 **共用同一套 accent / pageBg**，避免「封面一套、上课又一套」。

---

## 1. 为什么用「赤橙黄绿青蓝紫」

| 考虑 | 做法 |
|------|------|
| **七个层次好记** | 00 入門 = **紫**（课正文之前）；第 1–6 单元 = **赤→橙→黄→绿→青→蓝**，与标日 6 单元一一对应 |
| **日本风 Web** | 不用高饱和纯色的「彩虹海报感」；**中饱和 accent + 高明度渐变底**（类似乐天 / 教育 App 的柔和分区） |
| **区分单元** | 纵向滚动时，**整块底色**一眼知「现在在第几单元」，不靠灰阶 |
| **与地图共存** | 地图区保持浅白半透明；色块在 **目录区**，不抢站点 |

中国传统七色在这里是 **谱系标签**（单元角标显示「赤」「橙」等），不是硬套五行寓意。

## 2b. 探索地图 · 四层状态（v67）

| 状态 | 地图 | 下方目录 |
|------|------|----------|
| **未进入** `dormant` | 站点隐藏，仅见铁轨 | 整块灰、课次灰 |
| **START** | 00 入門 / 第1单元 珊瑚脉冲 | 00 条或第1单元标 START |
| **进行中** `active` | 站点弱彩圆环 | 单元弱彩（降饱和） |
| **通关** `cleared` | 金边+单元色实心 | 全彩 + ★単元完了 |

全六单元通关后：地图陆地 `is-land-revealed`（向参考图渐进上色，后续可加分区色）。

每单元含：`bg`（封面块渐变）、`pageBg`（进课后整页底）、`border`（描边）、`primaryDark`（按钮按下）。

## 3. 页面上怎么用

```text
顶栏（全 App）     → 米黄底 #FFF8E1 + 海军字 #1E3A5F（48px）；主按钮仍珊瑚 #FF8A80
封面卡片顶         → 仅「学習の道 · 新幹線 24 駅」+ 進捗（不再重复大标题）
─────────────────
[开机封面]         → 全幅探索地图（`home-splash.js`）· 点「开始学习」进 App
[紫] 00 入門条     → 课文页最上：发音与文字
[赤] 第1单元块     → 六单元手风琴（无地图条）· 日文主题 + 右侧中文
[橙] 第2单元块
…
[蓝] 第6单元块
（开发提示 · 小字灰）
```

| 元素 | 配色规则 |
|------|----------|
| 单元整块 | `background: theme.bg` |
| 左竖条 / 标签 | `theme.accent` |
| 课 pill 圆点 | `theme.accent`（可点课） |
| 中文副标 | 全局 `.zh-annotation`（#999 · 0.78em） |
| 喇叭 | 全局 `.btn-speak-icon`（`--mvp-muted`） |

### 00 入門页 · 顶栏与标题（`body.intro-body`）

| 部位 | 色 | 说明 |
|------|-----|------|
| 页面底 | `#f6f0fc` | 与首页 00 紫条 `pageBg` 同系 |
| 顶栏底 | `#fff` + 底边 `#d1c4e9` | 与内容区拉开，避免蓝底蓝字 |
| 「← 標日 あと学習」 | `#5e35b1` | 可点链接，对比 ≥ 正文 |
| 标题「00 入門」 | `#4527a0` | 与 accent `#8e6ec0` 同谱 |
| 主按钮 | `#5e35b1` 底 + 白字 | 不用 app `--primary` 海军蓝 |

实现：`css/mvp.css` 中 `body.intro-body` 块；真源仍读 `CURRICULUM_PART0_THEME`。
