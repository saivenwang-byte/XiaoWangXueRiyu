# 真机预览 / 浏览器出现 503 时

## 含义

**503 Service Unavailable** = 对方服务器暂时过载或未就绪，**不是**课内 JS 写错。

## 最常见原因（本项目）

| 情况 | 处理 |
|------|------|
| 真机框选了 **「公网」** | 改选 **「本机 8765」** 或点壳内 **「改用本机 8765」** |
| 本机 **8765 未启动** | 双击 **`重启本地服务.bat`** → 再开 **`Cursor真机持续预览.bat`** |
| 打开了 **Netlify 旧址** | 勿用 `qingjing-biaori.netlify.app`；用 GitHub Pages 或本机 |
| GitHub Pages 偶发繁忙 | 等 1～2 分钟或先用本机预览 |

## 正确链接

- **开发（推荐）**：`http://127.0.0.1:8765/cursor-miniapp-phone.html?live=1&src=local`
- **本机全宽**：`http://localhost:8765/index.html?v=` + `CACHE_VER`
- **学员公网**：`https://saivenwang-byte.github.io/XiaoWangXueRiyu/index.html?v=` + `CACHE_VER`

## 一键检查

双击 **`检查本机与网关.bat`**
