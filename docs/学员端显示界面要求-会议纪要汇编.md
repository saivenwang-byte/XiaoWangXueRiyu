> **用途**：把已拍板的「显示 / 布局 / 导航 / 真机目视」要求收成**一份真源**。  
> **首要执行（双界面）**：[学员端双界面显示标准-首要执行.md](./学员端双界面显示标准-首要执行.md) — **① Cursor 390×844 锁定壳** · **② 微信可转发 https 链**（交付判据）。  
> **强制**：在 **Cursor 内置浏览器刷新**、**真机框目视**、**声称 UI 已修好** 之前，**必须先重读本文 §0 清单** + 上条双界面标准 §二（Agent 见 `.cursor/rules/ui-display-reread-before-preview.mdc`）。  
> **维护**：新增拍板项 → 写入对应 § + 回链纪要文件；勿只改代码不写本文。

## 0. 刷新 / 目视验收前 · 30 秒清单（每次必对）

在 `打开双通道预览.bat` 或 Cursor 浏览器 **硬刷新**（`?v=` 与 `js/share-wechat.js` 的 `CACHE_VER` 一致）之前，逐条默念：

| # | 检查项 | 通过标准 |
|---|--------|----------|
| R1 | **底栏三 Tab 完整** | **课文** · **笔记**（人像+二字）· **注音**（あ 框）；无残缺标签、无 `tex` 乱码 |
| R2 | **底栏语义** | 课文 = 学习地图 `#view-home`；笔记 = 知识笔记 `#view-me`；注音 = `intro.html`（不在底栏塞设置/分享） |
| R3 | **高度预算** | 主内容 ≈ `100dvh − 顶栏 − 底栏`；顶栏/底栏固定，中间区域滚动（见 §1） |
| R4 | **笔记页** | 中间列表占满；**设置 · 分享链接** 在页底 dock、**默认折叠**；列表区有纵向滚动条 |
| R5 | **入門·基础 SK-01** | **注音全景表**为主；少边框套娃；非长文瀑布（见 §6） |
| R6 | **课内提示** | 单行「提示：」**直接展示全文**，无右侧 ▾；多行才折叠 |
| R7 | **五关折叠** | 收起 = 概述（不截断）；展开 = **日文在上、中文在下** |
| R8 | **平面轻量** | 笔记/课内块：**无阴影、无渐变**（规范 19 · 全局审查 20260525） |
| R9 | **编码** | `index.html` / `intro.html` 须 UTF-8 可读；**禁止** PowerShell 无 BOM 批量改中文文件 |
| R10 | **双界面（交付）** | **①** `cursor-miniapp-phone.html?live=1` **390×844 壳内**目视；**②** 微信 `index.html?v=CACHE_VER` 可转发 |
| R11 | **课内无第二底栏** | 仅全局底栏三钮；课内靠顶栏 Tab；内容区单独滚动 |
| R12 | **壳尺寸不变** | Cursor 验收时切换课文/笔记/注音/课内/五关，**外框仍 390×844**（见双界面标准 §二 L3） |
| R13 | **会話模块冻结** | **用户未明确要求不得改**；保持 ABC/展开收起/本句听完（见 §4.1） |

未勾满 **不得** 向用户说「界面 OK / 可发布」。

## 4. 笔记页（`#view-me` · 底栏「笔记」）

> 真源：`过程讨论内容/19-20260525-笔记板块规范.md` · `docs/笔记与小程序UI-全局审查-20260525.md`

| 要求 | 说明 |
|------|------|
| **命名** | 原「我的」→ **笔记**；底栏中间钮 |
| **主区最大化** | 计时 + 用法说明 + 各课手风琴 = **主滚动区**，占 `100dvh − 顶栏 − 底栏` 绝大部分 |
| **设置沉底** | **设置 · 分享链接**、清数据、复制链接 = **页内底部 dock**；**默认折叠**；非常用项不挤占列表首屏 |
| **滚动** | 主列表 **可见纵向滚动条**（细条即可）；外层 `#view-me` 不整页双滚动 |
| **结构** | 每课：课本 / 老师 / 学霸 / 薄弱 / 关联 / 我的笔记（六维）；手风琴**一课只开一个**（与单元规则一致） |
| **学霸块** | 「参考长表」「易错提醒」须能展开见内容；**不用**嵌套 `<details>`（微信 WebView） |
| **视觉** | 六色块 = 五关色板令牌；展开无阴影；计时数字海军蓝 |
| **折叠高度** | 折叠态一屏尽量 **≥4 课** 标题行可见（真机 390×753） |

### 4.1 会話模块 · 用户冻结（2026-05-26 截图拍板）

> **用户原话**：「会話模块我没有要求你改，保持原来的要求不变。」  
> **ABC 真源（全课 1–24）**：[会話-ABC三答-产品真源.md](./会話-ABC三答-产品真源.md) — **必读**  
> **冻结索引**：[MVP-UNIT1-LESSONS-1-4-FREEZE.md](./MVP-UNIT1-LESSONS-1-4-FREEZE.md) · [项目知识库-U1L1-MVP-UI统筹审查.md](./项目知识库-U1L1-MVP-UI统筹审查.md) §G

#### 对话在讲什么

| 部分 | 含义 |
|------|------|
| **发起句（NPC）** | **标日课文**里的问话/话题：`opener` 日文 + 下方中文 |
| **A** | **课文标准答**：第 1 行日文、第 2 行中文翻译（两行正文） |
| **B / C** | **场景变体**：第 1 行日文、第 2 行中文、第 3 行中文 **场景说明**（对谁、何场合、与 A/B/C 差异） |

**禁止**：删掉或合并 B/C；把第三行说明藏进只有点开才见的折叠提示。

#### 界面壳（与 ABC 内容无关者不改）

| 保持不动 | 说明 |
|----------|------|
| **场景折叠条** | 序号 + 场景徽章 + **展开/收起** |
| **ABC 三卡** | 标签 A「课文」/ B「变体B」/ C「变体C」+ 听录播/评分 |
| **本句听完 ✓** | 每场景底部；与传送链底栏分工不变 |
| **评语** | 句下全宽（记忆库 §3） |

**允许的全局改动（仅当用户另开口）**：五关底栏 dock、中间滑槽、顶栏 Tab——**不**改 ABC 三行/三卡逻辑。

## 5. 课内 · 五关（第 1–24 课 MVP）

| 要求 | 说明 | 纪要 / 归档 |
|------|------|-------------|

| **折叠概述** | 收起显示完整概述，**禁止** 36 字截断省略号 | 用户拍板 · `hyouga-fold-overview.js`（**不**改会話 §4.1 内 ABC 区） |

| **会話 ABC** | 见 **§4.1 冻结**；不得擅自改 A/B/C 条、变体标签、🔊🎤 行 | `dialogue-gate` · `MVP-UNIT1-LESSON1-FREEZE` |

| **彩蛋** | 课文通关→课文彩蛋；单元齐→单元彩蛋；全册→隐藏彩蛋；内容与触发逻辑**不变** | 用户拍板 2026-05-28 |

## 6. 00 入門 · 五十音（`intro.html`）

| 要求 | 说明 |
|------|------|
| **产品形态** | **五十音能力全景模型**（発音と文字），不是字典式长滚动 |
| **三层 Tab** | **基礎 / 協調 / 核心**（上日文下中文，**一行**；动作收 `title`） |
| **基础层** | **仅 SK-01 注音全景表**（行×段）；易错 **不在** 基础重复 |
| **核心层** | 三种字 + 寒暄 + 进站第1課（块可折叠，见下行细化） |
| **表格式** | 阶段一 = **总表一屏**（可横滑/内滚）；**不要** 单字无限瀑布 |
| **SK-01 表区** | 档案白卡内 **留出边框**（约 10px inset）；表顶与 Tab 间有间距；**单层** scroll 细边+圆角；取消双层套娃；**格线保留** |
| **SK-01 标题** | 保留 `SK-01` +「注音全景」 |
| **协同层** | SK-04～07 四色块；**每块四周留出边框**、间距 8px；校验行下无大块留白 |
| **核心层** | SK-08～10 可折叠块；与协同同 inset；**完成钮在卡片外** |
| **点字** | 点假名 → 底栏「先生のひとこと」（記憶/注意/🔊） |
| **少文字** | 去 Hermes 长文、域标签；🔊 浅灰不抢假名 |
| **顶栏** | 入門 CI 紫色系；完成回课文首页 |
