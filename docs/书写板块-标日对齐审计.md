## 4. 已知字形差异说明

笔顺路径来自 animCJK 中线，经 2a/2b 副线清洗后与教材**笔数**一致；
个别字（如 **す**）路径折线可能与教科书印刷体略有差别，以**笔序与笔数**为准。
顶栏标题用系统字体，格内为笔顺路径，属正常「字形来源不同」而非笔序错误。

## 5. 片假名 / 平假名对照（屏上）

- `KANA_STROKE_GUIDE` 键为**平假名**；片假名模式下格内仍绘平假名笔顺路径（与标日教辅一致）。
- **单字顶栏**：片假名大字 + 平假名副标 + 灰字「笔顺按平假名（标日）」；平假名模式仅显示该字。
- **清音表**：片假名格大字为カタ、副行小字为对应ひら；行标仅「あ行」等日文，无「阿行」中文列。
