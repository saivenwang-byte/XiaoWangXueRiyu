## 十一、统筹建议

### 立即可改（不依赖设计师）

| 优先级 | 项目 | 改动量 |
|:--:|------|:--:|
| P0 | 卡片间距 6px→16px | 改 CSS 3 处 |
| P0 | 卡片内边距 10px→20px | 改 CSS 3 处 |
| P0 | 启用封面地图 + 新干线虚线 | 移除 `display:none` + 加 SVG 路径 |
| P1 | 单元色条从左边框改顶部 | 改 CSS + journey-home.js |
| P1 | 选项选中态高亮 | 改 quiz-gate.js |

### 需要讨论后再改

| 项目 | 说明 |
|------|------|
| 主色从 #d32f2f→#FFB7C5 | 用户已习惯红色，需评估切换成本 |
| 6单元色从艳丽→柔和 | 影响面不小但可控 |
| 底部导航三Tab | 当前是 课文/笔记/注音，VI 是 ホーム/復習/マイ，需统一 |

### 依赖设计师出图

| 项目 | 说明 |
|------|------|
| 封面探索地图底图 | 日本四岛剪影+城市标注 |
| ぐるみ角色 | 插画师 |
| 24课彩蛋图 | 每课场景插画 |
| 6单元拼图 | 2×2拼接 |
| 终极彩蛋 6×4 | 24张景点 |

### 终极建议

**不要一次性全改。分三批：**

```
第一批（本周·纯代码）：间距+圆角+封面地图+新干线虚线
  → ROI 最高，用户立刻看到"旅途感"

第二批（下周·色系+组件）：单元色+按钮色+卡片色条
  → 需要我和你讨论确认色板

第三批（按节奏·插画）：24课场景图+ぐるみ
  → 你找设计师，我给规格
```
