# 产品设计规范 · 平面轻量 UI（包体与歧义控制）

> **地位**：标日 H5 / 未来微信小程序 **默认视觉铁律**。用户未特别声明「要渐变/阴影/立体」时，一律遵守本文。  
> **关联**：[DESIGN-L0](./DESIGN-L0-图标与人设全景.md) · [L3图标全景](./产品设计规范-L3图标全景清单.md) · [U1L1 UI统筹](./项目知识库-U1L1-MVP-UI统筹审查.md) · [课内单词标黄](./项目知识库-课内单词标黄范式.md) · `js/hyouga-glyphs.js`

---

## 1. 为什么要平面轻量

| 目标 | 做法 |
|------|------|
| **控制包体** | 少用大阴影、多层渐变、粗描边立体 SVG、重复装饰图 |
| **减少误判** | 状态只用 **纯色块** 区分（白 / 黄等），不用渐变过渡 |
| **通俗可认** | 图标=细线平面符号（遥控器/手机常见形），非 3D、非复杂插画 |

---

## 2. 禁止项（除非用户书面要求）

- `linear-gradient` / `radial-gradient` 用于 **课内行背景、标黄、按钮、折叠高亮**
- `box-shadow` / `text-shadow` 用于 **卡片、按钮、图标容器**（展开态外发光环亦避免）
- 描边 `stroke-width ≥ 2` 的密集 SVG 装饰（课内 L3 默认 **1.5**）
- 拟物、立体、多色填充图标；emoji 作 **课内列表主图标**（顶栏 Tab 等已改为 SVG 的除外）

---

## 3. 允许项

- **纯色** 背景：`#fff`、`#fff8e1`（淡黄注意）等
- **1px 实线** 边框区分模块
- **线性 SVG** 单色 `currentColor`，简单路径
- 展开/折叠用 **字符箭头**（▸ ▾）或小方块按钮，不用复杂 chevron 图

---

## 4. 课内单词 · 标黄（与范式联动）

| 状态 | 行背景 | 展开箭头 ▸ |
|------|--------|------------|
| 普通词 | **白** `#fff` | 无箭头 |
| **注意**（`VOCAB_WARN_IDS`） | **淡黄** `#fff8e1` + 左 3px `#ffe082` | — |
| 有延伸/活用 | 白或黄（仅 `注意` 行黄底） | **黄底** `#ffe082`、字色 `#e65100` |
| 展开区 | 淡黄纯色 `#fff8e1` | 无渐变 |

**禁止**：白→黄渐变行背景（学员无法判断「算不算注意」）。

---

## 5. 第1課五关（U1L1 MVP）

- 底栏传送链按钮：**纯色** 底，无渐变
- 折叠 `[open]`：仅 **边框色** 变化，无 `box-shadow` 外环
- 列表折叠条：**序号 + 标题**，无行内装饰 SVG（顶栏 Tab 保留模块图标）
