## 3. 隐私与用户隐私保护指引

1. 公众平台 → **设置 → 服务内容声明 → 用户隐私保护指引**  
2. 隐私政策 URL（与业务域名一致）：  
   `https://saivenwang-byte.github.io/XiaoWangXueRiyu-v2/privacy.html?v=387`
3. **收集的信息**（与隐私政策一致，须勾选/填写对应项）：
   - 本地学习进度、笔记、测验记录（仅设备本地，不上传）
   - 麦克风（用户主动跟读时，本地处理，不上传录音）
4. **开发者处理规则**：不上传至服务器；用户可「清除本机学习数据」

---

## 4. 未成年人保护

- 产品定位含**在校日语学习的初中生**（多数未满 18 周岁）
- 不收集未成年人身份信息、不实名注册
- 麦克风须监护人知情后由学员在本人设备授权
- 审核备注可写：「主要面向初中生课后练习，家长可陪同使用」

成年自学者同样适用，不额外收集信息。
