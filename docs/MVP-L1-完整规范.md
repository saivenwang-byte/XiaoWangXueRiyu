- 箭头 `▶` 旋转90°表示展开

## 四、各关内容格式

### 4.1 单词关

```
每行一词：
┌────────────────────────────────────┐
│ 1  わたし [代] 我   ⓪    🔊      │
│    (kana=わたし jp=私 pos=代)      │
└────────────────────────────────────┘
```

| 字段 | 格式 | 字号 |
|------|------|------|
| 序号 | 数字 | 12px |
| 日语 | kanji + ruby 注音 | 14px |
| 品词 | [代] 灰色标签 | 10px |
| 中文 | 灰色释义 | 11px #78909c |
| 声调 | ⓪①② 标记 | 11px |
| 喇叭 | 🔊 按钮 | 18px |

### 4.2 会话关

```
折叠横条模式：
┌──────────────────────────────────┐
│ ▶ B：はい、小野です。李秀麗...  │  ← 折叠条（展开前）
├──────────────────────────────────┤
│ はい、小野です。李秀麗さんですか。│  ← 展开后完整日文
│ 是的，我是小野。你是李秀丽吗？    │  ← 中文翻译
│ 🔊 点击喇叭跟读                  │
│ 👨‍🏫 先生提示：こちらこそ不能省略   │  ← 可折叠提示
└──────────────────────────────────┘

场景头：🛬 成田空港  出迎え
开场白：单独一行（说话人A）灰色背景
```

### 4.3 语法关

```
每节点一卡：
┌──────────────────────────────────┐
│ 〜は〜です                        │  ← 日文标题 (16px)
│ 名1是名2（名词谓语句・肯定）       │  ← 中文标题 (12px #999)
│                                  │
│ 日文说明：                        │
│ 「AはBです」で「A＝B」を表します   │
│                                  │
│ 中文说明：                        │
│ 名词谓语句肯定形。「は」提示主题    │
│ 读作「わ」                        │
│                                  │
│ 例句：                            │
│ わたしは学生です。  🔊            │
│ 我是学生。                        │
└──────────────────────────────────┘
```

### 4.4 作業关

```
每题一屏：
┌──────────────────────────────────┐
│ 問題 1/12          ← 上一题 → 下一题 │
│                                  │
│ わたし＿＿＿学生です。   🔊      │  ← 题干 + 喇叭
│                                  │
│ ○ は  ○ が  ○ の  ○ に       │  ← 选项（各带🔊）
│                                  │
│ [送信]                           │
└──────────────────────────────────┘

回答后：
✅ せいかい！ ／ ❌ ざんねん
👨‍🏫 提示：助词「は」提示主题         ← 知识卡片
[次へ]
```

### 4.5 拡張关

```
分类折叠：
┌──────────────────────────────────┐
│ ▶ 🔊 発音ポイント               │  ← 默认展开第一个
│ ▶ 📖 文法まとめ                  │
│ ▶ 🔮 活用予告                    │
│ ▶ 🎩 敬語レベル表示               │
│ ▶ 📜 語源メモ                    │
│ ▶ 💬 会話のキーポイント           │
│ ▶ 🎭 ロールプレイ課題             │
│ ▶ 📌 これまでのまとめ             │
│                                  │
│ [完了 → ホーム]                   │
└──────────────────────────────────┘
```

## 六、设计 Tokens

### 颜色

| Token | 值 | 用途 |
|-------|-----|------|
| `--c-primary` | `#d32f2f` | 主色、选中态、红色提示 |
| `--c-book` | `#1565c0` | 课本维度蓝 |
| `--c-teach` | `#e65100` | 教师维度橙 |
| `--c-star` | `#7b1fa2` | 学霸维度紫 |
| `--c-surface` | `#fff` | 卡片背景 |
| `--c-bg` | `#f5f5f5` | 页面背景 |
| `--c-text` | `#333` | 主文字 |
| `--c-text2` | `#999` | 次要文字 |
| `--c-border` | `#eee` | 分割线 |

### 字体

| 层级 | 字号 | 粗细 | 颜色 | 用途 |
|------|------|------|------|------|
| H2 课号 | 13px | 400 | #999 | 第N課 |
| H3 标题 | 15-18px | 700 | #333 | 日文标题 |

| Body ZH | 11px | 400 | #78909c | 中文翻译 |
| Label | 12px | 700 | #666 | 区块标签 |
| Hint | 11px | 400 | #b0bec5 | 提示文字 |
| Tag | 11px | 400 | — | 知识标签 |
| Sub | 9-10px | 400 | #999 | 次要标注 |

### 间距

| 元素 | 值 |
