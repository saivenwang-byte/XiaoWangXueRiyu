## 一、一句话摘要（可贴「隐私政策简述」）

本小程序/H5 为日语初级课后练习辅助工具，**不收集可识别个人身份的信息**；学习进度与笔记仅存用户设备本地；跟读功能在用户主动操作时本地使用麦克风，**不上传录音**；主要面向在校学习日语的初中生，建议家长知情同意后使用。

## 三、我们收集哪些信息（对照勾选）

### 3.1 不收集（明确写「否」或不勾选）

- 姓名、昵称（无账号体系）
- 手机号码
- 身份证号码
- 精确地理位置 / GPS
- 通讯录、相册（除用户主动复制链接到剪贴板外）
- 支付信息（无付费）
- 生物识别信息

### 3.2 仅在设备本地使用、不上传服务器

| 信息类型 | 收集方式 | 用途 | 存储位置 | 保留期限 |
|----------|----------|------|----------|----------|
| 学习进度（关卡、测验对错） | 自动（使用产生） | 续学、地图、复习 | 浏览器 `localStorage` | 直至用户清除 |
| 自学笔记 | 用户输入 | 个人记录 | 同上 | 同上 |
| 错题列表 | 自动 | 间隔复习 | 同上 | 同上 |
| 五十音书写完成标记 | 自动 | 书写页进度 | 同上 | 同上 |
| 入門（五十音）进度 | 自动 | 入門关状态 | `hyouga_intro_progress_v1` | 同上 |
| 学习时长（分钟） | 自动 | 界面展示学习日数 | `hyouga_study_time` | 同上 |
| 界面偏好（如是否显示中文） | 自动/用户切换 | 显示设置 | `hyouga_mvp_v1` | 同上 |

**开发者承诺**：上述数据**不传输至开发者自有服务器**，无用户账号库。

### 3.3 设备权限（须声明）

| 权限 | 是否使用 | 触发时机 | 数据处理 |
|------|----------|----------|----------|
| **麦克风** | 是（可选） | 用户点击会話「跟读/录音」 | 仅本地发音比对与即时反馈，**不保存、不上传**录音文件 |
| 扬声器/媒体 | 是 | 播放课文朗读 | 不涉及个人信息 |
| 剪贴板 | 是（可选） | 用户点击「复制学习链接」 | 仅写入公网学习 URL |

## 六、未成年人保护要点

- **用户群体**：含在校学习日语的**初中生**（多数未满 18 周岁）  
- **措施**：不实名注册；不收集未成年人身份信息；学习数据仅存本机；麦克风需本人设备授权，**建议监护人知情同意**  
- **家长权利**：可陪同使用、可在应用内「清除本机学习数据」、可关闭麦克风权限  
- **成年用户**：同样适用，不因年龄额外收集信息  

---

## 七、用户权利

| 权利 | 实现方式 |
|------|----------|
| 查询 | 学习数据均在应用内可见（进度、笔记） |
| 删除 | マイ → 设置 → **清除本机学习数据**；或清除浏览器/微信站点数据 |
| 撤回授权 | 系统设置关闭麦克风；停止使用即可 |
| 注销账号 | 不适用（无账号） |

---

## 八、安全与存储

- 数据存储于用户终端，由操作系统/浏览器/微信 WebView 管理安全  
- 开发者不运营中央数据库保存学员档案
