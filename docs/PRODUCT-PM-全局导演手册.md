## 2. 四层逻辑拆解（每次需求必做）

```text
用户表述
  ↓
教学逻辑   「零基础先建立音表地图，再分块，再开口」— 纪要 10、13
  ↓
产品逻辑   「入門=能力全景三层；首页=路径图；课内=四关」— 范式、11
  ↓
界面逻辑   「一层导航只表达一件事；中日双语一行」— v60
  ↓
实现边界   「动 intro / catalog / journey；不动 14/16/18 课内」
```

### 2.1 全 App 信息架构（固定）

```text
┌─────────────────────────────────────────┐
│  L1  目录层   学习の道 · 00入門 · 24站   │
├─────────────────────────────────────────┤
│  L2  学习层   単語 → 会話 → 文法 → テスト │
├─────────────────────────────────────────┤
│  L3  督导层   マイ（进度·薄弱·天数）      │
└─────────────────────────────────────────┘
```

**入門** 是 L1 的 **隐藏关**：不锁课、可随时查发音词典。

### 2.2 入門 · 五十音能力全景（固定）

| 层 | 日文 | 中文 | 用户动作 | Skill |
|----|------|------|----------|-------|
| 基礎 | 基礎 | 基础 | 点表听音 | SK-01 + 💡 |
| 連動 | 連動 | 联动 | 分色块练（音变分块） | SK-04～07 |
| 核心 | 核心 | 核心 | 跟读·进站 | SK-08～10 |

**作业法**（用户不必看到全文）：全景 → 分块 → 合成 → 开口 → 第1課。

## 4. 排版与视觉规约（美工）

### 4.1 层次

| 层级 | 用途 | 字级参考 |
|------|------|----------|
| L0 | 页标题 | 17px 粗 |
| L1 | 块标题 / 日文层名 | 15px 粗 |
| L2 | 假名格 | 16–18px |
| L3 | 罗马字/辅助 | 10px 灰 `#78909c` |
| 静音 | 说明 | 能不用就不用 |

### 4.2 色

| 用途 | 色 |
|------|-----|
| 单元/阶段 | `CURRICULUM_STAGE_THEMES`（首页已用） |
| 入門·易错 | 橙 `#ffe0b2` |
| 浊音 | 紫 `#e1bee7` |
| 拗音 | 青绿 `#b2dfdb` |
| 长促拨 | 粉 `#f8bbd0` |
| 主色 | `--mvp-primary`（按钮、当前层） |

### 4.2.1 单元手风琴（UX 铁律 · 2026-05-21）

- **课文首页**与**我的**页：同一时刻**最多展开 1 个单元**。  
- 用户点开另一单元时，先前展开的单元**必须自动收起**，避免列表过长、整屏只能上下滑。  
- 实现：`details` + `toggle` 互斥（`journey-home.js` · `app.js`）。  
- 详见：`docs/首页与マイ-信息架构与视觉统筹（讨论稿）.md` §0.1。

### 4.3 组件规则

| 组件 | 规则 |
|------|------|
| ▾ | 仅表示折叠；16px 图标区，44px 触控 |
| ★×4（单元） | 一课一星；满则金，未满灰 ○ |
| 👁 | 仅单元四课全通后，**行最右**，点开 L2 条带 |
| 🔊 | 浅灰 `#b0bec5`、小、右下角；**假名是主角** |
| 💡 | 点字开「先生のひとこと」底栏；象音→記憶→注意 |
| 三层 Tab | **上日文下中文**，禁止两行都像层名 |
| 进度 | ●●○ + `n/3`，禁止再叠 ①～⑤ 长文案 |

### 4.4 美化优先级（P0→P2）

1. **消歧**（导航、双语一行）  
2. **对齐**（栅格、左栏分色块、喇叭位置）  
3. **留白**（块间距 10–12px，一屏一事）  
4. **插图**（象音 emoji → 后续插画）  

invoke `design-review` 做前后截图对比。
