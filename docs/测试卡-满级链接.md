> **记住这一条**：验收插画与三级彩蛋时，必须用带 `testcard=1` 的链接。  
> 否则要手动打四关、攒金星，测不全图片与彩蛋。

---

## 这一条链接做什么

**开发者电脑竖屏**：双击 `开发者竖屏验收.bat` → `dev-phone-preview.html`（iframe 内真 App，满配）

打开后自动变成「满级存档」：

| 项目 | 状态 |
|------|------|
| 第 1–24 课 | 四关全通、四金星 |
| 六单元 L2 条带 | 四格全亮、可点 👁 / 单元彩蛋 |
| 每课 L1 | 课行右侧 44px 缩略图可点 |
| L3 通关 24 景 | 已解锁（首页礼物 / `?egg=ultimate`） |
| 目录 | 24 课均可进入（等同 `dev=1`） |

首页**不会**连环自动弹彩蛋（避免干扰）；需要时用下面「深链」直接打开 L1/L2/L3。

**关闭测试卡**：`?testcard=0`，或「我的」→ 清除学习数据。

## 手机同 WiFi（微信里发给自已）

1. 笔记本与手机连**同一 WiFi**（手机勿只用 4G）。
2. 双击 **`手机同WiFi预览-满级测试卡.bat`**，复制终端里「手机」那一行。
3. 发到微信文件传输助手，用手机点开。

格式示例（IP 以你机器为准）：

```
http://192.168.1.100:8765/index.html?v=95&testcard=1
```

> 不要用 `localhost` — 手机上打不开。
