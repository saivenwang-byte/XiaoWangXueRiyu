## 这一版能做什么

| 内容 | 说明 |
|------|------|
| **三课** | 标日初级第 14、16、18 课（与课本进度对应选学） |
| **四步** | 单词听读 → 情景会話多选跟读 → 文法卡片 → 小测验 |
| **语音** | 日语朗读（手机流量播放，不另收费） |
| **记录** | 错题会在「復習」里按 1/3/7 天提醒（本机保存） |

## 家长无需操作的部分

- 不用同一 WiFi，不用连家里电脑  
- 不用注册账号（进度存在孩子手机浏览器里）  
- 换手机或清缓存后，学习记录可能清空，属正常现象  

## 孩子卡住时（最常见 3 种）

1. **白屏 / 一直转圈** — 换「在浏览器中打开」，或换 4G 再开链接；链接末尾要有 `?v=46`  
2. **不知道点哪里** — 看每关顶部灰色提示；单词、会話列表可**向下滑**，上面有「第几个 / 共几个」  
3. **录不上音** — 允许麦克风；不要用极简版浏览器；仍不行用系统浏览器打开同一链接  

## 作者更新后

链接不变，版本号会变（如 `v=47`）。老师发新链接时，用最新一条即可。
