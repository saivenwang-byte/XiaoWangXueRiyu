## 2. 日本四岛 · 路径图（已拍板）

### 2.1 视觉原则

- **仅四岛**：北海道 · 本州 · 四国 · 九州（**不含**冲绳、鹿儿岛以南）。
- **示意图**：异形剪影即可，一眼知「日本列岛」，**不当真地图**。
- **通路**：**虚线**串联站点，像游戏关卡图。
- **方向**：**从南向北** — 起点在 **四国**（南端），沿虚线经九州 → 本州 → 北海道。

## 3. 海星 / 海胆 · 星级（取代「萝卜」）

### 3.1 意象（产品层）

| 符号 | 含义 | 文案方向 |
|------|------|----------|
| ⭐ 金星海星 ×3 | 本课 **金星通关** | 听·说·读等 **全满分** |
| ⭐ 银星海星 ×1–2 | 已学但未全满分 | 继续补强 |
| 🦔 海胆（或刺球） | 未通关 / 仅点开 | 刺 = 还有待攻克 |
| ○ 空站 | 未发布课 | 準備中 |

萝卜 **不** 出现在 UI；仅知识库历史比喻。

### 3.2 维度：听 · 说 · 读 · 写 · 绘 · 其他

| 维度 | 是否计入金星 | 与现有四关映射（技术草案） |
|------|----------------|---------------------------|
| **听** | ✅ 必须满分 | 会話：开场白 🔊 已听；测验：每题 🔊 已触发；可选统计「必听句」列表 |
| **说** | ✅ 必须满分 | 会話：用户回合 **每一句** 跟读 `passed` 且 **`score === 100`**（仅百分满分） |
| **读** | ✅ 必须满分 | 文法：本课节点全部「已阅」（翻面/滚动到底）；単語：本课词表 **已浏览完**（计探索，不强制背） |
| **写** | ❌ **排除** | 产品 **无书写关**；永不进入金星公式 |
| **绘** | ⏳ 待我们细化 | 一期建议 = **会話多场景均切换到过**（若仅单场景则等同会話完成）；无画布则不单独「画画」 |
| **其他** | ✅ 必须满分 | **テスト：全题正确**（100%，与「学习要扎实」一致） |

**金星（3 海星）**：上表所有 **✅ 维度** 同时 **100% 满分**；**没有 85 分及格线**，未全对不给三星。

**银星 / 海胆**：有学习记录但未达上述全满分。

### 3.2.1 全册满星 · 隐藏关 · 大礼包（2026-05-21 拍板）

| 项 | 规则 |
|----|------|
| 范围 | **00 入門（隐藏关）+ 第 1–24 課**，共 25 个站点 |
| 单课三星 | 听·说·读·绘(一期)·测验等纳入项 **全部 100%** |
| 全册通关 | 上述 **25 课全部为 3 海星** → 解锁 **最终故事书**（见 §4.3） |
| 游戏心智 | 与单机满成就一致；未满星可跳关复习；奖励 **仅三星可见** |

**写** 永不计入；大礼包条件也不含书写。

### 3.3 各维度 · 100% 判定（技术口径）

| 维度 | 100% 条件（草案） |
|------|-------------------|
| **听** | 本课会話：每场景 opener 🔊 标记已听；测验：每题进入时 TTS 已播放（或清单内句子全覆盖） |
| **说** | 会話用户回合 **每一句** 跟读 `passed === true` 且 **`score === 100`**（仅百分满分计三星） |
| **读** | 文法：本课全部节点至少翻开 1 次；単語：`seen.length === vocab.length` |
| **绘** | 会話：`dialogues.length` 个场景均点过「本场景听完」 |
| **测验** | 本课 `quizQuestions` **首轮全对**（有错记入復習，本课三星否，需全对重打） |

> 现状：`setGateDone` 在「走过」时即 true（如测验做完即可，可错题）。**三星与 gate 脱钩**，由 `dims` + `seaStars` 单独算。

### 3.4 存储（P1 实现时）

```javascript
lessons[14]: {
  gate1, gate2, gate3,           // 保留：粗粒度「走过」
  dims: { listen, speak, read, scene, quiz },  // 各 "full"|"partial"|"none"
  seaStars: 0 | 1 | 2 | 3,      // 仅当全部 dim===full → 3
  quizFirstPass: true | false,  // 测验是否首轮全对
  speakScores: [100,100,100],  // 每句跟读分
}
state.intro = { seaStars: 0|1|2|3 };  // 00 隐藏关，不进正课环
state.grandRewardUnlocked: false;   // 25 课全 3 星
state.dailyLog: { "2026-05-21": { listen, speak, readNodes, minutes } };
```

### 3.5 マイページ = 学习记录卡 · 督导体系（主战场）

**原则**：路径图看「站」；**マイ** 看「量 · 进度 · 效果 · 追查 · 督导」。

| 模块 | 我们展示什么 | 数据来源 |
|------|--------------|----------|
| **今日工作量** | 今日学习分钟（估）、听句数、跟读次数、读卡片数、测验对/错 | `dailyLog[today]` 事件累加 |
| **进度** | 已发布课：海星分布、全册满星进度 `x/25`、当前单元 | `lessons` + `catalog.released` |
| **效果评估** | 一句人话：「今日 3 课金星」「听说读有一项未满分」 | `computeSeaStars` 聚合 |
| **追查提示** | 到期错题（1/3/7 天）、にがて TOP3、未满分维度列表 | `dueMistakes` + `weakGrammarTop3` + `dims` |
| **督导提示** | 温和建议：「第14課 说 未满分，去会話再练」「2 道错题今日到期」 | 规则引擎 `supervisionHints(state)` |

现有 **マイ** 仅有：7 日 heatmap、`parent-progress` 三关 ✅、weak TOP3 → **需改版为「学习记录卡」**，不是另开页面。

**阅读完成的工作量**：不高——文法已有 `cardIndex` 遍历，在 `gn-next` 到最后一张时记 `readNodes`；単語沿用 `markFlashSeen` 计数；每日首次 `touchStudyDay` 时 rollup 到 `dailyLog`。**约 0.5～1 天** 与 `dims` 同批实现。

## 4. 漫画收藏 · 隐藏插画（趣味层 · 服务学习）

> **定位**：主功能仍是 **学日语**（四关 + 督导）；插画是 **三星通关后的可视化收藏**，给家长/学校「看得见的完成度」，激发好奇与好胜。

### 4.1 三层奖励梯

| 层级 | 触发 | 奖励 |
|------|------|------|
| **L1 课次** | 该课 **3 海星**（全维 100%） | 解锁 **1 张课文插画**（未三星前：剪影/「？」）；通关短庆祝 |
| **L2 单元** | 本单元已发布课 **全部 L1** | **4 连张或 6 连张** 漫画条（单元故事一小段） |
| **L3 全册隐藏彩蛋** | **第 1–24 課** 每课 **4 金星**（四关满分） | **全家福** `egg-grand.webp`（24 景合成 · 仅此时可见） |

00 入門 = 隐藏关；三星后给小彩蛋，计入 L3。

### 4.2 原则

- 插画 **不能替代四关**；无三星则无图。
- 风格：**绘本グルミ**统一视觉；小李仅课文/第三人称叙述，**不另设漫画人设**（见 `docs/story-art-brief-绘本参照.md`）。
- **マイ** 展示「漫画收藏 12/25」「单元条 2/6」供家长截图。

### 4.3 技术（与课内分离）

- 资产：`assets/story/l{nn}.webp`、`unit-{1..6}-strip.webp`、`egg-grand.webp`
- 状态：`state.story.panels / units / book / eggSeen`
- 模块：`js/story-reward.js`（解锁弹层 + 收藏册简页）
- 开发 **约 1～1.5 天**（P1c）；**25 张图** 随播种分期（可先占位）
