# 国内部署 · 个人无公司 · 链接 + 腾讯云 COS（页面与语音同域）

> **适用阶段**：无企业主体；学员侧用 **微信转发 HTTPS 链接** 学习。  
> **不适用**：正式上架「web-view 套壳」微信小程序（需企业或个体工商户后再接，见 [微信小程序-发布一次.md](./微信小程序-发布一次.md)）。  
> **真源代码**：仍在本仓库 Git；国内 COS 为 **对外发行副本**（与 GitHub Pages 可并行一段时间）。

---

## 一、目标架构

```text
学员微信收到链接
    https://learn.你的域名.com/index.html?v=300
         │
         ├─ index.html / js / css / assets / sw.js   ← 页面
         └─ /tts-cache/*.mp3                        ← 语音包（约 28MB，1674 个文件）

同一备案域名 + COS 存储 + CDN 加速（HTTPS）
```

| 配置项 | 文件 | 切国内后应设为 |
|--------|------|----------------|
| 页面正式域 | `js/public-url.config.js` → `HYOUGA_PUBLIC_ORIGIN` | `https://learn.你的域名.com` |
| 语音 MP3 根 | 同文件 → `HYOUGA_TTS_ORIGIN` | **与页面相同**（不要再用 jsDelivr） |
| 版本代际 | `js/share-wechat.js` → `CACHE_VER` | 每次发版 +1 |
| SW 清单 | `tts-cache/sw-manifest.json` | 运行 `python scripts/sync-tts-sw-manifest.py` |

播放链：`js/speech-engine.js` → 同源 `/tts-cache/{key}.mp3`；预缓存见 [TTS-P0-P1-国内加速与SW.md](./TTS-P0-P1-国内加速与SW.md)。

## 二、前置条件（个人可办）

| 项 | 说明 |
|----|------|
| 腾讯云账号 | 完成 **实名认证**（个人身份证） |
| 域名 | 在腾讯云 / DNSPod 购买（建议 `.com` / `.cn`），**未完成备案前不要用该域名对外正式推广** |
| ICP 备案 | 控制台 → **备案** → 新建 **个人** 网站备案（非经营性学习站；网站名称按管局要求填写，避免含「公司」「培训」等敏感词，以审核为准） |
| 备案接入 | 备案时选择 **腾讯云** 为接入商；备案号下来后再绑定 CDN |
| 耗时 | 备案常见 **约 1～3 周**；期间可先用 GitHub 链接，备案通过后再切正式链接 |

**费用粗算（仅供参考）**：域名约 50～100 元/年；COS 存储 + 流量按量（学员量不大时通常很低）；CDN 有免费额度；SSL 可用腾讯云免费证书。

## 三、腾讯云操作步骤（首次）

### 3.1 备案（ICP）

1. 登录 [腾讯云备案控制台](https://console.cloud.tencent.com/beian)。
2. 按向导提交：**个人**、证件照、域名、网站名称、核验（部分地区需幕布或 APP 核验）。
3. 备案通过后，在控制台可看到 **备案号**；网站底部需展示备案号（可在 `index.html` 或全局 footer 加一行，发版时一并上传）。

### 3.2 创建 COS 存储桶

1. [对象存储 COS](https://console.cloud.tencent.com/cos) → **创建存储桶**。
2. 建议：
   - **地域**：选离学员近的（如 **上海 / 广州**），后续 CDN 同源加速。
   - **访问权限**：私有 + 通过 CDN/OSS 回源；若仅 CDN 对外，桶可设为 **私有读写**。
   - **静态网站**（可选）：可在桶设置里开启「静态网站」，索引文档 `index.html`；**正式对外更推荐用 CDN 绑自定义域名**（见下）。

### 3.3 上传整站静态资源

**原则**：与本地 `日语学习` 根目录一致，**页面 + `tts-cache/` 同桶、同路径**。

**必须上传**（路径与仓库根目录一致）：

| 路径 | 说明 |
|------|------|
| `index.html`、`share.html`、`intro.html` 等入口页 | 学习 / 分享 |
| `js/`、`css/`、`assets/`、`icons/` | 应用资源 |
| `tts-cache/*.mp3`、`tts-cache/index.json`、`tts-cache/sw-manifest.json` | 语音包 |
| `sw.js` | Service Worker 预缓存 |
| 微信 / 小程序校验文件（若有） | 根目录 `*.txt`，见 3.6 |

**可不传**（减小上传量，不影响学员）：

| 路径 | 说明 |
|------|------|
| `docs/`、`scripts/`、`mock/`、`.git/` | 开发与文档 |
| `japanese_learning_miniapp/` | 小程序壳，学员 H5 不需要 |
| `Self Study Japanese/`、`tts-cache/*.sqlite` 等 | 非 H5 运行时 |

**上传方式（任选其一）**：

1. **COSBrowser**（图形界面）：安装 [COSBrowser](https://cloud.tencent.com/document/product/436/11366)，登录后把整个站点拖到桶根目录（保持目录结构）。
2. **coscmd**（命令行）：安装后 `coscmd config` → `coscmd upload -r ./  /`（在仓库根目录执行，注意排除 `docs` 等可用 `--ignore`）。
3. **本仓库脚本**：`python scripts/upload-cos.py`（需 `pip install cos-python-sdk-v5` 与 `docs/deploy-secrets.local.json`）。清单见 `scripts/domestic-ship-files.py`；操作手册见 [国内发行-A-Gitee过渡-B-COS正式.md](./国内发行-A-Gitee过渡-B-COS正式.md)。

**首次上传**：`tts-cache/` 约 **28MB、1600+ 文件**，需耐心等待；之后发版可只传变更的 `js/`、`index.html` 和新增 MP3。

### 3.4 配置 CDN + HTTPS + 自定义域名

1. [CDN 控制台](https://console.cloud.tencent.com/cdn) → 添加域名，如 `learn.你的域名.com`。
2. **源站类型**：COS 源 → 选择上面桶；**源站路径**留空（整站根目录）。
3. **HTTPS**：申请并绑定 **免费 SSL 证书**（腾讯云 SSL 或 CDN 一键 HTTPS）。
4. **DNS**：在域名 DNS 添加 CNAME，指向 CDN 分配的加速域名。
5. **缓存规则**（建议）：
   - `*.html`、`/js/*`、`/css/*`：缓存时间 **较短**（或遵循 `?v=` 发版）；发版后可在 CDN 刷新 URL。
   - `/tts-cache/*`：可 **较长**（文件带 hash 名或靠 `CACHE_VER` / SW 代际更新）；发版 bump 版本后 **刷新** `index.html` 与 `sw.js` 即可。

### 3.5 微信内访问注意

- 链接必须是 **`https://`**，证书有效。
- **无需** 为「纯链接转发」配置微信公众平台；学员点开链接 = 微信内置浏览器打开网页。
- 若日后上 **小程序 web-view**：须把 **同一域名** 配为「业务域名」，校验 `txt` 放桶根目录（与 3.6 相同）。

### 3.6 校验文件（备案后 / 日后小程序）

微信公众平台下载的 `MP_verify_xxxxx.txt` 或备案核验文件：

- 放在仓库 **根目录**；
- 上传到 COS **桶根目录**（与 `index.html` 同级）；
- 浏览器可访问：`https://learn.你的域名.com/MP_verify_xxxxx.txt`。

## 四、仓库配置切换（备案域名可用后做一次）

### 4.1 改公网地址（页面 + 语音同源）

编辑 `js/public-url.config.js`：

```javascript
window.HYOUGA_PUBLIC_ORIGIN = "https://learn.你的域名.com";
window.HYOUGA_TTS_ORIGIN = "https://learn.你的域名.com";
window.HYOUGA_TTS_CACHE_VER = "301";  // 与 share-wechat.js 的 CACHE_VER 一致
```

编辑 `js/share-wechat.js`：

```javascript
const CACHE_VER = "301";  // 发版 +1
```

（可选）`js/share-wechat.js` 内 `OFFICIAL_ORIGIN` 若仍写 GitHub，分享逻辑以 `HYOUGA_PUBLIC_ORIGIN` 为准；建议一并改为国内域，避免混淆。

### 4.2 同步 SW 清单并自检

```bat
python scripts/sync-tts-sw-manifest.py
发布前自检.bat
批量检查语音包.bat
```

确认 `tts-cache/sw-manifest.json` 中 `ttsBase` 为：

```text
https://learn.你的域名.com/tts-cache/
```

### 4.3 上传到 COS 并刷新 CDN

1. 上传变更后的 `js/`、`index.html`、`sw.js`、`tts-cache/sw-manifest.json` 及本版新增 MP3。
2. CDN → **刷新预热** → 提交 `https://learn.你的域名.com/index.html` 与 `/sw.js`（必要时刷新 `/js/public-url.config.js`）。

### 4.4 作者发链接（改 bat，避免仍复制 GitHub）

当前 `帮你发布好了.bat` 内链接写死为 GitHub。切国内后请二选一：

- **改 bat**：把 `LINK=` 改为读 `public-url.config.js` 的域名 + `CACHE_VER`；或  
- **新建** `帮你发布-国内.bat`，仅复制国内链接。

正式发给学员的链接格式：

```text
https://learn.你的域名.com/index.html?v=301
```

分享页：

```text
https://learn.你的域名.com/share.html
```

### 4.5 更新文档与知识库

- [链接转发.md](./链接转发.md) 中的「正式链接」改为国内域。
- 知识库 / `微信分享说明.txt` 同步一条国内链接 + 版本号。

**GitHub**：可继续 `git push` 作备份与协作；**学员正式链接**改为国内域，不再依赖 GitHub Pages 更新时效。

## 五、日常发版流程（切国内后）

```text
1. 本地改课 / 语音 / UI
2. 发布前自检.bat  → 全 [OK]
3. python scripts/sync-tts-sw-manifest.py
4. bump CACHE_VER / HYOUGA_TTS_CACHE_VER
5. git commit（可选，作版本记录）
6. 上传变更文件到 COS（整站或增量）
7. CDN 刷新 index.html、sw.js、public-url.config.js
8. 微信 4G 真机：打开正式链接 → 首课 🔊 → 关闭再开测二次是否更快
9. 双击（国内）发布 bat → 复制链接发微信
```

## 六、验收清单（微信 · 4G）

| 项 | 通过标准 |
|----|----------|
| 链接可开 | 好友未关注任何号，点开 `index.html?v=` 能进首页 |
| 地图 / 四关 | 能进课、能答题 |
| 语音首访 | 新环境（清缓存）首课约 10 句 🔊，P95 明显快于 GitHub/jsDelivr |
| 语音二次 | 同版本再进，🔊 接近秒播（SW + 浏览器缓存） |
| 转发 | 学员再转发同链接，好友同样可学 |
| 分享页 | `share.html` 可打开 |

失败时排查：CDN 是否 HTTPS、COS 是否缺文件、`public-url.config.js` 是否已上传、CDN 是否缓存了旧 JS。

## 七、与 GitHub / jsDelivr 的关系

| 环境 | 用途 |

## 九、常见问题

**Q：个人备案能通过吗？**  
A：非经营性个人网站常见可通过；名称与内容避免违规类目，以管局审核为准。

**Q：没有公司是不是不能用小程序？**  
A：个人主体 **一般不能** 用你们现有的 `web-view` 全站 H5 方案正式上线；**链接方案不受影响**。

**Q：语音能否「打进链接里」不用下载？**  
A：MP3 必须在用户设备缓存一次；同域 + 国内 CDN + SW 可让 **第二次** 接近秒播；首访可做「MVP 子集先下」（产品 backlog，见 TTS 文档）。

**Q：备案期间学员用什么？**  
A：继续 `docs/链接转发.md` 中的 GitHub 链接；备案完成后再切换并通知学员「新链接不变内容」。

**Q：COS 费用会很高吗？**  
A：静态站 + 28MB 语音，用户量不大时通常很低；可在腾讯云账单设告警。
