## 版本 C · 审核备注（开发者说明，可较长）

**1. 产品是什么**  
小程序内通过 web-view 打开 H5 学习页，提供日语初级上册 24 课课后练习（单词、会話、文法、测验），地图式课序导航。

**2. 面向谁**  
主要面向**在校学习日语的初中生**及使用同一教材的自学者；学员须**自行购买正版《标准日本语》**，本应用不随附教材全文。

**3. 非官方声明（重要）**  
**非人民教育出版社官方产品**，与《标准日本语》仅为**课后辅助关系**，无隶属、合作或授权关系。

**4. 隐私与权限**  
- 无登录、无手机号收集、学习数据仅存用户设备本地  
- 会話跟读时用户主动触发**麦克风**，仅本地发音反馈，不上传录音  
- 隐私政策：`https://saivenwang-byte.github.io/XiaoWangXueRiyu-v2/privacy.html`  
- 版权说明：`https://saivenwang-byte.github.io/XiaoWangXueRiyu-v2/legal.html`

**5. 服务类目**  
工具-信息查询 / 教育-教育信息展示；不提供在线授课、不招生、无付费。

**6. 测试路径**  
打开小程序 → 进入学习页 → 选择单元课次 → 完成单词/会話/测验任一关；设置内可查看隐私政策链接。

**7. 业务域名**  
`saivenwang-byte.github.io`（已配置校验文件）
