## 三、发现的问题

### 3.1 翻译机械味（7处）

| 课 | 节点 | 问题 |
|----|------|------|
| L4 l4_g3 | "相当于" | 列举事例（非全部），相当于"像…啦…啦之类的" |
| L9 l9_g3 | "相当于" | 表示转折，相当于"虽然…但是…" |
| L9 l9_g4 | "相当于" | 表示推测或确认（～吧）。相当于「～だろう」 |
| L13 l13_g4 | "用于" | 「どのくらい」可用于时间、数量、距离等 |
| L14 l14_g3 | "相当于" | 请求对方做某事（礼貌，相当于"请～"） |
| L20 l20_g1 | "用于" | 辞書形用于「ことができる」「前に」等句型 |
| L24 l24_g4 | "相当于" | 用于引出话题、铺垫请求，相当于"那个…我想…" |

**建议**：将"相当于"改为"即"或"意思是"，将"用于"改为"用来"。
